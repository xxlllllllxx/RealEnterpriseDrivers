蓝牙® 管理和配置

打开和关闭蓝牙无线电

如需禁用蓝牙时，可能会提供三种方法：软件开关、硬件开关或设备管理器。强烈反对混用这三种方法（例如，用软件开关关闭蓝牙，然后又用设备管理器打开蓝牙）。

软件开关是关闭和打开蓝牙无线电的首选方法，切勿通过在设备管理器中禁用/启用蓝牙设备驱动器来关闭或打开蓝牙无线电。但是，并非所有系统都提供软件开关。使用软件开关的步骤因操作系统而异，具体说明如下。

某些系统可能只有硬件开关，或同时具有软件开关和硬件开关。对于同时具有两种开关的情况，请勿同时使用两种开关（即：如果使用软件开关关闭蓝牙无线电，就算先关闭再打开了硬件开关，蓝牙无线电仍将保持关闭状态）。有关硬件开关的信息，请参阅系统的用户手册。

要在 Windows 7 中打开和关闭蓝牙无线电：

  1. 右键单击蓝牙任务栏图标。
  2. 选择打开适配器或关闭适配器。

或者，可以通过“蓝牙设置”窗口关闭蓝牙无线电：

  1. 右键单击蓝牙任务栏图标。
  2. 选择打开设置。
  3. 在选项选项卡上，选中关闭蓝牙适配器。

要在 Windows 8 和更高版本中打开和关闭蓝牙无线电：

  1. 打开“PC 设置”
    • 将光标移至右下角，以滑出 Charm 菜单，选择设置，然后单击更改电脑设置。
  2. 选择无线节点。
  3. 单击“蓝牙”滑块，以在打开和关闭蓝牙之间切换。

文件夹共享设置

要启用或禁用文件共享：

  1. 右击蓝牙任务栏图标，并选择打开设置。
  2. 在蓝牙设置窗口选择共享文件夹选项卡，然后执行以下操作之一：
    • 要启用文件共享，请选中允许远程设备浏览共享文件夹复选框，单击浏览以选择要与蓝牙设备共享的本地文件夹，然后单击确定。
    • 要禁用文件共享，请取消选中允许远程设备浏览共享文件夹复选框。

要启用或禁用通过 FTP 上传文件的用户通知：

  1. 右击蓝牙任务栏图标，并选择打开设置。
  2. 在“蓝牙设置”窗口选择共享文件夹选项卡，然后执行以下操作之一：
    • 要启用用户通知，请选中上传文件至共享文件夹时通知我复选框，然后单击“确定”。
    • 要禁用用户通知，请取消选中上传文件至共享文件夹时通知我复选框。

音频设置

注意：如果您使用的是 Windows 8 操作系统，则此部分不适用。

要管理蓝牙音频设置：

  1. 右击蓝牙任务栏图标，并选择打开设置。
  2. 在蓝牙设置窗口选择音频选项卡，然后执行下列一项或多项操作：
    • 要仅允许蓝牙音频与支持 SCMS-T 数字版权管理 (DRM) 内容保护的设备连接，请选中仅与支持SCMS-T内容保护的音频设备连接复选框。
    • 要在听音乐时自动切换 Skype 来电，请选中启用Skype切换选项，并从下拉菜单中选择先前连接的蓝牙音频耳机的名称。有关详细信息，请参阅在听音乐时接听 IP 语音 (VoIP) 来电。如果计算机上未安装 Skype 软件，则此选项不可用。
