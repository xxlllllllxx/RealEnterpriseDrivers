产品描述

概述

Intel® PROSet/Wireless Software for Bluetooth® Technology 允许使用各种蓝牙设备（如手机、耳塞、立体声耳机、智能手机、打印机、数码相机等）无线连接到计算机。通过蓝牙软件，您可以共享文件、听音乐、进行语音呼叫和使用蓝牙打印机。

借助 Intel® PROSet/Wireless Software for Bluetooth® Technology，使用蓝牙连接更加简单直观，因为它可以跟踪所有已连接的设备并自动连接，而且必要时还可以手动调整。

有关支持的操作系统和适配器的更多信息，请参阅 Intel® 无线产品。

规格

支持的蓝牙核心接口：

  • 主机控制器接口 (HCI)

支持的蓝牙模式：

  • 高级音频分发模式 (A2DP)（发射源/接收器）1
  • 音频/视频远程控制配置文件 (AVRCP)（控制器/目标）1
  • 基本成像配置文件 (BIP)（发起人/响应者）
  • 基本打印模式 (BPP)（发送器）
  • 文件传输协议 (FTP)（客户端/服务器）
  • 通用访问模式 (GAP)
  • 通用属性模式 (GATT)
  • 通用音频/视频分发配置文件 (GAVDP)（来源/接收）1
  • 通用对象交换模式 (GOEP)（客户端/服务器）
  • 免提模式 (HFP)（音频网关）1
  • 硬拷贝电缆更换配置文件 (HCRP)（客户端）
  • 耳机模式 (HSP)（音频网关）1
  • HID over GATT 模式 (HOGP)（主机），也称为低功耗 HID 模式2
  • 对象推送配置文件 (OPP)（客户端/服务器）
  • 电话簿存取规范 (PBAP)（客户端）
  • 同步模式 (SYNC)（客户端）

操作系统：

  • Windows Vista（32 位和 64 位）SP23
  • Windows 7（32 位和 64 位）
  • Windows 8（32 位和 64 位）
  • Windows 8.1（32 位和 64 位）

Intel® PROSet/Wireless Software 还引入了许多新功能：

  • Windows Media Player 和 Media Center 集成1
  • 与 Skype IP 语音 (VoIP) 软件的集成1
  • 与 Microsoft Office 的集成
  • 可以关闭蓝牙无线电3
  • Windows 8 和 Windows 8.1 支持飞行模式
  • 蓝牙 Low Energy 支持2

缩写词列表：

  • A2DP - 高级音频分发模式
  • AMP - 备用 MAC/PHY
  • AVRCP - 音频/视频远程控制模式
  • BIP - 基本图像模式
  • BPP - 基本打印模式
  • DRM - 数字版权管理
  • EDR - 提高的数据速率
  • FTP - 文件传输协议
  • GAP - 通用访问模式
  • GATT - 通用属性模式
  • HID - 人机界面设备
  • HFP - 免提模式
  • HOGP - HID over GATT 模式
  • HSP - 耳机模式
  • MAC/PHY - 媒体访问控制/物理层
  • OBEX - 对象交换
  • PBAP - 电话簿访问模式
  • SCMS-T - 连续复制管理系统
  • SYNCH - 同步模式
  • SRM - 单响应模式
  • VoIP - IP 语音
  • WBS - 宽带语音

1 Windows 8 和 Windows 8.1 本身支持音频配置文件。但媒体播放器与 VoIP 软件之间的交互在 Windows 8 及更高版本上超出了 Intel® PROSet/Wireless Software for Bluetooth® Technology 的范围。

2 本文档中所述的蓝牙 Low Energy 功能仅适用于 Windows 7。Windows 8 和 Windows 8.1 本身支持蓝牙 Low Energy，然而，蓝牙 Low Energy 支持在 Windows 8 及更高版本上超出了 Intel® PROSet/Wireless Software for Bluetooth® Technology 的范围。

3 仅在 Windows 7 中可以从系统任务栏菜单关闭蓝牙无线电。
