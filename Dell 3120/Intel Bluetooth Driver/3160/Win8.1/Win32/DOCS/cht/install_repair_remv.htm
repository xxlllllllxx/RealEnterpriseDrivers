安裝、修復、移除及升級

安裝

若要開始安裝，請啟用硬體並從安裝磁碟啟動 Setup.exe。這樣將會啟動安裝精靈。

遵循安裝精靈指示的步驟：

  1. 按一下歡迎使用頁面上的下一步。
  2. 閱讀並接受授權合約頁面上的授權合約。然後按一下下一步。
  3. 如有需要，請在 [目的地資料夾] 頁面變更目的地資料夾。
    預設的安裝資料夾是 “C:\Program Files\Intel\Bluetooth”。
  4. 在安裝程式準備就緒頁面上，按一下安裝以開始安裝程序。隨即會顯示安裝進度。
  5. 安裝程序完成時，請按一下 [完成]。
  6. 安裝之後，便可從系統匣區域的藍牙® 系統匣圖示功能表中啟動應用程式。

修復

若要進行修復，請遵循下列步驟：

  1. 開啟控制台。
  2. 在控制台的程式集群組中，按一下解除安裝程式連結 (或是連按兩下程式和功能圖示)。
  3. 在應用程式清單中選擇並連按兩下「Intel® PROSet/Wireless Software for Bluetooth® Technology」。
  4. 按一下安裝精靈視窗中的下一步。
  5. 按一下修復並遵循相關指示。
  6. 若要完成修復，請重新啟動系統。完成重新啟動之後，所有變更便會生效。

移除

若要解除安裝，請遵循下列步驟：

  1. 開啟控制台。
  2. 在控制台的程式集群組中，按一下解除安裝程式連結 (或是連按兩下程式和功能圖示)。
  3. 在應用程式清單中選擇並連按兩下「Intel® PROSet/Wireless Software for Bluetooth® Technology」。
  4. 按一下安裝精靈視窗中的下一步。
  5. 按一下移除並遵循相關指示。
  6. 若要完成移除，請重新啟動系統。完成重新啟動之後，便會移除藍牙軟體。

升級

執行下列其中一項任務以執行升級：

  • 執行擁有最新版軟體的安裝程式套件，並且遵循精靈指示。
  • 針對目前已安裝的軟體版本執行解除安裝，重新啟動電腦，並且執行擁有最新版軟體的安裝程式套件。
