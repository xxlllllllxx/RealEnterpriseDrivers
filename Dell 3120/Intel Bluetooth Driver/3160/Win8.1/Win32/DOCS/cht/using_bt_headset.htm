連線至藍牙® 耳機及喇叭

注意：如果您是使用 Windows 8 作業系統，則本節不適用。

藍牙立體聲及單聲道耳機可用於從電腦收聽音樂及接聽 VoIP 來電。此外，您也可以將電腦變成藍牙裝置使用的藍牙喇叭。

支援的音效規範包括：

  • A2DP 接收與來源
  • AVRCP 控制器與目標
  • HFP/HSP 音頻閘道
  • 與 Windows Media Player/Center 整合
  • 與 Skype 整合

透過藍牙耳機或喇叭從電腦收聽音樂

您可以將藍牙立體聲耳機連接至電腦以收聽音樂，並控制電腦上執行的音訊播放器軟體。連線之後，您可以使用藍牙立體聲音訊裝置針對電腦上執行的音訊播放器停止/暫停/繼續播放、移至下一曲/上一曲，以及控制調高/調低音量。

在新增藍牙裝置的服務視窗中 (請參閱新增並連線至新的藍牙® 裝置)：

  1. 按一下服務視窗中的收聽音樂連結以建立連線。
  2. 建立連線之後，便會更新服務視窗訊息，而且藍牙耳機或喇叭會變成作業系統的預設音訊裝置以收聽從電腦播放的音樂。
  3. 如果連線時並未從藍牙耳機聽到音樂，請檢查預設音訊裝置設定是否如下列預設音訊裝置一節中所描述。
  4. 若要中斷現有音訊連線，請在服務視窗中按一下收聽音樂項目的中斷連線，則作業系統預設的音訊裝置將會切換回其原始設定。

預設音訊裝置

建立音訊連線之後，Windows 作業系統會自動將藍牙音訊裝置設為預設播放裝置。在一些罕見的情況下，藍牙音訊預設裝置可能無法主動運作。例如，如果您先手動將其他音訊裝置設定為預設音訊裝置，然後才建立與藍牙音訊裝置的連線，則藍牙音訊裝置可能無法運作。在這種情況下，請前往系統工具列功能表中的「Windows 音效」內容，或是「控制台」的「硬體和音效」區段中的「管理音訊裝置」，並且將藍牙音訊裝置設定為預設裝置。

使用藍牙耳機接聽 VoIP 來電

藍牙單聲道耳機可用來當做 VoIP 電話傳輸程式的音訊通訊裝置。

在新增藍牙裝置的服務視窗中 (請參閱新增並連線至新的藍牙® 裝置)：

  1. 按一下服務視窗中的將 VoIP 來電轉接至免持聽筒連結。
  2. 建立連線之後，服務視窗訊息會更新，而且會將耳機設為 VoIP 程式的預設通訊音訊裝置 (例如，Skype)。
  3. 若要中斷現有音訊連線，請在服務視窗中按一下將 VoIP 來電轉接至免持聽筒項目中的中斷連線。

收聽音樂期間接聽網際網路語音協定 (VoIP) 來電

收聽音樂時，您可以使用藍牙音訊裝置的按鈕順暢地切換藍牙立體聲及單聲道耳機，以接受或刪除 Skype 的來電。進行 Skype 通話時會自動暫停播放音樂，結束通話之後即恢復播放。

注意：您可以在 Windows Media Player 中啟用藍牙外掛程式，並確認系統允許搭配 btplayerctrl.exe 外掛程式使用 Skype 存取，即可讓這些功能順利地運作。

透過電腦喇叭從藍牙裝置播放音樂

使用電腦的喇叭從藍牙裝置執行音訊播放。

在新增藍牙裝置的服務視窗中 (請參閱新增並連線至新的藍牙® 裝置)：

  1. 按一下服務視窗中的播放音樂連結。
  2. 完成這項操作之後，便會將來自遠端藍牙裝置的音訊串流轉送至電腦喇叭 (或是電腦上設定的其他預設音訊裝置)。
  3. 如果有使用中的播放音樂連線，您便可使用藍牙遠端控制視窗來使用藍牙控制遠端媒體播放器。系統匣中隨即會出現藍牙遠端控制圖示。按一下該圖示之後，隨即會顯示藍牙遠端控制視窗。視窗支援播放/停止、上一曲/下一曲及調高/調低音量控制項等標準動作。
  4. 若要停止將電腦當做藍牙喇叭使用，請按一下服務視窗中的中斷連線連結。
