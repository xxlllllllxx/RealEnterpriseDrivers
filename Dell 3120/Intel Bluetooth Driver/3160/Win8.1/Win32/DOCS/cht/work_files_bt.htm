透過藍牙® 傳輸檔案

與其他藍牙裝置共用電腦上的本機檔案與資料夾；在電腦與藍牙裝置之間傳輸任何類型的檔案；甚至是在藍牙裝置上建立或刪除檔案及資料夾；以及從 Microsoft Office 功能表使用「傳送到」精靈將檔案傳送至藍牙裝置。

支援的設定檔及角色：

  • 檔案傳輸規範 (FTP) (用戶端/伺服器)
  • 基本圖像設定檔 (BIP) (發送方/回應方)
  • 物件發送設定檔 (OPP) (用戶端/伺服器)

支援的功能：

  • 物件交換 (OBEX) 1.5 單一回應模式 (SRM)
  • 將物件推送規範 (OPP) 用戶端與 Microsoft Office 整合

將本機資料夾與其他藍牙裝置共用

遵循本使用者指南中資料夾共用設定一節描述的步驟，即可與其他藍牙裝置共用電腦上的資料夾。

共用資料夾的用途是做為透過 FTP 傳輸檔案的根資料夾，以及接收透過 OPP 與 BIP 傳輸檔案的預設資料夾。

瀏覽遠端藍牙裝置的共用資料夾

連線至支援 FTP 的藍牙裝置時，系統可能會使用類似檔案總管的標準視窗來進行裝置和電腦間的一般檔案操作，一如在電腦上進行本機操作一樣：

  • 瀏覽檔案和資料夾
  • 傳輸檔案和資料夾
  • 刪除檔案和資料夾
  • 重新命名檔案和資料夾
    提示：若遠端裝置支援早於 v1.2 前的 FTP 伺服器角色，那麼便只能重新命名空的資料夾。
  • 建立資料夾

若要從「服務視窗」存取遠端裝置上的共用資料夾：

  1. 開啟裝置的「服務」視窗 (請參閱新增並連線至新的藍牙® 裝置)。
  2. 按一下瀏覽遠端資料夾。

若要從「Windows 檔案總管」存取遠端裝置上的共用資料夾：

  1. 開啟電腦資料夾。
  2. 按一下有遠端裝置名稱的圖示。

提示：如果遠端裝置可暫時支援 FTP 伺服器角色 (意即可開啟或關閉 FTP 伺服器角色)，那麼電腦資料夾中可能會沒有連結。在此情況下，請使用遠端裝置「服務」視窗。

提示：資料交換期間，遠端裝置可能會要求您提供密碼，此時必須在兩種裝置上輸入相同的密碼才能連線。如果密碼不符，則會終止連線。

傳送檔案至藍牙裝置

傳送檔案至藍牙裝置精靈是用來將檔案傳送至遠端藍牙裝置。精靈僅可傳送單一檔案，但可同時啟動數個精靈。精靈會提供選擇目標遠端裝置以及欲傳輸之檔案的步驟順序。傳送檔案至藍牙裝置精靈可在數個情境中執行。

提示：BIP 通常用於從行動裝置將圖片傳送至電腦。OPP 則用於傳送所有其他類型的檔案。傳送檔案至藍牙裝置精靈可支援這兩種設定檔。

從 Microsoft Office 將檔案傳送至藍牙裝置

支援以下 Microsoft Office 版本：

  • Microsoft Office 2010
  • Microsoft Office 2013

支援以下 Microsoft Office 應用程式與項目：

  • Word：文件
  • Excel：活頁簿
  • PowerPoint：簡報
  • Access：資料庫
  • Outlook：約會、連絡人、郵件、記事、工作

Microsoft Office 項目會在可傳送至遠端 Bluetooth 裝置前即自動儲存。

註：若 Microsoft Office 項目先前未曾儲存，則 Word、Excel 和 PowerPoint 中會出現儲存提示。

註：若 Microsoft Office 項目為剛建立的資料庫，則 Access 應用程式應重新啟動。

若要傳送 Microsoft Office 項目至遠端 Bluetooth 裝置：

  1. 根據您使用的 Microsoft Office 版本和 Microsoft Office 應用程式而定，在子功能表共用、儲存並傳送、或儲存並發佈中選擇傳送至 Bluetooth 動作。請參閱 Microsoft Office 說明，瞭解如何執行此項工作。隨即會出現傳送檔案至 Bluetooth 裝置精靈。
  2. 傳送檔案至藍牙裝置精靈開啟之後，請選擇檔案傳送的目的地藍牙裝置，然後按一下下一步按鈕。
  3. 隨即會出現檔案傳輸進度視窗，並且會將項目傳送至選擇的 Bluetooth 裝置。傳輸完成之後，您會立即接獲通知。

從「Windows 檔案總管」將檔案傳送至藍牙裝置

若要從「Windows 檔案總管」將檔案傳送至藍牙裝置：

  1. 在「Windows 檔案總管」中以滑鼠右鍵按一下檔案圖示，並選擇操作功能表中之傳送到項目中的 藍牙 動作。隨即會出現傳送檔案至藍牙裝置精靈。
  2. 傳送檔案至藍牙裝置精靈開啟之後，請選擇檔案傳送的目的地藍牙裝置，然後按一下下一步按鈕。
  3. 隨即會出現檔案傳輸進度視窗，並且會將所選檔案傳送至選擇的藍牙裝置。傳輸完成之後，您會立即接獲通知。

從服務視窗將檔案傳送至藍牙裝置

若要從服務視窗將檔案傳送至藍牙裝置：

  1. 按一下服務視窗中的傳送檔案連結。隨即會出現「開啟」對話視窗。
  2. 選擇清單中的檔案，然後按一下開啟按鈕。隨即會出現傳送檔案至藍牙裝置精靈。
  3. 傳送檔案至藍牙裝置精靈開啟後，按一下下一步按鈕。
  4. 隨即會出現檔案傳輸進度視窗，並且會將所選檔案傳送至選擇的藍牙裝置。傳輸完成之後，您會立即接獲通知。

從藍牙裝置接收檔案

當另一個藍牙裝置使用 BIP 或 OPP 將檔案傳輸至您的電腦時，會出現正在接收檔案視窗，並在其中提供傳入檔案的預設目的地路徑。

提示：BIP 通常用於從行動裝置將圖片傳送至電腦。OPP 則用於傳送所有其他類型的檔案。

您可以在 [正在接收檔案] 視窗中，變更用來儲存傳入檔案/圖片的預設目的地資料夾。若要這麼做，請遵循下列步驟：

  1. 按一下瀏覽並選擇想要的資料夾視窗。
  2. 選取路徑之後，按一下 [接受] 開始傳輸檔案。
  3. 若要拒絕傳入檔案，請按一下拒絕。

從藍牙裝置修改檔案和資料夾

支援 FTP 設定檔的藍牙裝置可以針對位於本機共用資料夾中的檔案和資料夾進行變更。如果遠端裝置嘗試覆寫、重新命名或刪除共用內容，則會要求特殊權限。而且，在執行前述變更之前會出現權限要求視窗。在權限要求視窗中會有下列選項：

  • 選擇僅本次允許，並按一下確定，以暫時允許藍牙裝置執行變更。
  • 選擇永遠允許此裝置，並按一下確定，以在藍牙裝置配對時允許藍牙裝置執行變更。
  • 若要暫時拒絕藍牙裝置執行變更，請按一下取消。

如果欲傳輸的檔案未變更任何現有的檔案，則傳輸該檔案時應不會出現權限要求視窗。

接收檔案進度

當其他藍牙裝置開始傳輸檔案到您的電腦且該傳輸已被接受時，如出現下列情形，則正在接收視窗可能會出現：

  • 如果遠端裝置為 BIP 或 OPP 用戶端，則正在接收視窗即會出現。
  • 如果遠端裝置為 FTP 用戶端，則可遵循使用者指南中資料夾共用設定一節所述的步驟來設定正在接收視窗外觀。

正在接收視窗會顯示每個檔案的下列資訊：

  • 傳輸狀態 (接收成功、未接收、傳輸進行中等)
  • 目前正在傳輸之檔案的傳輸鮑率
  • 接收成功的檔案大小
