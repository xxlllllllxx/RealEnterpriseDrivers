Produktbeskrivelse

Oversigt

Intel® PROSet/trådløs software til Bluetooth®-teknologi muliggør en trådløs forbindelse til en computer ved brug af en række Bluetooth-enheder såsom mobiltelefoner, øretelefoner, stereoheadset, smartphones, printere, digitalkameraer m.v. Du kan bruge Bluetooth-software til at dele filer, lytte til musik, foretage stemmeopkald og bruge Bluetooth-printere.

Med Intel® PROSet/trådløs software til Bluetooth®-teknologi er oprettelse af forbindelse ved brug af Bluetooth enkelt og intuitivt igennem sporing af og automatisk oprettelse af forbindelse til alle forbundne enheder, ligesom det også muliggør manuelle justeringer efter behov.

Der findes yderligere oplysninger om understøttede operativsystemer og adaptere i Intel® trådløse produkter.

Specifikationer

Understøttede Bluetooth Core Interfaces:

  • Host Controller Interface (HCI)

Understøttede Bluetooth-profiler:

  • A2DP (Advanced Audio Distribution Profile) (kilde/modtager)1
  • AVRCP (Audio/Video Remote Control Profile) (controller/mål)1
  • BIP (Basic Imaging Profile) (initiator/besvarer)
  • BPP (Basic Printing Profile) (sender)
  • Filoverførselsprofil (FTP) (klient/server)
  • GAP (Generic Access Profile)
  • GATT (Generic Attribute Profile)
  • GAVDP (Generic Audio/Video Distribution Profile) (kilde/synk)1
  • GOEP (Generic Object Exchange Profile) (klient/server)
  • Hands-Free Profile (HFP) (Audio Gateway) med understøttelse af Wide-Band Speech (WBS)1
  • HCRP (Hardcopy Cable Replacement Profile) (klient)
  • Headsetprofil (HSP) (gateway til lyd)1
  • HID over GATT-profil (HOGP) (vært), også kendt som Low Energy HID-profil2
  • OPP (Object Push Profile) (klient/server)
  • PBAP (Phone Book Access Profile) (klient)
  • SYNC (Synchronization Profile) (klient)

Operativsystemer:

  • Windows Vista (32-bit og 64-bit) SP23
  • Windows 7 (32-bit og 64-bit)
  • Windows 8 (32-bit og 64-bit)
  • Windows 8.1 (32-bit og 64-bit)

Intel® PROSet/Wireless-software introducerer også en række nye funktioner:

  • Windows Media Player- og Media Center-integration1
  • Integration med Skype Voice over IP (VoIP)-software1
  • Integration med Microsoft Office
  • Mulighed for at slukke Bluetooth-radioen3
  • Understøttelse af flytilstand på Windows 8 og Windows 8.1
  • Bluetooth Low Energy-understøttelse2

1 Windows 8 og Windows 8.1 er udviklet til at understøtte lydprofiler. Omvendt er interaktion med medieafspillere og VoIP-software ikke muligt for Intel® PROSet/trådløs software til Bluetooth®-teknologi fra Windows 8 og nyere.

2 Bluetooth-lavenergifunktionen, der er beskrevet i dette dokment, er alene gældende for Windows 7. Windows 8 og Windows 8.1 er udviklet til at understøtte Bluetooth-lavenergi, mens funktionen ikke understøttes i Intel® PROSet/trådløs software til Bluetooth®-teknologi fra Windows 8 og nyere.

3 Mulighed for at slukke Bluetooth-radio fra proceslinjemenuen er kun tilgængelig i Windows 7.
