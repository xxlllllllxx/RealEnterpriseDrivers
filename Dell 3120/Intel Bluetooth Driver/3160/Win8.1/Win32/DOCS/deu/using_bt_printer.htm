Herstellen von Verbindungen zu Bluetooth®-Druckern

Gehen Sie im Fenster Dienste des hinzugefügten Bluetooth-Geräts (siehe unter Hinzufügen und Herstellen einer Verbindung mit einem neuen Bluetooth®-Gerät) wie folgt vor:

  1. Klicken Sie auf den Link Dateien ausdrucken, und wählen Sie die Datei(en) aus, die an den Bluetooth-Drucker gesendet werden soll(en).
  2. Sobald der Druckvorgang abgeschlossen ist, erhalten Sie eine Benachrichtigung.
