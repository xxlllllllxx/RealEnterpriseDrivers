Περιγραφή προϊόντος

Επισκόπηση

Το λογισμικό Intel® PROSet/Wireless Software for Bluetooth® Technology επιτρέπει την ασύρματη σύνδεση του υπολογιστή με διάφορες συσκευές Bluetooth, όπως είναι τα κινητά τηλέφωνα, τα ακουστικά-ψείρες, τα στερεοφωνικά ακουστικά, τα smartphone, οι εκτυπωτές, οι ψηφιακές φωτογραφικές μηχανές κ.λ.π. Με το λογισμικό Bluetooth μπορείτε να μοιράζεστε αρχεία, να ακούτε μουσική, να κάνετε φωνητικές κλήσεις και να χρησιμοποιείτε εκτυπωτές Bluetooth.

Με το λογισμικό Intel® PROSet/Wireless Software for Bluetooth® Technology, η σύνδεση μέσω Bluetooth γίνεται απλά και έξυπνα (όλες οι συνδεδεμένες συσκευές εντοπίζονται και συνδέονται αυτόματα). Μπορείτε όμως και να κάνετε χειροκίνητες ρυθμίσεις, όποτε χρειάζεται.

Για περισσότερες πληροφορίες σχετικά με τα λειτουργικά συστήματα και τους προσαρμογείς που υποστηρίζονται, ανατρέξτε στην ενότητα Ασύρματα προϊόντα Intel®.

Τεχνικά χαρακτηριστικά

Υποστηριζόμενες βασικές διασυνδέσεις Bluetooth:

  • Διασύνδεση κεντρικού ελεγκτή (Host Controller Interface (HCI))

Υποστηριζόμενα προφίλ Bluetooth:

  • Προφίλ Προηγμένης Διανομής Ήχου (Advanced Audio Distribution Profile (A2DP)) (Πηγή/Καταβόθρα)1
  • Προφίλ απομακρυσμένου ελέγχου ήχου/εικόνας [Audio/Video Remote Control Profile (AVRCP)] (Ελεγκτής/Στόχος)1
  • Βασικό προφίλ απεικόνισης [Basic Imaging Profile (BIP)] (Εκκινητής/Αποκριτής)
  • Βασικό Προφίλ Εκτύπωσης (Basic Printing Profile (BPP))  (Αποστολέας)
  • Πρωτόκολλο Μεταφοράς Αρχείων (File Transfer Protocol (FTP)) (Πελάτης/Διακομιστής)
  • Προφίλ γενικής πρόσβασης (GAP)
  • Προφίλ γενικών χαρακτηριστικών (GATT)
  • Γενικό προφίλ διανομής ήχου/εικόνας [Generic Audio/Video Distribution Profile (GAVDP)] (Πηγή/Δέκτης)1
  • Γενικό Προφίλ Ανταλλαγής Αρχείων (Generic Object Exchange Profile (GOEP)) (Πελάτης/Διακομιστής)
  • Προφίλ Hands-Free (HFP) (πύλη ήχου) με υποστήριξη Wide-Band Speech (WBS)1
  • Προφίλ ασύρματης εκτύπωσης εντύπων [Hardcopy Cable Replacement Profile (HCRP)] (Πελάτης)
  • Προφίλ Ακουστικού (Headset Profile (HSP)) (Πύλη ήχου)1
  • Προφίλ HID over GATT (HOGP) (κεντρικός υπολογιστής), γνωστό και ως "προφίλ HID χαμηλής ενέργειας"2
  • Προφίλ ώθησης αντικειμένων [Object Push Profile (OPP)] (Πελάτης/Διακομιστής)
  • Προφίλ πρόσβασης σε τηλεφωνικό κατάλογο [Phone Book Access Profile (PBAP)] (Πελάτης)
  • Προφίλ Συγχρονισμού (Synchronization Profile (SYNC)) (Πελάτης)

Λειτουργικά συστήματα:

  • Windows Vista (32-bit και 64-bit) SP23
  • Windows 7 (32-bit και 64-bit)
  • Windows 8 (32-bit και 64-bit)
  • Windows 8.1 (32-bit και 64-bit)

Το λογισμικό Intel® PROSet/Wireless Software εισάγει επίσης μερικές νέες δυνατότητες:

  • Ενοποίηση με το Windows Media Player και το Windows Media Center1
  • Ενοποίηση με το λογισμικό Skype Voice over IP (VoIP)1
  • Ενοποίηση με το Microsoft Office
  • Δυνατότητα απενεργοποίησης της ασύρματης επικοινωνίας Bluetooth3
  • Υποστήριξη λειτουργίας πτήσης στα Windows 8 και Windows 8.1
  • Υποστήριξη Bluetooth χαμηλής ενέργειας2

Λίστα αρκτικόλεξων:

  • A2DP  Advanced Audio Distribution Profile (Προφίλ Προηγμένης Διανομής Ήχου)
  • AMP  Alternate MAC/PHY (Εναλλακτικό MAC/PHY)
  • AVRCP  Audio/Video Remote Control Profile (Προφίλ Απομακρυσμένου Ελέγχου Ήχου/Εικόνας)
  • BIP  Basic Imaging Profile (Βασικό Προφίλ Απεικόνισης)
  • BPP  Basic Printing Profile (Βασικό Προφίλ Εκτύπωσης)
  • DRM  Digital Rights Management (Διαχείριση Ψηφιακών Δικαιωμάτων)
  • EDR  Enhanced Data Rate (Ενισχυμένη Ταχύτητα Μετάδοσης Δεδομένων)
  • FTP  File Transfer Protocol (Πρωτόκολλο Μεταφοράς Αρχείων)
  • GAP  Προφίλ γενικής πρόσβασης
  • GATT  Προφίλ γενικών χαρακτηριστικών
  • HID  Συσκευή ανθρώπινης διασύνδεσης
  • HFP  Hands-Free Profile (Προφίλ Hands-Free)
  • HOGP  Προφίλ HID over GATT
  • HSP  Headset Profile (Προφίλ Ακουστικού)
  • MAC/PHY  Media Access Control/Physical Layer (Έλεγχος Πρόσβασης Πολυμέσων/Φυσικό Στρώμα)
  • OBEX  Object Exchange (Ανταλλαγή Αντικειμένων)
  • PBAP  Phone Book Access Profile (Προφίλ Πρόσβασης σε Τηλεφωνικό Κατάλογο)
  • SCMS-T  Serial Copy Management System (Σύστημα Διαχείρισης Σειριακής Αντιγραφής)
  • SYNCH  Synchronization Profile (Προφίλ Συγχρονισμού)
  • SRM  Single Response Mode (Λειτουργία Μονής Απόκρισης)
  • VoIP  Voice over IP (Φωνή μέσω IP)
  • WBS  Wide-Band Speech
