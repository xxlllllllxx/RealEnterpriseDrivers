Installation, Repair, Removal and Upgrade

Installation

To begin installation, enable the hardware and launch Setup.exe from your installation disk. This will start the Installation Wizard.

Follow the steps specified by the Installation Wizard:

  1. Click Next on the Welcome page.
  2. Read the license agreement and accept it on the License Agreement page. Then click Next.
  3. If required, change the destination folder on the Destination Folder page.
    The default installation is “C:\Program Files\Intel\Bluetooth”.
  4. On the Ready to Install the Program page, click Install to start the installation process. The installation progress will display.
  5. When the installation process is complete, click Finish.
  6. After installation, the application can be launched from the Bluetooth® tray icon menu in the system tray area.

Repair

To do repairs follow these steps:

  1. Open the Control Panel.
  2. Click on the Uninstall a program link in the Programs group (or double-click on the Programs and Features icon) of the Control Panel.
  3. Select the Intel® PROSet/Wireless Software for Bluetooth® Technology in the applications list and double-click on it.
  4. Click Next on the Setup Wizard window.
  5. Click Repair and follow the instructions.
  6. To finish repairing, reboot the system. All changes will take place after the reboot is complete.

Removal

To uninstall follow these steps:

  1. Open the Control Panel.
  2. Click on the Uninstall a program link in the Programs group (or double-click Programs and Features icon) of the Control Panel.
  3. Select the Intel® PROSet/Wireless Software for Bluetooth® Technology in the applications list and double-click on it.
  4. Click Next on the Setup Wizard window.
  5. Click Remove and follow the instructions.
  6. To finish removal, reboot the system. The Bluetooth software will be removed after the reboot is complete.

Upgrade

Do one of the following to perform an upgrade:

  • Run the installer package that has the newest version of the software, and follow the Wizard instructions.
  • Perform uninstallation of the current version of the software installed, reboot your PC, and run the installer package that has the newest version of the software.
