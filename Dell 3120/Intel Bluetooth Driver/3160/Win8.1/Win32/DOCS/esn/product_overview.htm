Descripción del producto

Descripción general

Intel® PROSet/Wireless Software for Bluetooth® Technology facilita la conexión inalámbrica de un equipo con diversos dispositivos Bluetooth como teléfonos móviles, auriculares de tapón, auriculares estéreo, dispositivos de mano inteligentes, impresoras o cámaras digitales, entre otros. El software Bluetooth permite compartir archivos, escuchar música, realizar llamadas de voz y utilizar impresoras Bluetooth.

Con Intel® PROSet/Wireless Software for Bluetooth® Technology, la conexión por Bluetooth es sencilla e intuitiva ya que no sólo se localizan todos los dispositivos conectados y se establece la conexión con ellos de forma automática sino que, además, es posible aplicar ajustes manuales cuando se desee.

Para obtener más información sobre los adaptadores y los sistemas operativos admitidos, visite Productos inalámbricos Intel®.

Especificaciones

Interfaces básicas de Bluetooth admitidas:

  • Interfaz de controlador de host (HCI)

Perfiles Bluetooth admitidos:

  • Perfil avanzado de distribución de audio (A2DP) (emisor/receptor)1
  • Perfil de control remoto de audio y vídeo (AVRCP) (controlador/objetivo)1
  • Perfil básico de imágenes (BIP) (iniciador/destinatario)
  • Perfil básico de impresión (BPP) (emisor)
  • Perfil de transferencia de archivos (FTP) (cliente/servidor)
  • Perfil de acceso genérico (GAP)
  • Perfil de atributo genérico (GATT)
  • Perfil de distribución de audio y vídeo genérico (GAVDP) (fuente/colector)1
  • Perfil genérico de intercambio de objetos (GOEP) (cliente/servidor)
  • Perfil de manos libres (HFP) (puerta de enlace de audio) con compatibilidad con voz de banda ancha (WBS)1
  • Perfil de recambio del cable de impresión (HCRP) (cliente)
  • Perfil de auriculares (HSP) (puerto de enlace de audio)1
  • Perfil HID por GATT (HOGP) (host), también denominado perfil HID de bajo consumo2
  • Perfil de inserción de objetos (OPP) (cliente/servidor)
  • Perfil de acceso a la libreta de teléfonos (PBAP) (cliente)
  • Perfil de sincronización (SYNC) (cliente)

Sistemas operativos:

  • Windows Vista (32 y 64 bits) SP23
  • Windows 7 (32 y 64 bits)
  • Windows 8 (32 y 64 bits)
  • Windows 8.1 (32 y 64 bits)

Intel® PROSet/Wireless Software también incorpora una serie de características nuevas:

  • Integración con Reproductor de Windows Media y Media Center1
  • Integración con el software de voz sobre IP (VoIP) de Skype1
  • Integración con Microsoft Office
  • Capacidad para apagar la radio Bluetooth3
  • Soporte con el modo avión en Windows 8 y Windows 8.1
  • Compatibilidad con Bluetooth Low Energy2

Lista de acrónimos:

  • A2DP: perfil avanzado de distribución de audio
  • AMP: MAC/PHY alternativa
  • AVRCP: perfil de control remoto de audio y vídeo
  • BIP: perfil de imagen básico
  • BPP: perfil de impresión básico
  • DRM: administración de derechos digitales
  • EDR: velocidad de datos mejorada
  • FTP: protocolo de transferencia de archivos
  • GAP: perfil de acceso genérico
  • GATT: perfil de atributo genérico
  • HID: dispositivo de interfaz humana
  • HFP: perfil manos libres
  • HOGP: perfil HID por GATT
  • HSP: perfil de auriculares
  • MAC/PHY: control de acceso a medios/capa física
  • OBEX: intercambio de objetos
  • PBAP: perfil de acceso a libreta de teléfonos
  • SCMS-T: sistema de administración de copias serie
  • SYNCH: perfil de sincronización
  • SRM: modo de respuesta individual
  • VoIP: voz sobre IP
  • WBS: voz de banda ancha

1 Windows 8 y Windows 8.1 admiten perfiles de audio en formato nativo. Sin embargo, la interacción con reproductores multimedia y software VoIP está fuera del ámbito de Intel® PROSet/Wireless Software for Bluetooth® Technology en Windows 8 y posteriores.

2 La función Bluetooth Low Energy tal como se describe en este documento solo es aplicable a Windows 7. Windows 8 y Windows 8.1 soportan Bluetooth Low Energy de forma nativa y, por tanto, el soporte Bluetooth Low Energy está fuera de alcance para Intel® PROSet/Wireless Software for Bluetooth® Technology en Windows 8 y posteriores.

3 El apagado de la radio Bluetooth desde el menú de la bandeja del sistema únicamente está disponible en Windows 7.
