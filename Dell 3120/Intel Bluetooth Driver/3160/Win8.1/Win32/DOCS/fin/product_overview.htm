Tuotekuvaus

Yleiskuvaus

Intel® PROSet/Wireless Software for Bluetooth® Technology mahdollistaa langattoman yhteyden muodostamisen tietokoneeseen erilaisilla Bluetooth-laitteilla, kuten matkapuhelimilla, korvanapeilla, stereokuulokkeilla, kämmentietokoneilla, tulostimilla ja digitaalikameroilla. Bluetooth-ohjelmistoa käyttämällä voit jakaa tiedostoja, kuunnella musiikkia, soittaa äänipuheluja ja käyttää Bluetooth-tulostimia.

Intel® PROSet/Wireless Software for Bluetooth® Technologyn avulla Bluetooth-yhteyden muodostaminen on helppoa ja intuitiivista. Ohjelmisto havaitsee kaikki yhdistetyt laitteet ja yhdistää ne automaattisesti, mutta mahdollistaa myös manuaaliset säädöt tarvittaessa.

Lisätietoja tuetuista käyttöjärjestelmistä ja sovittimista: Intel® Wireless -tuotteet.

Tekniset tiedot

Tuetut Bluetooth-ydinliittymät:

  • Host Controller Interface (HCI)

Tuetut Bluetooth-profiilit:

  • Advanced Audio Distribution Profile (A2DP) (Lähde/Nielu)1
  • Äänen/videon kaukosäädinprofiili (AVRCP) (ohjain/kohde)1
  • Basic Imaging Profile (BIP) (aloittaja/vastaaja)
  • Basic Printing Profile (BPP) (Lähettäjä)
  • File Transfer Profile (FTP) (Asiakasohjelma/Palvelin)
  • Generic Access Profile (GAP)
  • Generic Attribute Profile (GATT)
  • Yleinen äänen/videon jakeluprofiili (GAVDP) (lähde/kohde)1
  • Generic Object Exchange Profile (GOEP) (Asiakasohjelma/Palvelin)
  • Handsfree-profiili (HFP) (ääniyhdyskäytävä) ja Wide-Band Speech -tuki (WBS)1
  • Hardcopy Cable Replacement -profiili (HCRP) (työasema)
  • Headset Profile (HSP) (Ääniyhdyskäytävä)1
  • HID over GATT -profiili (HOGP) (isäntä), tunnetaan myös nimellä Low Energy HID -profiili2
  • Object Push -profiili (OPP) (työasema/palvelin)
  • Puhelinmuistion käyttöoikeusprofiili (PBAP) (työasema)
  • Synchronization Profile (SYNC) (Asiakasohjelma)

Käyttöjärjestelmät:

  • Windows Vista (32- ja 64-bittinen) SP23
  • Windows 7 (32- ja 64-bittinen)
  • Windows 8 (32- ja 64-bittinen)
  • Windows 8.1 (32- ja 64-bittinen)

Intel® PROSet/Wireless Software -ohjelmisto sisältää myös lukuisia uusia toimintoja:

  • Windows Media Playerin ja Media Centerin integrointi1
  • Integrointi Skype Voice over IP (VoIP) -ohjelmiston kanssa1
  • Integrointi Microsoft Officen kanssa
  • Bluetooth-radion poistaminen käytöstä3
  • Lentokonetilan tuki Windows 8- ja Windows 8.1 -käyttöjärjestelmässä
  • Bluetooth Low Energy -tuki2

Akronyymiluettelo:

  • A2DP – Advanced Audio Distribution Profile
  • AMP – Alternate MAC/PHY
  • AVRCP – Audio/Video Remote Control Profile
  • BIP – Basic Imaging Profile
  • BPP – Basic Printing Profile
  • DRM – Digital Rights Management
  • EDR – Enhanced Data Rate
  • FTP – File Transfer Protocol
  • GAP – Generic Access Profile
  • GATT – Generic Attribute Profile
  • HID – Human Interface Device
  • HFP – Hands-Free Profile
  • HOGP – HID over GATT profile
  • HSP – Headset Profile
  • MAC/PHY – Media Access Control/Physical Layer
  • OBEX – Object Exchange
  • PBAP – Phone Book Access Profile
  • SCMS-T – Serial Copy Management System
  • SYNCH – Synchronization Profile
  • SRM – Single Response Mode
  • VoIP – Voice over IP
  • WBS – Wide-Band Speech

1 Windows 8 ja Windows 8.1 tukevat ääniprofiileja sisäisesti. Mediasoittimen ja VoIP-ohjelmiston käyttäminen ei ole mahdollista Intel® PROSet/Wireless Software for Bluetooth® Technology -ohjelmistolla Windows 8 -käyttöjärjestelmästä eteenpäin.

2 Tässä oppaassa kuvattu Bluetooth Low Energy -ominaisuus on käytettävissä vain Windows 7 -käyttöjärjestelmässä. Windows 8 ja Windows 8.1 tukevat Bluetooth Low Energy -ominaisuutta, mutta Bluetooth Low Energy -tuki ei ole mahdollinen Intel® PROSet/Wireless Software for Bluetooth® Technology -ohjelmistolla Windows 8 -käyttöjärjestelmästä alkaen.

3 Bluetooth-radion voi poistaa käytöstä tehtäväpalkista vain Windows 7 -käyttöjärjestelmässä.
