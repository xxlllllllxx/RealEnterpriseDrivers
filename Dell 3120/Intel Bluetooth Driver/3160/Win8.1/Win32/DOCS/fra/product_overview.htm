Description du produit

Présentation

La technologie Intel® PROSet/Logiciel sans fil pour Bluetooth® permet de connecter sans fil divers périphériques Bluetooth, tels que des téléphones mobiles, des oreillettes, des casques stéréo, des smartphones, des imprimantes, des appareils photos et plus encore. Grâce au logiciel Bluetooth, vous pouvez partager des fichiers, écouter de la musique, effectuer des appels vocaux et utiliser des imprimantes Bluetooth.

Avec Intel® PROSet/Logiciel sans fil pour technologie Bluetooth®, la connexion Bluetooth devient simple et intuitive. Il suit tous les périphériques connectés et s'y connecte automatiquement tout en autorisant des ajustements manuels le cas échéant.

Pour plus d'informations sur les systèmes d'exploitation et les cartes prises en charge, reportez-vous aux Produits sans fil Intel®.

Caractéristiques

Interface Bluetooth principale prise en charge :

  • HCI (Host Controller Interface)

Profils Bluetooth pris en charge :

  • A2DP (Advanced Audio Distribution Profile) (Source/Récepteur)1
  • AVRCP (Audio/Video Remote Control Profile, profil de contrôle à distance audio/vidéo) (contrôleur/cible)1
  • BIP (Basic Imaging Profile, profil d'imagerie de base) (initiateur/répondeur)
  • BPP (Basic Printing Profile) (Émetteur)
  • FTP (File Transfer Profil) (Client/Serveur)
  • Profil d'accès générique (GAP - Generic Access Profile)
  • Profil d'attribut générique (GATT - Generic Attribute Profile)
  • GAVDP (Generic Audio/Video Distribution Profile, profil de distribution audio/vidéo générale) (source/récepteur)1
  • GOEP (Generic Object Exchange Profile) (Client/Serveur)
  • Profil mains libres (HFP - Hands-Free Profile) (passerelle audio) avec prise en charge WBS (Wide-Band Speech - Discours à large bande)1
  • HCRP (Hardcopy Cable Replacement Profile, profil de remplacement de câbles d'impression) (client)
  • HSP (Headset Profile) (Passerelle audio)1
  • Profil HID sur GATT (HOGP) (Hôte), aussi appelé Profil HID à faible consommation d'énergie2
  • OPP (Object Push Profile, profil d'envoi d'objets) (client/serveur)
  • PBAP (Phone Book Access Profile, profil d'accès au carnet d'adresses) (client)
  • SYNC (Synchronization Profile) (Client)

Systèmes d'exploitation :

  • Windows Vista (32 bits et 64 bits) SP23
  • Windows 7 (32 bits et 64 bits)
  • Windows 8 (32 bits et 64 bits)
  • Windows 8.1 (32 bits et 64 bits)

Intel® PROSet/Wireless Software propose aussi un certain nombre de nouvelles fonctions :

  • Intégration Windows Media Player et Media Center1
  • Intégration avec le logiciel de voix sur IP (VoIP) Skype1
  • Intégration avec Microsoft Office
  • Possibilité de désactiver la radio Bluetooth3
  • Prise en charge du mode Avion sous Windows 8 et Windows 8.1
  • Prise en charge du Bluetooth à faible consommation d'énergie2

Liste des acronymes :

  • A2DP – Advanced Audio Distribution Profile
  • AMP – Alternate MAC/PHY
  • AVRCP – Audio/Video Remote Control Profile
  • BIP – Basic Imaging Profile
  • BPP – Basic Printing Profile
  • DRM – Digital Rights Management
  • EDR – Enhanced Data Rate
  • FTP – File Transfer Protocol
  • GAP – Generic Access Profile (Profil d'accès générique)
  • GATT – Generic Attribute Profile (Profil d'attribut générique)
  • HID – Human Interface Device (Périphérique d'interface utilisateur)
  • HFP – Hands-Free Profile
  • HOGP – HID over GATT Profile (Profil HID sur GATT)
  • HSP – Headset Profile
  • MAC/PHY – Media Access Control/Physical Layer
  • OBEX – Object Exchange
  • PBAP – Phone Book Access Profile
  • SCMS-T – Serial Copy Management System
  • SYNCH – Synchronization Profile
  • SRM – Single Response Mode
  • VoIP – Voice over IP
  • WBS – Wide-Band Speech (Discours à large bande)

1 Les systèmes Windows 8 et Windows 8.1 prennent en charge les profils audio en natif. Par conséquent, l'interaction entre les lecteurs multimédia et le logiciel VoIP n'est pas prise en charge avec le logiciel Intel® PROSet/Wireless pour Bluetooth® à partir de Windows 8.

2 La fonctionnalité Bluetooth basse énergie décrite dans ce document s'applique uniquement à Windows 7. Windows 8 et Windows 8.1 prennent en charge la fonctionnalité Bluetooth basse énergie en natif. Par conséquent, cette fonctionnalité n'est pas prise en charge avec le logiciel Intel® PROSet/Wireless pour Bluetooth® à partir de Windows 8.

3 La possibilité de désactiver la radio Bluetooth à partir de la barre d'état système est disponible sous Windows 7 uniquement.
