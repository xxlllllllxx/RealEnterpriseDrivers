Ajout et connexion d'un nouveau périphérique Bluetooth®

Windows 7

L'ajout et la connexion d'un périphérique Bluetooth se fait en trois étapes principales :

  1. Recherchez le périphérique :
    1. Mettez le périphérique Bluetooth sous tension et passez-le en mode découverte. Pour savoir comment procéder, consultez le manuel d'utilisation de votre périphérique.
    2. À partir de l'ordinateur, cliquez sur l'icône Bluetooth dans la barre d'état système, puis sélectionnez l'élément de menu Ajouter un périphérique. Le processus de recherche affiche tous les périphériques Bluetooth se trouvant à portée de l'ordinateur.
  2. Établissez un couplage et une liaison sécurisée avec le périphérique :
    1. Une fois que le périphérique Bluetooth souhaité s'affiche dans la fenêtre Ajouter un périphérique, double-cliquez sur l'icône du périphérique ou sélectionnez-la, puis cliquez sur Suivant. Selon le type de périphérique et les paramètres de sécurité, vous serez peut-être invité à établir un couplage avec le périphérique.
      Remarque : la plupart des périphériques Bluetooth nécessitent une authentification (ou un couplage) pour autoriser une connexion avant que vous puissiez rechercher des services.
    2. Si les codes PIN saisis ou générés sur l'ordinateur et le périphérique Bluetooth sont identiques, l'ordinateur et le périphérique Bluetooth sont couplés. Lorsque vous y êtes invité, sélectionnez Oui pour confirmer que les codes PIN sont identiques, puis cliquez sur Suivant.
    3. Un message s'affiche dans la fenêtre Ajouter un périphérique pour confirmer la réussite du couplage. Cliquez sur le lien Périphériques et imprimantes1 pour afficher le périphérique ajouté, puis cliquez sur le bouton Fermer dans la fenêtre Ajouter un périphérique pour terminer.
  3. Connecter l'ordinateur au service du périphérique Bluetooth :
    1. Double-cliquez sur l'icône du périphérique ajouté pour ouvrir la fenêtre Services du périphérique.
    2. La fenêtre Services indique les opérations possibles avec le périphérique Bluetooth (par ex., Parcourir le dossier distant, Écouter de la musique).

Windows 8 et versions ultérieures

Par défaut, le système d'exploitation Windows 8 et versions ultérieures utilise une interface moderne pour l'ajout de périphériques. Vous trouverez ci-dessous la liste des étapes nécessaires pour ajouter un périphérique et s'y connecter.

  1. Recherchez le périphérique :
    1. Mettez le périphérique Bluetooth sous tension et passez-le en mode découverte. Pour savoir comment procéder, consultez le manuel d'utilisation de votre périphérique.
    2. À partir de l'ordinateur, cliquez sur l'icône Bluetooth dans la barre d'état système, puis sélectionnez l'élément de menu Ajouter un périphérique Bluetooth. Une interface moderne s'ouvre alors et affiche les périphériques se trouvant à portée de l'ordinateur.
  2. Établissez un couplage et une liaison sécurisée avec le périphérique :
    1. Sélectionnez le périphérique souhaité. Selon le type de périphérique et les paramètres de sécurité, vous serez peut-être invité à établir un couplage avec le périphérique.
    2. Si les codes PIN saisis ou générés sur l'ordinateur et le périphérique Bluetooth sont identiques, l'ordinateur et le périphérique Bluetooth sont couplés.
