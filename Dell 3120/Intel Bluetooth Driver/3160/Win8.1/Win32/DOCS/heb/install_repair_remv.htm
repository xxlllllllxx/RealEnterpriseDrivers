התקנה, תיקון, הסרה ושדרוג

התקנה

כדי להתחיל בהתקנה, אפשר את החומרה והפעל את Setup.exe מתוך תקליטור ההתקנה. פעולה זו תגרום להפעלה אשף ההתקנה.

בצע את הפעולות המוצגות באשף ההתקנה:

  1. לחץ על הבא בדף ברוכים הבאים.
  2. קרא את הסכם הרישיון וקבל אותו בדף הסכם רישיון. לאחר מכן לחץ על הבא.
  3. אם יש צורך, שנה את תיקיית היעד בדף תיקיית יעד.
    מיקום ברירת המחדל להתקנה הוא “C:\Program Files\Intel\Bluetooth”.
  4. בדף מוכן להתקנת התוכנית, לחץ על התקן כדי להתחיל בתהליך ההתקנה. התקדמות ההתקנה תוצג.
  5. כאשר תהליך ההתקנה הושלם, לחץ על סיום.
  6. לאחר ההתקנה, ניתן יהיה להפעיל את היישום מהתפריט של סמל מגש Bluetooth® באזור מגש המערכת.

תיקון

כדי לבצע תיקונים, בצע את הפעולות הבאות:

  1. פתח את לוח הבקרה.
  2. לחץ על הקישור הסרת תוכנית בקבוצה תוכניות (או לחץ פעמיים על הסמל תוכניות ותכונות) בלוח הבקרה.
  3. בחר בתוכנת Intel® PROSet/Wireless Software for Bluetooth® Technology ברשימת היישומים ולחץ עליה פעמיים.
  4. לחץ על הבא בחלון אשף ההתקנה.
  5. לחץ על תיקון ומלא אחר ההוראות.
  6. לסיום התיקון, אתחל את המערכת. כל השינויים יוחלו לאחר השלמת האתחול.

הסרה

כדי להסיר את ההתקנה, בצע את השלבים הבאים:

  1. פתח את לוח הבקרה.
  2. לחץ על הקישור הסרת תוכנית בקבוצה תוכניות (או לחץ פעמיים על הסמל תוכניות ותכונות) בלוח הבקרה.
  3. בחר בתוכנת Intel® PROSet/Wireless Software for Bluetooth® Technology ברשימת היישומים ולחץ עליה פעמיים.
  4. לחץ על הבא בחלון אשף ההתקנה.
  5. לחץ על הסר ופעל בהתאם להוראות.
  6. לסיום ההסרה, אתחל את המערכת. תוכנת Bluetooth תוסר לאחר השלמת האתחול.

שדרוג

בצע אחת מהפעולות הבאות כדי לבצע שדרוג:

  • הפעל את חבילת ההתקנה שכולל את הגרסה החדשה ביותר של התוכנה ופעל בהתאם להוראות של האשף.
  • הסר את ההתקנה של הגרסה הנוכחית של התוכנה, אתחל את המחשב והפעל את חבילת ההתקנה שכוללת את הגרסה החדשה ביותר של התוכנה.
