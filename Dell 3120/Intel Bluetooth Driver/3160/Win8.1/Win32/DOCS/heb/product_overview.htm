תיאור מוצר

סקירה כללית

תוכנת Intel® PROSet/Wireless Software for Bluetooth® Technology מאפשרת ליצור חיבור אלחוטי למחשב באמצעות מגוון התקני Bluetooth, כגון טלפונים ניידים, אוזניות, אוזניות סטריאו, התקני כף יד חכמים, מדפסות, מצלמות דיגיטליות ועוד. באמצעות תוכנת Bluetooth תוכל לשתף קבצים, להאזין למוסיקה, לבצע שיחות וידאו ולהשתמש במדפסות Bluetooth.

בעזרת תוכנת Intel® PROSet/Wireless Software for Bluetooth® Technology, החיבור באמצעות Bluetooth מתבצע בצורה פשוטה ואינטואיטיבית, על-ידי ניהול מעקב אחר כל ההתקנים המחוברים והתחברות אליהם באופן אוטומטי, אך התוכנה גם מאפשרת התאמות ידניות בעת הצורך.

לקבלת מידע נוסף על מערכות הפעלה ומתאמים נתמכים, ראה מוצרי Intel® Wireless.

מפרטים

ממשקי ליבה נתמכים של Bluetooth:

  • Host Controller Interface (HCI)

פרופילי Bluetooth נתמכים:

  • 1Advanced Audio Distribution Profile (A2DP) (Source/Sink)
  • פרופיל שלט רחוק שמע/וידאו (AVRCP) (בקר/יעד)1
  • פרופיל הדמיה בסיסי (BIP) (יוזם/משיב)
  • Basic Printing Profile (BPP) (שולח)
  • פרוטוקול העברת קבצים (FTP) (שרת/לקוח)
  • פרופיל גישה כללי (GAP)
  • פרופיל תכונה כללי (GATT)
  • פרופיל חלוקת שמע/וידאו כללי (GAVDP) (מקור/קולט)1
  • Generic Object Exchange Profile (GOEP) (לקוח/שרת)
  • Hands-Free Profile (HFP) (Audio Gateway)‎ עם תמיכה ב-1Wide-Band Speech support (WBS)‎
  • פרופיל החלפת כבל עותק קשיח (HCRP) (לקוח)
  • Headset Profile (HSP) (שער שמע)1
  • HID מעל פרופיל GATT (HOGP) (Host)‎, המכונה גם פרופיל 2Low Energy HID
  • פרופיל אובייקט דחיפה (OPP) (לקוח/שרת)
  • פרופיל גישה לספר טלפונים (PBAP) (לקוח)
  • פרופיל סנכרון (SYNC) (לקוח)

מערכות הפעלה:

  • (32 סיביות ו-64 סיביות) Windows Vista SP23
  • (32 סיביות ו-64 סיביות) Windows 7
  • (32 סיביות ו-64 סיביות) Windows 8
  • (32 סיביות ו-64 סיביות) Windows 8.1

תוכנת Intel® PROSet/Wireless Software מציעה גם כמה תכונות חדשות:

  • שילוב של Windows Media Player ו-1Media Center
  • שילוב עם תוכנת 1Skype Voice over IP (VoIP)
  • שילוב עם Microsoft Office
  • אפשרות לכבות רדיו Bluetooth‏3
  • תמיכה במצב טיסה ב-Windows 8 וב-Windows 8.1
  • תמיכה ב-Bluetooth Low Energy‏2

רשימת ראשי תיבות:

  • (פרופיל חלוקת שמע מתקדם) Advanced Audio Distribution Profile - A2DP
  • (חלופי MAC/PHY) Alternate MAC/PHY - AMP
  • (פרופיל בקרה מרחוק של שמע/וידאו) Audio/Video Remote Control Profile - AVRCP
  • (פרופיל הדמיה בסיסי) Basic Imaging Profile - BIP
  • (פרופיל הדפסה בסיסי) Basic Printing Profile - BPP
  • ניהול זכויות דיגיטלי (Digital Rights Management) - DRM
  • (קצב העברת נתונים משופר) Enhanced Data Rate - EDR
  • (פרוטוקול העברת קבצים) File Transfer Protocol - FTP
  • GAP - פרופיל גישה כללי
  • GATT - פרופיל תכונה כללי
  • HID - התקן ממשק אנוש
  • (פרופיל דיבורית) Hands-Free Profile - HFP
  • HOGP - HID מעל פרופיל GATT
  • (פרופיל אוזנייה) Headset Profile - HSP
  • (בקרת גישה למדיה/שכבה פיזית) Media Access Control/Physical Layer - MAC/PHY
  • (החלפת אובייקטים) Object Exchange - OBEX
  • (פרופיל גישה לספר טלפונים) Phone Book Access Profile - PBAP
  • (מערכת ניהול העתקה טורית) Serial Copy Management System - SCMS-T
  • פרופיל סנכרון - SYNCH
  • (מצב תגובה אחת)Single Response Mode - SRM
  • (קול באמצעות IP) Voice over IP - VoIP
  • WBS - Wide-Band Speech

1 Windows 8 ו-Windows 8.1 תומכים בפרופילי שמע באופן מקורי. בתורה, אינטראקציה עם נגני מדיה ותוכנת VoIP אינה בטווח עבור טכנולוגיית Intel® PROSet/Wireless Software עבור Bluetooth® מ-Windows 8 ואילך.

2 תכונת Bluetooth Low Energy כפי שמתואר במסמך זה ישימה עבור Windows 7 בלבד. Windows 8 ו-Windows 8.1 תומכים באופן מקורי ב-Bluetooth Low Energy, ובתורה, תמיכת Bluetooth Low Energy אינה בטווח עבור טכנולוגיית Intel® PROSet/Wireless Software עבור Bluetooth® מ-Windows 8 ואילך.

3 היכולת לכבות את רדיו Bluetooth מתפריט מגש המערכת זמינה רק עבור Windows 7.
