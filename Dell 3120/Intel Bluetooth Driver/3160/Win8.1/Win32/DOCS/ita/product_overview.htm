Descrizione del prodotto

Panoramica

Intel® PROSet/Wireless Software for Bluetooth® Technology consente il collegamento wireless a un computer utilizzando una varietà di dispositivi, quali telefoni cellulari, auricolari, cuffie stereo, palmari, stampanti, fotocamere digitali e molti altri. Utilizzando il software Bluetooth, è possibile condividere file, ascoltare musica, effettuare chiamate vocali e utilizzare stampanti Bluetooth.

Con Intel® PROSet/Wireless Software for Bluetooth® Technology, la connessione tramite Bluetooth è semplice e intuitiva, tramite il rilevamento di tutti i dispositivi collegati e il collegamento automatico, ma consente anche di effettuare regolazioni manuali quando necessario.

Per ulteriori informazioni sui sistemi operativi e gli adattatori supportati, vedere Prodotti wireless Intel®.

Specifiche

Interfacce Bluetooth principali supportate:

  • Host Controller Interface (HCI)

Profili Bluetooth supportati:

  • Advanced Audio Distribution Profile (A2DP) (source/sink)1
  • Audio/Video Remote Control Profile (AVRCP) (Controller/Destinazione)1
  • Basic Imaging Profile (BIP) (Iniziatore/Soccorritore)
  • Basic Printing Profile (BPP) (sender)
  • FTP (File Transfer Profile) (client/server)
  • Profilo accesso generico (GAP)
  • Profilo attributo generico (GATT)
  • Profilo GAVDP (Generic Audio/Video Distribution Profile) (Sorgente/Sink)1
  • GOEP (Generic Object Exchange Profile) (client/server)
  • Profilo Hands-Free (HFP) (gateway audio) con supporto WBS (Wide-Band Speech)1
  • Profilo HCRP (Hardcopy Cable Replacement Profile) (Client)
  • HSP (Headset Profile) (gateway audio)1
  • Profilo HOGP (HID over GATT) (Host), anche noto come profilo Low Energy HID2
  • Profilo OPP (Object Push Profile) (Client/Server)
  • Profilo PBAP (Phone Book Access Profile) (Client)
  • SYNC (Synchronization Profile) (client)

Sistemi operativi:

  • Windows Vista (32 bit e 64 bit) SP23
  • Windows 7 (32 bit e 64 bit)
  • Windows 8 (32 bit e 64 bit)
  • Windows 8.1 (32 bit e 64 bit)

Intel® PROSet/Wireless Software introduce inoltre alcune nuove funzioni:

  • Integrazione con Windows Media Player e Media Center1
  • Integrazione con il software VoIP (Voice over IP) Skype1
  • Integrazione con Microsoft Office
  • Possibilità di spegnere la radio Bluetooth3
  • Supporto per la modalità aereo in Windows 8 e Windows 8.1
  • Supporto di Bluetooth Low Energy2

Elenco degli acronimi:

  • A2DP – Advanced Audio Distribution Profile
  • AMP – Alternate MAC/PHY
  • AVRCP – Audio/Video Remote Control Profile
  • BIP – Basic Imaging Profile
  • BPP – Basic Printing Profile
  • DRM – Digital Rights Management
  • EDR – Enhanced Data Rate
  • FTP – File Transfer Protocol
  • GAP – Generic Access Profile
  • GATT – Generic Access Profile
  • HID – Human Interface Device
  • HFP – Hands-Free Profile
  • HOGP – Profilo HID over GATT
  • HSP – Headset Profile
  • MAC/PHY – Media Access Control/Physical Layer
  • OBEX – Object Exchange
  • PBAP – Phone Book Access Profile
  • SCMS-T – Serial Copy Management System
  • SYNCH – Synchronization Profile
  • SRM – Single Response Mode
  • VoIP – Voice over IP
  • WBS – Wide-Band Speech

1 Windows 8 e Windows 8.1 supportano i profili audio a livello nativo. Mentre l'interazione con i lettori multimediali e il software VoIP non è supportata da Intel® PROSet/Wireless Software for Bluetooth® Technology a partire da Windows 8.

2 La funzione Bluetooth Low Energy descritta in questo documento è applicabile solo per Windows 7. Windows 8 e Windows 8.1 supportano Bluetooth Low Energy a livello nativo, mentre Bluetooth Low Energy non è supportato da Intel® PROSet/Wireless Software for Bluetooth® Technology a partire da Windows 8.

3 La possibilità di spegnere la radio Bluetooth dal menu della barra delle applicazioni è disponibile solo per Windows 7.
