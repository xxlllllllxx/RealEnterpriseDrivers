Sincronizzazione dei dati personali tramite Bluetooth®

una serie di telefoni e altri dispositivi Bluetooth forniscono l'accesso ai contatti e ad altri dati PIM (gestore di informazioni personali) utilizzando un servizio di sincronizzazione Bluetooth per eseguire il backup dei contatti dal dispositivo Bluetooth in un file e per ripristinare i contatti da un file di backup.

Profili e ruoli supportati:

  • Client SYNC per eseguire il backup e il ripristino dei contatti
  • Client PBAP (Phone Book Access Profile) per il backup dei contatti

Backup e ripristino delle informazioni di contatti personali utilizzando il profilo SYNC

Nota: è possibile eseguire solo il backup e il ripristino del database dei contatti completo nel dispositivo.

Nella finestra Servizi del dispositivo Bluetooth aggiunto (vedere Aggiunta e collegamento di un nuovo dispositivo Bluetooth®) effettuare una delle seguenti operazioni:

  1. Per effettuare il backup, fare clic sul pulsante Backup dei dati PIM con SYNC, selezionare il nome del file di backup dei contatti e fare clic su Salva nella finestra di dialogo Salva come file. Il processo di backup PIM viene avviato con un indicatore di stato sul display e, una volta salvati i dati nel file selezionato, verrà visualizzato un messaggio indicante il completamento.
  2. Per ripristinare, fare clic sul pulsante Ripristina dati PIM con SYNC, selezionare un file precedentemente salvato e fare clic su Apri nella finestra di dialogo Apri. Il processo PIM viene avviato con un indicatore di stato sul display e, una volta ripristinati i dati sul dispositivo Bluetooth, verrà visualizzato un messaggio indicante il completamento.
    Nota: il servizio Ripristina dati PIM con SYNC non è disponibile se il dispositivo non supporta il profilo SYNC.

Backup delle informazioni di contatti personali utilizzando PBAP

Nella finestra Servizi del dispositivo Bluetooth aggiunto (vedere Aggiunta e collegamento di un nuovo dispositivo Bluetooth®) completare le seguenti operazioni:

  1. Fare clic sul pulsante Sincronizza dati PIM con Phone Book Access. Verrà aperta la Finestra di dialogo Backup della rubrica.
  2. Selezionare le posizioni della rubrica (Memoria interna del telefono, Memoria SIM) di cui eseguire il backup e i file di destinazione.
  3. Premere il pulsante Backup per avviare l'operazione di backup PBAP.
  4. L'avanzamento del backup PBAP verrà mostrato nella finestra Ricezione.

Nota: l'operazione di backup PBAP non viene avviata se non vengono selezionate le posizioni della rubrica.

Nota: non è possibile eseguire il backup di entrambe le posizioni della rubrica nello stesso file di destinazione.
