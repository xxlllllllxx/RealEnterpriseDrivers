インストール、修復、削除、およびアップグレード

インストール

インストールを開始するには、ハードウェアを有効にして、インストール ディスクから Setup.exe を起動します。インストール ウィザードが実行されます。

[インストール ウィザード] に示された手順に従います (以下参照)。

  1. [ようこそ] ページの [次へ] をクリックします。
  2. 使用許諾契約書を読み、[使用許諾契約書の同意] ページの内容に同意します。[次へ] ボタンをクリックします。
  3. 必要に応じて、[移動先フォルダ] ページで、移動先フォルダを変更します。
    デフォルトのインストール場所は、「C:\Program Files\Intel\Bluetooth」です。
  4. [インストールの準備が完了しました] ページの [インストール] をクリックして、インストールを開始します。インストールの進行状況が表示されます。
  5. インストール プロセスが完了したら、[終了] をクリックします。
  6. インストール後は、システム トレイ領域の Bluetooth® トレイ アイコン メニューからアプリケーションを起動できます。

修復

修復を実行するには、次の手順に従います。

  1. [コントロール パネル] を開きます。
  2. [コントロール パネル] の [プログラム] グループの [プログラムのアンインストール] リンクをクリックします (または、[プログラムと機能] アイコンをダブルクリックします)。
  3. アプリケーションのリストで Intel® PROSet/Wireless Software for Bluetooth® Technology を選択してダブルクリックします。
  4. [セットアップ ウィザード] ウィンドウの [次へ] をクリックします。
  5. [修復] をクリックし、指示に従います。
  6. 修復を完了するには、システムを再起動します。再起動が完了すると、すべての変更が有効になります。

削除

アンインストールするには、次の手順に従います。

  1. [コントロール パネル] を開きます。
  2. [コントロール パネル] の [プログラム] グループの [プログラムのアンインストール] リンクをクリックします (または、[プログラムと機能] アイコンをダブルクリックします)。
  3. アプリケーションのリストで Intel® PROSet/Wireless Software for Bluetooth® Technology を選択してダブルクリックします。
  4. [セットアップ ウィザード] ウィンドウの [次へ] をクリックします。
  5. [削除] をクリックし、指示に従います。
  6. 削除を完了するには、システムを再起動します。再起動が完了すると、Bluetooth ソフトウェアは削除されます。

アップグレード

アップグレードを実行するには、次のいずれかの操作を実行します。

  • 最新バージョンのソフトウェアが含まれるインストーラ パッケージを実行し、ウィザードの指示に従います。
  • 現在インストールされているバージョンのソフトウェアをアンインストールし、コンピュータを再起動し、最新バージョンのソフトウェアが含まれるインストーラ パッケージを実行します。
