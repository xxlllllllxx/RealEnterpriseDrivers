製品の説明

概要

Intel® PROSet/Wireless Software for Bluetooth® Technology を使用すると、モバイル、電話、イヤホン、ステレオ ヘッドセット、スマート ハンドヘルド、プリンタ、デジタル カメラなどの多様な Bluetooth デバイスでコンピュータにワイヤレス接続できます。Bluetooth ソフトウェアを使用することで、ファイルの共有、音楽の再生、音声の発信、Bluetooth プリンタの使用が可能になります。

Intel® PROSet/Wireless Software for Bluetooth® Technology を使用すると、簡単に、また直感的に Bluetooth 接続を利用できます。すべての接続デバイスが追跡され、自動的にデバイスに接続されますが、必要に応じて手動で調整することもできます。

サポートされるオペレーティング システムとアダプタの詳細については、「インテル® ワイヤレス製品」を参照してください。

仕様

サポートしている Bluetooth コア インタフェース:

  • Host Controller Interface (HCI: ホスト コントローラ インタフェース)

サポートしている Bluetooth プロファイル:

  • Advanced Audio Distribution Profile (A2DP: 高度オーディオ配信プロファイル) (ソース/シンク)1
  • Audio/Video Remote Control Profile (AVRCP) (コントローラ/ターゲット)1
  • Basic Imaging Profile (BIP) (発信側/応答側)
  • Basic Printing Profile (BPP: 基本印刷プロファイル) (送信クライアント)
  • File Transfer Profile (FTP: ファイル転送プロトコル) (クライアント/サーバー)
  • Generic Access Profile (GAP: 一般アクセス プロファイル)
  • Generic Attribute Profile (GATT: 一般属性プロファイル)
  • Generic Audio/Video Distribution Profile (GAVDP) (ソース/シンク)1
  • Generic Object Exchange Profile (GOEP: 一般オブジェクト交換プロファイル) (クライアント/サーバー)
  • Wide-Band Speech (WBS) 対応ハンズフリー プロファイル (HFP) (オーディオ ゲートウェイ)1
  • Hardcopy Cable Replacement Profile (HCRP) (クライアント)
  • Headset Profile (HSP: ヘッドセット プロファイル) (オーディオ ゲートウェイ)1
  • HID over GATT プロファイル (HOGP)(ホスト)(Low Energy HID プロファイルとも呼ばれます)2
  • Object Push Profile (OPP) (クライアント/サーバー)
  • Phone Book Access Profile (PBAP) (クライアント)
  • Synchronization Profile (SYNC: 同期プロファイル) (クライアント)

オペレーティング システム:

  • Windows Vista (32 ビット/64 ビット) SP23
  • Windows 7 (32 ビット/64 ビット)
  • Windows 8 (32 ビット/64 ビット)
  • Windows 8.1 (32 ビット/64 ビット)

Intel® PROSet/Wireless Software には、いくつかの新機能が搭載されています (以下参照)。

  • Windows Media Player および Media Center の統合1
  • Skype VoIP (ボイス オーバー IP) ソフトウェアの統合1
  • Microsoft Office の統合
  • Bluetooth 無線をオフにする機能3
  • Windows 8 と Windows 8.1 での機内モードのサポート
  • Bluetooth Low Energy 対応2

頭字語のリスト:

  • A2DP - Advanced Audio Distribution Profile (高度オーディオ配信プロファイル)
  • AMP - Alternate MAC/PHY
  • AVRCP - Audio/Video Remote Control Profile (オーディオ/ビデオ リモート コントロール プロファイル)
  • BIP - Basic Imaging Profile (ベーシック イメージング プロファイル)
  • BPP - Basic Printing Profile (基本印刷プロファイル)
  • DRM - Digital Rights Management (デジタル著作権管理)
  • EDR - Enhanced Data Rate (エンハンスド データ レート)
  • FTP - File Transfer Protocol (ファイル転送プロトコル)
  • GAP - Generic Access Profile (一般アクセス プロファイル)
  • GATT - Generic Attribute Profile (一般属性プロファイル)
  • HID - Human Interface Device (ヒューマン インタフェース デバイス)
  • HFP - Hands-Free Profile (ハンズフリー プロファイル)
  • HOGP - HID over GATT プロファイル
  • HSP - Headset Profile (ヘッドセット プロファイル)
  • MAC/PHY - Media Access Control/Physical Layer (メディア アクセス コントロール/物理層)
  • OBEX - Object Exchange (オブジェクト交換)
  • PBAP - Phone Book Access Profile (電話帳アクセス プロファイル)
  • SCMS-T - Serial Copy Management System (シリアル コピー マネジメント システム)
  • SYNC - Synchronization Profile (同期プロファイル)
  • SRM - Single Response Mode (シングル レスポンス モード)
  • VoIP - Voice over IP (ボイス オーバー IP)
  • WBS - Wide-Band Speech

1 Windows 8 および Windows 8.1 はネイティブで音声プロファイルをサポートしています。そのため、Windows 8 以降の Intel® PROSet/Wireless Software for Bluetooth® Technology の場合、メディア プレーヤーおよび VoIP ソフトウェアとの相互作用は対象外です。

2 このドキュメントに記載されている Bluetooth の低エネルギー機能は、Windows 7 にのみ適用されます。Windows 8 および Windows 8.1 はネイティブで Bluetooth の低エネルギー機能をサポートしています。そのため、Windows 8 以降の Intel® PROSet/Wireless Software for Bluetooth® Technology の場合、Bluetooth の低エネルギーのサポートは対象外です。

3 システムのトレイ メニューから Bluetooth 無線をオフにする機能は、Windows 7 でのみ使用できます。
