Bluetooth® ヘッドセットおよびスピーカーとの接続

注: Windows 8 オペレーティング システムを使用している場合、この項は適用されません。

Bluetooth ステレオおよびモノラル ヘッドセットを使用すると、コンピュータで音楽を聴き、VoIP コールに応答することができます。また、コンピュータを Bluetooth スピーカーにして、Bluetooth デバイスから使用することもできます。

サポートしているオーディオ プロファイルは、次のとおりです。

  • A2DP シンク/ソース
  • AVRCP コントローラ/ターゲット
  • HFP/HSP オーディオ ゲートウェイ
  • Windows Media Player/Center の統合
  • Skype の統合

Bluetooth ヘッドセットまたはスピーカーでコンピュータから音楽を聴く

Bluetooth ステレオ ヘッドセットをコンピュータに接続すると、音楽を聴き、コンピュータで実行されているオーディオ プレーヤー ソフトウェアを制御することができます。接続時には、Bluetooth ステレオ オーディオ デバイスを使用して、コンピュータで実行されているオーディオ プレーヤーで、再生の停止/一時停止/再開、次/前のトラックへの移動、音量の上下を行うことができます。

追加された Bluetooth デバイスの [サービス] ウィンドウで (「新しい Bluetooth® デバイスの追加と接続」を参照)、次の操作を実行します。

  1. [サービス] ウィンドウの [音楽を聴く] リンクをクリックし、接続を確立します。
  2. 接続が確立されると、[サービス] ウィンドウ メッセージが更新されます。また、オペレーティング システムでは、コンピュータから再生される音楽を聴くデフォルトのオーディオ デバイスが、Bluetooth ヘッドセットまたはスピーカーになります。
  3. 接続時に Bluetooth ヘッドセットから音楽を聴かない場合は、後述の「デフォルト オーディオ デバイス」の項に従って、[既定のオーディオ デバイス] 設定を指定してください。
  4. 既存のオーディオ接続を切断するには、[サービス] ウィンドウの [音楽を聴く] 項目の [切断] をクリックします。オペレーティング システムのデフォルト オーディオ デバイスは、元の設定に戻ります。

デフォルト オーディオ機器

オーディオ接続が確立されると、Windows オペレーティング システムで、Bluetooth オーディオ デバイスがデフォルト再生デバイスに自動的に設定されます。まれに、Bluetooth のデフォルト オーディオ デバイスが自動的に動作しないことがあります。たとえば、Bluetooth オーディオ デバイスとの接続が確立される前に、デフォルト オーディオ デバイスとして別のオーディオ デバイスを手動で設定した場合に、Bluetooth オーディオ デバイスが動作しないことがあります。この場合、システム トレイ メニュー、または [コントロール パネル] の [ハードウェアとサウンド] セクションの [オーディオ デバイスの管理] から、Windows の [サウンド] プロパティを開き、Bluetooth オーディオ デバイスをデフォルト (既定) のデバイスに設定します。

Bluetooth ヘッドセットを VoIP コールで使用する

Bluetooth モノラル ヘッドセットは、VoIP 電話プログラムのオーディオ通信デバイスとして使用できます。

追加された Bluetooth デバイスの [サービス] ウィンドウで (「新しい Bluetooth® デバイスの追加と接続」を参照)、次の操作を実行します。

  1. [サービス] ウィンドウの [VoIPコールをハンズフリー デバイスに転送] リンクをクリックします。
  2. 接続が確立されると、[サービス] ウィンドウのメッセージは更新され、Skype などの VoIP プログラムのデフォルト通信オーディオ デバイスとしてヘッドセットが設定されます。
  3. 既存のオーディオ接続を切断するには、[サービス] ウィンドウの [VoIPコールをハンズフリー デバイスに転送] 項目の [切断] をクリックします。

音楽を再生しながら VoIP (ボイス オーバー IP) コールを受信する

音楽を聴いているときに Bluetooth オーディオ デバイスのボタンを使用すると、Bluetooth ステレオおよびモノラル ヘッドセットをシームレスに切り替えて、着信した Skype コールに応答したり、コールを拒否したりできます。音楽は、アクティブな Skype コールの通話中には自動的に一時停止され、コールが終了すると再開されます。

注: このような機能がシームレスに動作するようにするには、Windows Media Player で Bluetooth プラグインを有効にし、"btplayerctrl.exe" プラグインで Skype アクセスを許可します。

コンピュータのスピーカーで Bluetooth デバイスから音楽を再生

コンピュータのスピーカーを使用して、Bluetooth デバイスからオーディオ再生を実行します。

追加された Bluetooth デバイスの [サービス] ウィンドウで (「新しい Bluetooth® デバイスの追加と接続」を参照)、次の操作を実行します。

  1. [サービス] ウィンドウで、[音楽再生] リンクをクリックします。
  2. この操作が完了すると、リモートの Bluetooth デバイス (またはコンピュータに設定されている別のデフォルト オーディオ デバイス) からコンピュータのスピーカーにオーディオ ストリーミングが転送されます。
  3. アクティブな [音楽再生] 接続がある場合は、[Bluetoothリモート コントロール] ウィンドウで、Bluetooth を使用するリモートのメディア プレーヤーを制御できます。システム トレイに [Bluetoothリモート コントロール] アイコンが表示されます。このアイコンをクリックすると、[Bluetoothリモート コントロール] ウィンドウが表示されます。このウィンドウは、再生/停止、次/前のトラックへの移動、音量の上下などの標準的な操作をサポートしています。
  4. Bluetooth スピーカーとしてのコンピュータの使用を止めるには、[サービス] ウィンドウの [切断] をクリックします。
