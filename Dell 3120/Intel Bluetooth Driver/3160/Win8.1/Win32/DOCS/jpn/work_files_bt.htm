Bluetooth® でのファイル転送

コンピュータのローカル ファイルとフォルダを他の Bluetooth デバイスと共有すること、コンピュータと Bluetooth デバイス間で任意のタイプのファイルを転送すること、Bluetooth デバイスでファイルとフォルダの作成または削除を行うこと、Microsoft Office のメニューから送信ウィザードを使用して Bluetooth デバイスにファイルを送信することができます。

サポートされるプロファイルとロール:

  • File Transfer Profile (FTP: ファイル転送プロトコル) (クライアント/サーバー)
  • Basic Imaging Profile (BIP) (発信側/応答側)
  • Object Push Profile (OPP) (クライアント/サーバー)

サポートされている機能:

  • Object Exchange (OBEX オブジェクト交換) 1.5 Single Response Mode (SRM: シングル レスポンス モード)
  • Object Push Profile (OPP: オブジェクト プッシュ プロファイル ) クライアント用としての Microsoft Office の統合

他の Bluetooth デバイスとのローカル フォルダの共有

コンピュータのフォルダを他の Bluetooth デバイスと共有するには、このユーザー ガイドの「フォルダの共有設定」で説明されている手順を実行してください。

共有フォルダは、FTP 経由でファイル転送を行う場合のルート フォルダ、および OPP と BIP 経由でファイルを受信する場合のデフォルトのフォルダとして使用されます。

リモート Bluetooth デバイスの共有フォルダの参照

FTP をサポートしている Bluetooth デバイスに接続した場合は、ローカルの場合と同様の以下のような一般的なファイル操作をデバイスとコンピュータとの間で行うために、標準のエクスプローラに似たウィンドウが使用されることがあります。

  • ファイルとフォルダの参照
  • ファイルとフォルダの転送
  • ファイルとフォルダの削除
  • ファイルとフォルダの名前変更
    注: リモート デバイスが v1.2 より前の FTP サーバー ロールをサポートしている場合、空のフォルダ名のみを変更できます。
  • フォルダの作成

[サービス] ウィンドウからリモート デバイスの共有フォルダにアクセスするには:

  1. デバイスの [サービス] ウィンドウを開きます (「新しい Bluetooth® デバイスの追加と接続」を参照)。
  2. [リモートフォルダを参照する] をクリックします。

Windows エクスプローラからリモート デバイスの共有フォルダにアクセスするには:

  1. [コンピュータ] フォルダを開きます。
  2. リモート デバイス名のアイコンをクリックします。

注: リモート デバイスが FTP サーバー ロールを一時的にサポートする場合 (つまり、FTP サーバー ロールのオン/オフを切り替えることができる場合)、[コンピュータ] フォルダのリンクを空にすることもできます。この場合、リモート デバイスの [サービス] ウィンドウを使用します。

注: データ交換時には、リモート デバイスからパスワードの入力を求められることがあります。この場合、接続を有効にするには、両方のデバイスが同じパスワードを入力する必要があります。パスワードが一致しない場合、接続は中止されます。

Bluetooth デバイスへのファイルの送信

リモート Bluetooth デバイスにファイルを送信するには、Bluetoothデバイスのファイル送信ウィザード ウィザードを使用します。ウィザードでは 1 つのファイルを送信できますが、同時に複数のウィザードを起動することができます。対象のリモート デバイスと送信するファイルを選択する一連の手順が表示されます。Bluetoothデバイスのファイル送信ウィザード ウィザードは、さまざまなシナリオで利用できます。

注: 通常、BIP はモバイル デバイスからコンピュータに画像を送信する際に使用されます。OPP はその他のすべてのファイルを送信するために使用されます。Bluetoothデバイスのファイル送信ウィザード ウィザードは両方のプロファイルをサポートしています。

Microsoft Office から Bluetooth デバイスへのファイル送信

次のバージョンの Microsoft Office がサポートされています:

  • Microsoft Office 2010
  • Microsoft Office 2013

次の Microsoft Office アプリケーションおよびアイテムがサポートされています:

  • Word: ドキュメント
  • Excel: ワークブック
  • PowerPoint: プレゼンテーション
  • Access: データベース
  • Outlook: 予定、連絡先、メッセージ、メモ、タスク

Microsoft Office アイテムは、リモート Bluetooth デバイスに送信される前に自動的に保存されます。

注: Microsoft Office アイテムがまったく保存されていない場合、Word、Excel、および PowerPoint で保存プロンプトが表示されます。

注: Microsoft Office アイテムがデータベースで作成したばかりの場合、Access アプリケーションが再起動されます。

Microsoft Office アイテムをリモート Bluetooth デバイスに送信するには:

  1. Microsoft Office および Microsoft Office アプリケーションのバージョンに応じて、[共有]、[保存して送信]、または [保存して発行] サブメニューの [Bluetooth への送信] アクションを選択します。操作方法については、Microsoft Office のヘルプを参照してください。[Bluetooth デバイスへのファイルの送信] ウィザードが表示されます。
  2. Bluetoothデバイスのファイル送信ウィザード ウィザードが開いたら、ファイルの送信先とする Bluetooth デバイスを選択し、[次へ] ボタンをクリックします。
  3. ファイル転送の進行状況を示すウィンドウが表示され、選択した Bluetooth デバイスにアイテムが送信されます。転送が完了すると通知されます。

Windows エクスプローラから Bluetooth デバイスへのファイル送信

Windows エクスプローラから Bluetooth デバイスにファイルを送信するには:

  1. Windows エクスプローラで、ファイル アイコンを右クリックし、コンテキスト メニューの [送る] 項目の [Bluetooth] アクションを選択します。Bluetoothデバイスのファイル送信ウィザード ウィザードが表示されます。
  2. Bluetoothデバイスのファイル送信ウィザード ウィザードが開いたら、ファイルの送信先とする Bluetooth デバイスを選択し、[次へ] ボタンをクリックします。
  3. ファイルの転送の進行状況ウィンドウが表示され、選択した Bluetooth デバイスに選択したファイルが送信されます。転送が完了すると、通知が表示されます。

[サービス] ウィンドウから Bluetooth デバイスへのファイル送信

[サービス] ウィンドウから Bluetooth デバイスにファイルを送信するには:

  1. [サービス] ウィンドウで、[ファイルを送信する] リンクをクリックします。[開く] ダイアログ ボックスが表示されます。
  2. 一覧からファイルを選択し、[開く] ボタンをクリックします。Bluetoothデバイスのファイル送信ウィザード ウィザードが表示されます。
  3. Bluetoothデバイスのファイル送信ウィザード ウィザードが開いたら、[次へ] ボタンをクリックします。
  4. ファイルの転送の進行状況ウィンドウが表示され、選択した Bluetooth デバイスに選択したファイルが送信されます。転送が完了すると、通知が表示されます。

Bluetooth デバイスからのファイルの受信

別の Bluetooth デバイスが BIP または OPP を使用してコンピュータにファイルの転送を開始する場合は、[ファイルを受信中です] ウィンドウに、受信ファイル用のデフォルトの宛先パスが表示されます。

注: 通常、BIP はモバイル デバイスからコンピュータに画像を送信する際に使用されます。OPP はその他のすべてのファイルを送信するために使用されます。

[ファイルを受信しています] ウィンドウでは、受信ファイル/画像が保存されるデフォルトの移動先フォルダを変更することができます。これを実行するには、以下の手順に従います。

  1. [参照] をクリックし、目的のフォルダ ウィンドウを選択します。
  2. パスを選択したら、[承諾] をクリックして、ファイル転送を開始します。
  3. ファイルの受信を拒否するには、[拒否] をクリックします。

Bluetooth デバイスからのファイルとフォルダの変更

FTP プロファイルをサポートする Bluetooth デバイスでは、ローカルの共有フォルダ内にあるファイルとフォルダを変更できます。リモート デバイスが共有コンテンツの上書き、名前の変更、または削除を行う場合は、特殊な許可が必要です。このような変更を行う前に [許可の要求] ウィンドウが表示されます。[許可の要求] ウィンドウには次のオプションがあります。

  • [今回のみ許可する] を選択し [OK] をクリックして、Bluetooth デバイスからの変更を一時的に許可します。
  • [常にこのデバイスを許可する] を選択し [OK] をクリックして、Bluetooth デバイスがペアリングされている間の Bluetooth デバイスからの変更を許可します。
  • Bluetooth デバイス からの変更を一時的に拒否するには、[キャンセル] をクリックします。

転送対象のファイルによって既存のファイルが変更されない場合は、転送時に [許可の要求] ウィンドウが表示されません。

ファイル受信の進行状況

お使いのコンピュータに対して別の Bluetooth デバイスからファイル転送が開始され、転送を受け入れるときに、[受信] ウィンドウが表示されることがあります。

  • リモート デバイスが BIP クライアントまたは OPP クライアントの場合、[受信] ウィンドウが表示されます。
  • リモート デバイスが FTP クライアントの場合、このユーザー ガイドの「フォルダの共有設定」に記載されている手順に従って、[受信] ウィンドウの表示を構成できます。

[受信] ウィンドウには、各ファイルの次の情報が表示されます。

  • 転送の状態 (正常に受信、非受信、転送の進行中など)
  • 現在転送されているファイルの転送ボーレート
  • 正常に受信されたファイルのサイズ
