제품 설명

개요

Intel® PROSet/Wireless Software for Bluetooth® Technology를 사용하면 이동전화, 이어버드, 스테레오 헤드셋, 스마트 핸드헬드, 프린터, 디지털 카메라 등 다양한 Bluetooth 장치를 사용하여 컴퓨터에 무선으로 연결할 수 있습니다. Bluetooth 소프트웨어를 사용하여 파일을 공유하고 음악을 듣고 음성 통화를 하고 Bluetooth 프린터를 사용할 수 있습니다

Intel® PROSet/Wireless Software for Bluetooth® Technology를 사용하면 연결된 모든 장치를 추적하고 장치에 자동으로 연결하는 방식으로 Bluetooth를 간단하게 직관적으로 연결할 수 있지만, 필요할 경우 수동 조정 또한 가능합니다.

지원되는 운영 체제 및 어댑터에 대한 자세한 정보는 Intel® 무선 제품을 참조하십시오.

사양

지원되는 Bluetooth 핵심 인터페이스:

  • HCI(Host Controller Interface)

지원되는 Bluetooth 프로파일:

  • A2DP(Advanced Audio Distribution Profile)(소스/싱크)1
  • A/V 원격 제어 프로필(AVRCP)(컨트롤러/대상)1
  • 기본 이미징 프로필(BIP)(개시자/응답자)
  • BPP(Basic Printing Profile)(발신자)
  • FTP(File Transfer Profile)(클라이언트/서버)
  • GAP(일반 액세스 프로필)
  • GATT(일반 속성 프로필)
  • 기본 A/V 배포 프로필(GAVDP)(소스/싱크)1
  • GOEP(Generic Object Exchange Profile)(클라이언트/서버)
  • WBS(Wide-Band Speech)가 지원되는 HFP(핸즈프리 프로필)(오디오 게이트웨이)1
  • 하드카피 케이블 교체 프로필(HCRP)(클라이언트)
  • HSP(Headset Profile)(오디오 게이트웨이)1
  • 저에너지 HID 프로필이라고도 하는 HOGP(HID over GATT profile)(호스트)2
  • 개체 푸시 프로필(OPP)(클라이언트/서버)
  • 전화번호부 액세스 프로필(PBAP)(클라이언트)
  • SYNC(Synchronization Profile)(클라이언트)

운영 체제:

  • Windows Vista(32비트 및 64비트) SP23
  • Windows 7(32비트 및 64비트)
  • Windows 8(32비트 및 64비트)
  • Windows 8.1(32비트 및 64비트)

Intel® PROSet/Wireless Software에는 다음과 같은 여러 가지 새로운 기능이 있습니다.

  • Windows Media Player 및 Media Center 통합1
  • Skype VoIP(Voice over IP) 소프트웨어와 통합1
  • Microsoft Office와 통합
  • Bluetooth 무선 장치 끄기 기능3
  • Windows 8 및 Windows 8.1에서 비행기 모드 지원
  • Bluetooth 저에너지 지원2

약어 목록:

  • A2DP - Advanced Audio Distribution Profile
  • AMP - Alternate MAC/PHY
  • AVRCP - Audio/Video Remote Control Profile
  • BIP - Basic Imaging Profile
  • BPP - Basic Printing Profile
  • DRM - Digital Rights Management
  • EDR - Enhanced Data Rate
  • FTP - File Transfer Protocol
  • GAP - 일반 액세스 프로필
  • GATT - 일반 속성 프로필
  • HID - 휴먼 인터페이스 장치
  • HFP - Hands-Free Profile
  • HOGP - HID over GATT profile
  • HSP - Headset Profile
  • MAC/PHY - Media Access Control/Physical Layer
  • OBEX - Object Exchange
  • PBAP - Phone Book Access Profile
  • SCMS-T - Serial Copy Management System
  • SYNCH - Synchronization Profile

  • VoIP - Voice over IP
  • WBS - 광대역 음성

1 Windows 8 및 Windows 8.1은 기본적으로 오디오 프로필을 지원합니다. 결과적으로, 미디어 플레이어 및 VoIP 소프트웨어와의 상호 작용은 Windows 8부터 Bluetooth® 기술용 Intel® PROSet/Wireless Software의 범위에 포함되지 않습니다.

2 이 문서에 설명되어 있는 Bluetooth 저에너지(Low Energy) 기능은 Windows 7에만 적용됩니다. Windows 8 및 Windows 8.1은 기본적으로 Bluetooth 저에너지 기능을 지원합니다. 결과적으로, Bluetooth 저에너지 지원은 Windows 8부터 Bluetooth® 기술용 Intel® PROSet/Wireless Software의 범위에 포함되지 않습니다.

3 시스템 트레이 메뉴에서 Bluetooth 무선 장치를 끄는 기능은 Windows 7에서만 가능합니다.
