Bluetooth® 저에너지 HID 장치 연결

참고: 이 문서에 설명된 Bluetooth 저에너지 기능은 Windows 7에만 적용됩니다. Windows 8은 기본적으로 Bluetooth 저에너지를 지원하지만, Intel® PROSet/Wireless Software for Bluetooth® Technology에 대한 Bluetooth 저에너지 HID는 지원하지 않습니다.

Bluetooth 저에너지 HID 장치가 추가되면(새 Bluetooth® 장치 추가 및 연결 참조) Bluetooth 저에너지 HID 장치와의 연결이 자동으로 설정됩니다. 갑자기 연결이 끊길 경우 소프트웨어가 연결을 다시 설정하려 시도합니다.

저에너지 HID 장치의 배터리 상태 확인

연결된 저에너지 HID 장치의 배터리 충전 수준이 10% 미만인 경우, 배터리 상태 아이콘이 시스템 트레이에 표시됩니다.

저에너지 HID 장치의 배터리 상태를 확인하려면 다음 단계를 따르십시오.

  1. 시스템 트레이에서 배터리 상태 아이콘을 클릭합니다.
  2. 저에너지 HID 장치 목록 및 배터리 상태가 표시되는 플라이아웃 창이 나타납니다. 배터리 상태는 연결된 저에너지 HID 장치의 배터리 충전 수준이 10% 미만인 경우에만 표시됩니다.
  3. 플라이아웃 창을 숨기려면 창 외부를 클릭합니다.
