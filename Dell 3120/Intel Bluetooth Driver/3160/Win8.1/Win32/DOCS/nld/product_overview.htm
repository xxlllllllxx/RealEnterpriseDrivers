Productbeschrijving

Overzicht

Met behulp van de Intel® PROSet/Wireless-software voor Bluetooth®-technologie kunt u draadloos verbinding maken met een computer via allerlei Bluetooth-apparaten, zoals mobiele telefoons, oordopjes, stereoheadsets, smart handheld-apparaten, printers, digitale camera's, enzovoort. Met de Bluetooth-software kunt u bestanden delen, muziek beluisteren, telefoongesprekken voeren en Bluetooth-printers gebruiken.

Met behulp van de Intel® PROSet/Wireless-software voor Bluetooth®-technologie kunt u eenvoudig en intuïtief verbinding maken via Bluetooth. U traceert alle verbonden apparaten en maakt daarmee automatisch verbinding. Zo nodig zijn handmatige aanpassingen mogelijk.

Meer informatie over ondersteunde besturingssystemen en adapters vindt u in draadloze Intel®-producten.

Specificaties

Ondersteunde Bluetooth-interfaces:

  • Host Controller Interface (HCI)

Ondersteunde Bluetooth-profielen:

  • A2DP (Advanced Audio Distribution Profile) (bron/doel)1
  • Audio/Video Remote Control Profile (AVRCP) (controller/doel)1
  • Basic Imaging Profile (BIP) (initiator/beantwoorder)
  • Basic Printing Profile (BPP) (afzender)
  • FTP (File Transfer Profile) (client/server)
  • Generic Access Profile (GAP)
  • Generic Attribute Profile (GATT)
  • Generic Audio/Video Distribution Profile (GAVDP) (bron/sink)1
  • GOEP (Generic Object Exchange Profile) (client/server)
  • Hands-Free Profile (HFP) (Audio Gateway) met Wide-Band Speech-ondersteuning (WBS)1
  • Hardcopy Cable Replacement Profile (HCRP) (client)
  • HSP (Headset Profile) (audiogateway)1
  • HID-over-GATT-profile (HOGP) (Host), ook bekend als Low Energy HID-profiel2
  • Object Push Profile (OPP) (client/server)
  • Phone Book Access Profile (PBAP) (client)
  • SYNC (Synchronization Profile) (client)

Besturingssystemen:

  • Windows Vista (32-bits en 64-bits) SP23
  • Windows 7 (32-bits en 64-bits)
  • Windows 8 (32-bits en 64-bits)
  • Windows 8.1 (32-bits en 64-bits)

Intel® PROSet/Wireless Software is daarnaast voorzien van een aantal nieuwe functies:

  • Integratie met Windows Media Player en Media Center1
  • Integratie met Skype VoIP-software (Voice over IP)1
  • Integratie met Microsoft Office
  • Mogelijkheid om de Bluetooth-radio uit te schakelen3
  • Ondersteuning voor de vliegtuigmodus in Windows 8 en Windows 8.1
  • Bluetooth Low Energy-ondersteuning2

Lijst met acroniemen:

  • A2DP – Advanced Audio Distribution Profile
  • AMP – Alternate MAC/PHY
  • AVRCP – Audio/Video Remote Control Profile
  • BIP – Basic Imaging Profile
  • BPP – Basic Printing Profile
  • DRM – Digital Rights Management
  • EDR – Enhanced Data Rate
  • FTP – File Transfer Protocol
  • GAP – Generic Access Profile
  • GATT – Generic Attribute Profile
  • HID – Human Interface Device
  • HFP – Hands-Free Profile
  • HOGP - HID over GATT profile
  • HSP – Headset Profile
  • MAC/PHY – Media Access Control/Physical Layer
  • OBEX – Object Exchange
  • PBAP – Phone Book Access Profile
  • SCMS-T – Serial Copy Management System
  • SYNCH – Synchronization Profile
  • SRM – Single Response Mode
  • VoIP – Voice over IP
  • WBS – Wide-Band Speech

1 Windows 8 en Windows 8.1 ondersteunen van zichzelf audioprofielen. Interactie met mediaspelers en VoIP-software is niet mogelijk voor Intel® PROSet/Wireless-software voor Bluetooth®-technologie onder Windows 8 en hoger.
