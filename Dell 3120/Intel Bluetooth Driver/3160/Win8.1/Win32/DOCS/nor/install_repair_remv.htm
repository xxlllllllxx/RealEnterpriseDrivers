Installering, reparasjon og oppgradering

Installasjon

Hvis du vil starte installeringen, aktiverer du maskinvaren og starter filen Setup.exe fra installasjonsdisken. Installasjonsveiviser starter.

Følg trinnene i installasjonsveiviseren:

  1. Klikk på Neste på siden Velkommen.
  2. Les gjennom og godta lisensavtalen på siden Lisensavtale. Deretter klikker du på Neste.
  3. Endre om nødvendig målmappen på siden Målmappe.
    Standardbanen for installasjonen er “C:\Program Files\Intel\Bluetooth”.
  4. På siden Klar til å installere programmet klikker du på Installer for å starte installasjonsprosessen. Installasjonsfremdriften vises.
  5. Når installasjonsprosessen er fullført, klikker du Fullfør.
  6. Etter installeringen kan programmet startes fra Bluetooth®-ikonmenyen i systemstatusfeltet.

Reparasjon

Følg disse trinnene for å utføre reparasjoner:

  1. Åpne Kontrollpanel.
  2. Klikk på koblingen Avinstaller et program i gruppen Programmer (eller dobbeltklikk på ikonet Programmer og funksjoner) i Kontrollpanel.
  3. Velg Intel® PROSet/Wireless Software for Bluetooth® Technology på programlisten, og dobbeltklikk på den.
  4. Klikk på Neste i vinduet Installasjonsveiviser.
  5. Klikk på Reparer, og følg instruksjonene.
  6. Start systemet på nytt for å fullføre reparasjonen. Alle endringer vil finne sted etter omstarten.

Fjerning

Følg disse trinnene for å avinstallere:

  1. Åpne Kontrollpanel.
  2. Klikk på koblingen Avinstaller et program i gruppen Programmer (eller dobbeltklikk på ikonet Programmer og funksjoner) i Kontrollpanel.
  3. Velg Intel® PROSet/Wireless Software for Bluetooth® Technology på programlisten, og dobbeltklikk på den.
  4. Klikk på Neste i vinduet Installasjonsveiviser.
  5. Klikk på Fjern, og følg instruksjonene.
  6. Start systemet på nytt for å fullføre fjerningen. Bluetooth-programvaren vil bli fjernet etter omstarten.

Oppgradering

Hvis du vil utføre en oppgradering, gjør du ett av følgende:

  • Kjør installasjonspakken som har den nyeste versjonen av programvaren, og følg instruksjonene i Veiviseren.
  • Avinstaller den eksisterende versjonen av programvaren, start PC-en på nytt, og kjør installasjonspakken som har den nyeste versjonen av programvaren.
