Produktbeskrivelse

Oversikt

Med Intel® PROSet/Wireless Software for Bluetooth® Technology kan du koble til en datamaskin trådløst med en rekke ulike Bluetooth-enheter, for eksempel mobiltelefoner, ørepropper, stereohodetelefoner, smarte håndholdte enheter, digitale kameraer med mer. Ved hjelp av Bluetooth-programmet kan du dele filer, lytte til musikk, ringe og bruke Bluetooth-skrivere.

Med Intel® PROSet/Wireless Software for Bluetooth® Technology er det enkelt å koble til med Bluetooth fordi teknologien sporer opp alle tilkoblede enheter og kobler til dem til automatisk, men den tillater også manuelle justeringer etter behov.

Hvis du vil ha mer informasjon om hvilke operativsystemer og adaptere som støttes, kan du se Intel® Wireless-produkter.

Spesifikasjoner

Støttede Bluetooth Core-grensesnitt:

  • Host Controller Interface (HCI)

Støttede Bluetooth-profiler:

  • A2DP (kilde/mottaker) (A2DP - Advanced Audio Distribution Profile)1
  • AVRCP (kontroller/mål) (AVRCP - Profil for ekstern kontroll av lyd/bilde)1
  • BIP (initiator/responder) (BIP - Profil for grunnleggende bildebehandling)
  • BPP (sender) (BPP - Basic Printing Profile)
  • FTP (klient/server) (FTP - File Transfer Protocol)
  • Generic Access Profile (GAP)
  • Generic Attribute Profile (GATT)
  • Generisk profil for distribusjon av lyd/bilde (GAVDP) (kilde/mottak)1
  • GOEP (klient/server) (GOEP - Generic Object Exchange Profile)
  • Hands-Free Profile (HFP) (Audio Gateway) med støtte for Wide-Band Speech (WBS)1
  • Profil for kabelerstatning for papirkopi (HCRP) (klient)
  • HSP (gateway for lyd) (HSP - Headset Profile)1
  • HID over GATT-profil (HOGP) (vert), også kjent som Low Energy HID-profil2
  • Profil for objektsending (OPP) (klient/server)
  • Profil for telefonboktilgang (PBAP) (klient)
  • SYNC (klient) (SYNC - Synchronization Profile)

Operativsystemer:

  • Windows Vista (32-biter og 64-biter) SP23
  • Windows 7 (32-biter og 64-biter)
  • Windows 8 (32-biter og 64-biter)
  • Windows 8.1 (32-biter og 64-biter)

Intel® PROSet/Wireless Software introduserer også en rekke nye funksjoner:

  • Integrasjon av Windows Media Player og Media Center1
  • Integrasjon med Skype VoIP-programvare (VoIP - Voice over IP)1
  • Integrasjon med Microsoft Office
  • Mulighet for å slå av Bluetooth-radioen3
  • Flymodusstøtte på Windows 8 og Windows 8.1
  • Støtte for Bluetooth Low Energy2

  • FTP - File Transfer Protocol

  • HSP - Headset Profile

  • WBS - Wide-Band Speech

1 Windows 8 og Windows 8.1 har innebygd støtte for lydprofiler. Det gjør at samhandling med mediespillere og VoIP-programvare ikke lenger dekkes av Intel® PROSet/Wireless Software for Bluetooth® Technology i Windows 8 og nyere.

2 Bluetooth Low Energy-funksjonen som er beskrevet i dette dokumentet, gjelder bare for Windows 7. Windows 8 og Windows 8.1 støtter Bluetooth Low Energy som standard, så med tiden vil Bluetooth Low Energy ikke lenger dekkes av Intel® PROSet/Wireless Software for Bluetooth® Technology i Windows 8 og nyere.

3 Mulighet for å slå av Bluetooth-radio fra menyen i systemstatusfeltet er bare tilgjengelig i Windows 7.
