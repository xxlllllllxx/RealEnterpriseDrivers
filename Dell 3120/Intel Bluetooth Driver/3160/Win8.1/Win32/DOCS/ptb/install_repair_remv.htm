Instalação, reparo, remoção e atualização

Instalação

Para começar a instalação, habilite o hardware e inicialize o arquivo Setup.exe a partir do disco de instalação. Isso iniciará o Assistente de Instalação.

Siga as etapas especificadas pelo Assistente de instalação:

  1. Clique em Avançar na página Bem-vindo.
  2. Na página Contrato de Licença de Uso, leia e aceite o contrato de licença. Em seguida, clique em Avançar.
  3. Se necessário, altere a pasta de destino na página Pasta de Destino.
    A instalação padrão é “C:\Program Files\Intel\Bluetooth”.
  4. Na página Pronto para Instalar o Programa, clique em Instalar para iniciar o processo de instalação. O andamento da instalação será exibido.
  5. Quando o processo de instalação terminar, clique em Concluir.
  6. Após a instalação, é possível iniciar o aplicativo a partir do menu do ícone Bluetooth® exibido na bandeja do sistema.

Reparo

Para fazer reparos, siga estas etapas:

  1. Abra o Painel de Controle.
  2. Clique no link Desinstalar um programa no grupo Programas (ou clique duas vezes no ícone Programas e Recurso) do Painel de Controle.
  3. Selecione o software Intel® PROSet/Wireless para a tecnologia Bluetooth® na lista de aplicativos e clique duas vezes nele.
  4. Clique em Avançar na janela do Assistente de Instalação.
  5. Clique em Reparar e siga as instruções.
  6. Para finalizar a reparação, reinicie o sistema. Todas as alterações serão implementadas após a conclusão da reinicialização.

Remoção

Para desinstalar, siga estas etapas:

  1. Abra o Painel de Controle.
  2. Clique no link Desinstalar um programa no grupo Programas (ou clique duas vezes no ícone Programas e Recurso) do Painel de Controle.
  3. Selecione o software Intel® PROSet/Wireless para a tecnologia Bluetooth® na lista de aplicativos e clique duas vezes nele.
  4. Clique em Avançar na janela do Assistente de Instalação.
  5. Clique em Remover e siga as instruções.
  6. Para finalizar a remoção, reinicie o sistema. O software Bluetooth será removido após a conclusão da reinicialização.

Atualização

Faça o seguinte para realizar uma atualização:

  • Execute o pacote de instalação que possui a versão mais recente do software e siga as instruções do Assistente.
  • Desinstale a versão atualmente instalada do software, reinicie o computador e execute o pacote de instalação que possui a versão mais recente do software.
