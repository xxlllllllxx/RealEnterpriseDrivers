Descrição do produto

Visão geral

O software Intel® PROSet/Wireless para a tecnologia Bluetooth® proporciona conexão sem fio para um computador usando diversos dispositivos Bluetooth, como telefones celulares, fones de ouvido mono e estéreo, smartphones, impressoras e câmeras digitais, entre outros. Ao usar do software Bluetooth você pode compartilhar arquivos, ouvir músicas, fazer chamadas de voz e usar impressoras Bluetooth.

Com o software Intel® PROSet/Wireless para a tecnologia Bluetooth®, conectar usando Bluetooth é simples e intuitivo, pois ele acompanha todos os dispositivos conectados e conecta-se a eles automaticamente, permitindo também ajustes manuais sempre que necessário.

Para obter mais informações sobre sistemas operacionais e adaptadores, consulte Produtos Wireless da Intel®.

Especificações

Principais interfaces compatíveis com Bluetooth:

  • Host Controller Interface (HCI)

Perfis compatíveis com Bluetooth:

  • Perfil de distribuição de áudio avançado (Advanced Audio Distribution Profile, A2DP) (Origem/sink)1
  • Perfil de controle remoto de áudio/vídeo (AVRCP) (Controlador/Destino)1
  • Perfil básico de imagem (BIP) (Iniciador/Resposta)
  • Perfil básico de impressão (Basic Printing Profile, BPP) (Remetente)
  • Perfil de transferência de arquivo (File Transfer Profile, FTP) (Cliente/servidor)
  • Perfil de acesso genérico (GAP)
  • Perfil de atributo genérico (GATT)
  • Perfil de distribuição genérica de áudio/vídeo (GAVDP) (Origem/Sincronizar)1
  • Perfil de troca de objetos genéricos (Generic Object Exchange Profile, GOEP) (Cliente/servidor)
  • Perfil mãos livres (HFP) (Gateway de áudio) com suporte a WBS (Wide-Band Speech, diálogo em banda larga)1
  • Cópia impressa do perfil de reposição do cabo (HCRP)(Cliente)
  • Perfil de fone de ouvido (Headset Profile, HSP) (Gateway de áudio)1
  • Perfil HID sobre GATT (HOGP) (Host), também conhecido como perfil HID de baixo consumo2
  • Perfil de transmissão do objeto (OPP) (Cliente/Servidor)
  • Perfil de acesso à agenda telefônica (PBAP) (Cliente)
  • Perfil de sincronização (Synchronization Profile, SYNC) (Cliente)

Sistemas operacionais:

  • Windows Vista (32 e 64 bits) SP23
  • Windows 7 (32 e 64 bits)
  • Windows 8 (32 e 64 bits)
  • Windows 8.1 (32 e 64 bits)

O Intel® PROSet/Wireless Software também possui vários recursos novos:

  • Integração entre Windows Media Player e Media Center1
  • Integração com software do Skype de voz via IP (Voice over IP, VoIP)1
  • Integração com o Microsoft Office
  • Capacidade para desativar o rádio Bluetooth3
  • Modo avião oferece suporte ao Windows 8 e o Windows 8.1
  • Suporte a Bluetooth de baixo consumo2

Lista de acrônimos:

  • A2DP – perfil avançado de distribuição de áudio
  • AMP – MAC/PHY alternado
  • AVRCP – Perfil de controle remoto de áudio/vídeo
  • BIP – Perfil básico de imagem
  • BPP – Perfil básico de impressão
  • DRM – Gerenciamento de direitos digitais
  • EDR – Taxa de dados aprimorada
  • FTP – Protocolo de transferência de arquivo
  • GAP – Perfil de acesso genérico
  • GATT – Perfil de atributo genérico
  • HID – Dispositivo de interface humana
  • HFP – Perfil de viva-voz
  • HOGP – Perfil HID sobre GATT
  • HSP – Perfil de fone de ouvido
  • MAC/PHY – Controle de acesso de mídia/camada física
  • OBEX – Troca de objetos
  • PBAP – Perfil de acesso à agenda telefônica
  • SCMS-T – Sistema de gerenciamento de cópia serial
  • SYNCH – Perfil de sincronização
  • SRM – Modo de resposta única
  • VoIP – Voz via IP
  • WBS – Diálogo em banda larga
