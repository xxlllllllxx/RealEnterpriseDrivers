Adicionar e conectar novo dispositivo Bluetooth®

Windows 7

Existem três etapas principais para adicionar e conectar um dispositivo Bluetooth:

  1. Descubra o dispositivo:
    1. Ative o dispositivo Bluetooth e coloque-o no estado visível. Consulte o manual do usuário do dispositivo para saber como fazer isso.
    2. No computador, clique no ícone Bluetooth e selecione o item Adicionar um Dispositivo no menu para realizar um processo de busca que irá mostrar todos os dispositivos localizados no alcance do computador.
  2. Emparelhar e estabelecer um vínculo seguro com o dispositivo:
    1. Depois que o dispositivo Bluetooth desejado aparecer na janela Adicionar um Dispositivo, clique duas vezes no ícone do dispositivo, ou então selecione o ícone e clique em Avançar. Dependendo do tipo de dispositivo e das configurações de segurança, você pode ser solicitado a emparelhar o dispositivo.
      Observação: grande parte dos dispositivos Bluetooth exige autenticação (ou emparelhamento) para permitir uma conexão antes da procura pelos serviços.
    2. Se o código de segurança PIN inserido ou gerado no computador e no dispositivo Bluetooth forem o mesmo, o computador e o dispositivo Bluetooth tornam-se emparelhados. Selecione Sim para confirmar que os códigos de segurança PIN são os mesmos, quando solicitados, e clique em Avançar.
    3. Uma mensagem na janela Adicionar um Dispositivo confirmará o emparelhamento bem sucedido. Clique no link1 Dispositivos e Impressoras para visualizar os dispositivos adicionados, e finalize clicando no botão Fechar na janela Adicionar um Dispositivo.
  3. Conectar o seu computador ao serviço do dispositivo Bluetooth:
    1. Clique duas vezes no ícone do dispositivo adicionado para abrir a janela de Serviços do dispositivo.
    2. A janela Serviços irá mostrar o que pode ser feito com o dispositivo Bluetooth (p.ex. Procurar Pasta Remota, Ouvir Música).

Windows 8 e versões posteriores

Por padrão, o Windows 8 e versões posteriores usam uma interface moderna para adicionar dispositivos. Siga as etapas listadas para adicionar e conectar a um dispositivo.

  1. Descubra o dispositivo:
    1. Ative o dispositivo Bluetooth e coloque-o no estado visível. Consulte o manual do usuário do dispositivo para saber como fazer isso.
    2. No computador, clique no ícone Bluetooth e selecione no menu o item Adicionar um Dispositivo Bluetooth. Uma interface moderna exibirá os dispositivos localizados no alcance do computador.
  2. Emparelhar e estabelecer um vínculo seguro com o dispositivo:
    1. Selecione o dispositivo desejado. Dependendo do tipo de dispositivo e das configurações de segurança, você pode ser solicitado a emparelhar o dispositivo.
    2. Se o código de segurança PIN inserido ou gerado no computador e no dispositivo Bluetooth forem o mesmo, o computador e o dispositivo Bluetooth tornam-se emparelhados.
