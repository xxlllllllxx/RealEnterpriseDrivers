Sincronizando informações pessoais via Bluetooth®

Diversos telefones e demais dispositivos Bluetooth oferecem acesso a contatos e outros dados do gerenciador de informações pessoais (PIM) usando um serviço de sincronização Bluetooth para fazer o backup dos contatos armazenados no dispositivo Bluetooth em um arquivo e para restaurar os contatos a partir do arquivo de backup.

Perfis e funções compatíveis:

  • Cliente SYNC para fazer backup e restaurar
  • Cliente Perfil de acesso a agenda telefônica (PBAP) para o backup dos contatos

Backup e restaurar informações de contatos pessoais usando o perfil SYNC

Observação: É possível fazer backup ou restaurar no dispositivo somente o banco de dados completo de contatos.

Na janela Serviços do dispositivo Bluetooth adicionado (veja Adicionar e conectar novo dispositivo Bluetooth®) faça o seguinte:

  1. Para realizar o backup, clique no botão Fazer backup de dados PIM com SYNC, selecione o nome do arquivo de backup de contatos e clique Salvar na caixa de diálogo Salvar como. O processo de backup PIM é iniciado com um indicador de andamento sendo exibido. Assim que os dados estiverem salvos no arquivo selecionado, será exibida uma mensagem de conclusão.
  2. Para restaurar, clique no botão Restaurar os dados do PIM com SYNC, selecione um arquivo de origem previamente salvo e clique em Abrir na caixa de diálogo Abrir. O processo PIM é iniciado com um indicador de andamento sendo exibido. Assim que os dados estiverem restaurados no dispositivo Bluetooth, será exibida uma mensagem de conclusão.
    Observação: o link do serviço Restaurar os dados do PIM com SYNC não estará disponível caso o dispositivo não suporte o perfil SYNC.

Backup informações de contatos pessoais usando PBAP

Na janela Serviços do dispositivo Bluetooth adicionado (consulte Adicionar e conectar novo dispositivo Bluetooth®) faça o seguinte:

  1. Clique no botão Sincronizar Dados do PIM com Acesso à Agenda Telefônica. O Caixa de diálogo Backup da agenda telefônica será exibida.
  2. Selecione as localizações da agenda telefônica (memória interna do telefone, cartão de memória SIM) para fazer backup dos arquivos de destino.
  3. Pressione o botão Backup para iniciar a operação de backup PBAP.
  4. O progresso da operação de backup PBAP será exibido na janela Recebendo.

Observação: a operação de backup PBAP não será iniciada se nenhuma localização da agenda telefônica for selecionada.

Observação: não é possível fazer o backup de ambas as localizações da agenda telefônica para o mesmo arquivo de destino.
