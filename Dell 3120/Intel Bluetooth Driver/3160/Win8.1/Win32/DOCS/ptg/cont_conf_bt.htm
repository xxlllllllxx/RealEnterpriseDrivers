Gestão e configuração de Bluetooth®

Rádio Bluetooth ligado e desligado

Se for necessário desativar o Bluetooth, existem três métodos para o fazer: através de um interruptor de software, um interruptor de hardware ou do Gestor de Dispositivos. Recomenda-se não misturar os três métodos (por exemplo, desligar o Bluetooth com o interruptor de software e ligá-lo com o Gestor de Dispositivos).

O método preferido para ligar e desligar o rádio Bluetooth é através do interruptor de software; recomenda-se não ativar/desativar o controlador do dispositivo Bluetooth no Gestor de Dispositivos. No entanto, o interruptor de software pode não estar disponível em todos os sistemas. Os procedimentos para utilizar o interruptor de software variam de acordo com os sistemas operativos e são descritos abaixo.

Alguns sistemas podem ter um interruptor de hardware em vez de, ou além de, um interruptor de software. Se tiver ambos, evite utilizá-los ao mesmo tempo (isto é, se o rádio Bluetooth for desligado com o interruptor de software, permanecerá desligado mesmo que tente ligá-lo e desligá-lo com o interruptor de hardware). Consulte o Manual do utilizador do sistema para obter mais informações sobre o interruptor de hardware.

No Windows 7, para ligar e desligar o rádio Bluetooth:

  1. Clique com o botão direito do rato no ícone Bluetooth do tabuleiro.
  2. Selecione Ligar o Adaptador ou Desligar o Adaptador.

Em alternativa, o rádio Bluetooth pode ser desligado através da janela de Definições Bluetooth:

  1. Clique com o botão direito do rato no ícone Bluetooth do tabuleiro.
  2. Selecione Abrir Definições.
  3. No separador Opções, selecione Desligar o adaptador Bluetooth.

No Windows 8 e posterior, para ligar e desligar o rádio Bluetooth:

  1. Abra as Definições do PC
    • Mova o cursor do rato até ao canto inferior direito para apresentar o menu Atalhos, selecione Definições e clique em Alterar definições do PC.
  2. Selecione o nó Sem Fios.
  3. Clique na barra de deslizamento Bluetooth para alternar entre Bluetooth ligado e desligado.

Definições de partilha de pastas

Para ativar ou desativar a partilha de ficheiros:

  1. Clique com o botão direito do rato no ícone Bluetooth no tabuleiro do sistema e selecione Abrir Definições.
  2. Na janela Definições Bluetooth selecione o separador Pasta Partilhada e, em seguida, faça uma das seguintes ações:
    • Para ativar a partilha de ficheiros, selecione a caixa de verificação Permitir que os dispositivos remotos procurem a pasta partilhada, clique em Procurar para selecionar a pasta local a partilhar com os dispositivos Bluetooth e, em seguida, clique em OK.
    • Para desativar a partilha de ficheiros, desmarque a caixa de verificação Permitir que os dispositivos remotos procurem a pasta partilhada.

Para ativar ou desativar notificações de utilizador sobre o carregamento de ficheiros via FTP:

  1. Clique com o botão direito do rato no ícone de tabuleiro Bluetooth e selecione Abrir Definições.
  2. Na janela Definições Bluetooth, selecione o separador Pasta Partilhada e, em seguida, efetue uma das seguintes operações:
    • Para ativar as notificações de utilizador, selecione a caixa de verificação Notificar-me quando forem carregados ficheiros para a pasta partilhada e, em seguida, clique em OK.
    • Para desativar as notificações de utilizador, desmarque a caixa de verificação Notificar-me quando forem carregados ficheiros para a pasta partilhada.

Definições de áudio

Nota: esta secção não é aplicável se estiver a utilizar o sistema operativo Windows 8.

Para gerir as definições áudio Bluetooth:

  1. Clique com o botão direito do rato no ícone Bluetooth no tabuleiro do sistema e selecione Abrir Definições.
  2. Na janela Definições Bluetooth, selecione o separador Áudio e, em seguida, faça uma ou mais das seguintes ações:
    • Para permitir uma ligação áudio Bluetooth apenas com dispositivos que suportem proteção de conteúdos através de gestão de direitos digitais SCMS-T, selecione a caixa de verificação Ligar a um dispositivo de áudio apenas se este suportar a protecção de conteúdos SCMS-T.
    • Para permitir comutação automática de chamadas do Skype enquanto ouve música, selecione a opção Activar Comutação do Skype, e selecione o nome do auricular áudio Bluetooth ligado previamente no menu suspenso. Para mais informações, consulte Receber chamadas VoIP (Voice over IP) enquanto ouve música. Esta opção não está disponível se o software do Skype não estiver instalado no computador.
