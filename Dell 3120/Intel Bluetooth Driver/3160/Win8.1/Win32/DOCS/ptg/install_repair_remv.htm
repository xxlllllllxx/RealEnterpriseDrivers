Instalação, reparação, remoção e atualização

Instalação

Para começar a instalação, ative o hardware e inicie o Setup.exe a partir do seu disco de instalação. Isto irá iniciar o Assistente de Instalação.

Siga os passos especificados pelo Assistente de Instalação:

  1. Clique em Seguinte na página de Bem-vindo.
  2. Leia o contrato de licença e aceite-o na página Contrato de Licença. Em seguida, clique em Seguinte.
  3. Se for necessário, altere a pasta de destino na página Pasta de Destino.
    A instalação predefinida é “C:\Program Files\Intel\Bluetooth”.
  4. Na página Preparado para Instalar o Programa, clique em Instalar para iniciar o processo de instalação. O progresso da instalação será apresentado.
  5. Quando o processo de instalação estiver concluído, clique em Concluir.
  6. Após a instalação, a aplicação pode ser iniciada a partir do menu de ícone do tabuleiro do sistema Bluetooth® na área de tabuleiro do sistema.

Reparação

Para fazer reparações, siga estes passos:

  1. Abra o Painel de Controlo.
  2. Clique na hiperligação Desinstalar um programa no grupo Programas (ou clique duas vezes no ícone Programas e Funcionalidades) do Painel de Controlo.
  3. Selecione o Intel® PROSet/Wireless Software for Bluetooth® technology na lista de aplicações e clique duas vezes.
  4. Clique em Seguinte na janela Assistente de Configuração.
  5. Clique em Reparar e siga as instruções.
  6. Para concluir a reparação, reinicie o sistema. Todas as alterações estarão em efeito após o reinício.

Remoção

Para desinstalar, siga estes passos:

  1. Abra o Painel de Controlo.
  2. Clique na hiperligação Desinstalar um programa no grupo Programas (ou clique duas vezes no ícone Programas e Funcionalidades) do Painel de Controlo.
  3. Selecione o Intel® PROSet/Wireless Software for Bluetooth® technology na lista de aplicações e clique duas vezes.
  4. Clique em Seguinte na janela Assistente de Configuração.
  5. Clique em Remover e siga as instruções.
  6. Para concluir a remoção, reinicie o sistema. O software Bluetooth estará removido após o reinício.

Atualização

Faça uma das seguintes ações para efetuar uma atualização:

  • Execute o pacote de instalação com a versão mais recente do software e siga as instruções do Assistente.
  • Efetue a desinstalação da versão atual instalada do software, reinicie o seu PC e execute o pacote de instalação com a versão mais recente do software.
