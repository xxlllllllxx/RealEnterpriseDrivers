Descrição do produto

Visão geral

O Intel® PROSet/Wireless Software for Bluetooth® technology permite uma ligação sem fios a um computador utilizando vários dispositivos Bluetooth, como telemóveis, auscultadores de isolamento, auriculares estéreo, PDAs inteligentes, impressoras, máquinas fotográficas digitais e outros. Com o software Bluetooth, pode partilhar ficheiros, ouvir música, fazer chamadas de voz e utilizar impressoras Bluetooth.

Com o software Intel® PROSet/Wireless Software for Bluetooth® technology, fazer ligações com Bluetooth é simples e intuitivo através da deteção dos dispositivos ligados e da ligação automática a eles, sem impedir ajustes manuais quando necessário.

Para obter mais informação sobre sistemas e adaptadores operacionais suportados, consulte Produtos sem fios Intel®.

Especificações

Interfaces Bluetooth Nucleares Suportadas:

  • Interface de Controlador Anfitrião (HCI)

Perfis Bluetooth Suportados:

  • Perfil de distribuição de áudio avançado (A2DP) (Origem/Sink)1
  • Perfil de controlo remoto de áudio/vídeo (AVRCP) (Controlador/Alvo)1
  • Perfil básico de imagem (BIP) (Iniciador/Dispositivo de resposta)
  • Perfil de impressão básico (BPP) (Remetente)
  • Perfil de transferência de arquivos (FTP) (Cliente/Servidor)
  • Perfil de acesso genérico (GAP)
  • Perfil de atributo genérico (GATT)
  • Perfil de distribuição geral de áudio/vídeo (GAVDP) (Fonte/Dissipador)1
  • Perfil de troca de objectos genérico (GOEP) (Cliente/Servidor)
  • Perfil de mãos-livres (HFP) (Gateway de Áudio) com suporte de Voz de Banda Larga (WBS)1
  • Perfil de substituição de cabo de impressão (HCRP) (Cliente)
  • Perfil de auricular (HSP) (Gateway de Áudio)1
  • HID sobre perfil GATT (HOGP) (Anfitrião), também designado perfil Low Energy HID2
  • Perfil de transferência de objetos (OPP) (Cliente/Servidor)
  • Perfil de acesso à lista telefónica (PBAP) (Cliente)
  • Perfil de sincronização (SYNC) (Cliente)

Sistemas operativos:

  • Windows Vista (32 bits e 64 bits) SP23
  • Windows 7 (32 bits e 64 bits)
  • Windows 8 (32 bits e 64 bits)
  • Windows 8.1 (32 bits e 64 bits)

O Intel® PROSet/Wireless Software também introduz várias funcionalidades novas:

  • Integração com o Windows Media Player e Media Center1
  • Integração com o software VoIP (Voice over IP) Skype1
  • Integração com o Microsoft Office
  • Capacidade para desligar o rádio Bluetooth3
  • Suporte para o modo de voo no Windows 8 e no Windows 8.1
  • Suporte de Bluetooth Low Energy2

Lista de Acrónimos:

  • A2DP – Perfil de Distribuição de Áudio Avançado (Advanced Audio Distribution Profile)
  • AMP – MAC/PHY Alternativo
  • AVRCP – Perfil de Controlo Remoto de Áudio/Vídeo (Audio/Video Remote Control Profile)
  • BIP – Perfil de Processamento de Imagens Básico (Basic Imaging Profile)
  • BPP – Perfil de Impressão Básico (Basic Printing Profile)
  • DRM – Gestão de Direitos Digitais (Digital Rights Management)
  • EDR – Velocidade de Dados Melhorada (Enhanced Data Rate)
  • FTP – Protocolo FTP (File Transfer Protocol)
  • GAP – Perfil de Acesso Genérico
  • GATT – Perfil de Atributo Genérico
  • HID – Dispositivo de Interface Humana
  • HFP – Perfil de Mãos-livres (Hands-Free Profile)
  • HOGP – HID sobre perfil GATT
  • HSP – Perfil de Auricular (Headset Profile)
  • MAC/PHY – Controlo de Acesso a Suportes de Dados/Camada Física (Media Access Control/Physical Layer)
  • OBEX – Troca de Objectos (Object Exchange)
  • PBAP – Perfil de Acesso à Lista Telefónica (Phone Book Access Profile)
  • SCMS-T – Serial Copy Management System
  • SYNCH – Perfil de Sincronização (Synchronization Profile)
  • SRM – Modo de Resposta Única (Single Response Mode)
  • VoIP – Voice over IP
  • WBS – Voz de Banda Larga
