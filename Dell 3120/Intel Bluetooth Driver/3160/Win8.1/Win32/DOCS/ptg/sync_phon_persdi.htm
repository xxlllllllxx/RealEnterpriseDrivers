Sincronizar informações pessoais através do Bluetooth®

Alguns telefones e outros dispositivos Bluetooth permitem o acesso a contactos e outros dados do Gestor de Informação Pessoal (PIM) utilizando um serviço de sincronização Bluetooth para fazer cópias de segurança dos seus contactos a partir do seu dispositivo Bluetooth para um ficheiro, e para restaurar contactos a partir de um ficheiro de cópia de segurança.

Funções e perfis suportados:

  • Cliente SYNC para fazer cópia de segurança e restaurar contactos
  • Cliente de Perfil de Acesso à Lista Telefónica (PBAP) para fazer cópia de segurança de contactos

Cópias de segurança e restaurar as informações de contactos pessoais utilizando o perfil SYNC

Nota: Só é possível criar uma cópia de segurança ou restaurar a totalidade da base de dados de contactos existente no dispositivo.

Na janela Serviços do dispositivo Bluetooth adicionado (consulte Adicionar e ligar a um novo dispositivo Bluetooth®) efetue uma das seguintes ações:

  1. Para cópia de segurança, clique no botão Efectuar cópia de segurança dos Dados do PIM com SYNC, selecione o nome do ficheiro de cópia de segurança de contactos e clique em Guardar na caixa de diálogo Guardar como Ficheiro. O processo de cópia de segurança do PIM começa com um indicador de progresso no ecrã e será apresentada uma mensagem de conclusão após os dados terem sido guardados no ficheiro selecionado.
  2. Para restaurar, clique no botão Restaurar os Dados do PIM com SYNC, selecione um ficheiro de origem guardado previamente e clique em Abrir na caixa de diálogo Abrir. O processo de restauro do PIM começa com um indicador de progresso no ecrã e uma mensagem de conclusão será apresentada após os dados terem sido restaurados para o dispositivo Bluetooth.
    Nota: a hiperligação para o serviço Restaurar os Dados do PIM com SYNC não está disponível se o dispositivo não suportar o Perfil SYNC.

Cópias de segurança das informações de contactos pessoais utilizando o PBAP

Na janela Serviços do dispositivo Bluetooth adicionado (ver Adicionar e ligar a um novo dispositivo Bluetooth®) efetue os seguintes passos:

  1. Clique no botão Sincronizar Dados do PIM com o Acesso à Lista Telefónica. A Caixa de diálogo da cópia de segurança da lista de contactos será apresentada.
  2. Selecione as localizações da lista de contactos (memória interna do telefone, memória SIM) das quais pretende efetuar cópias de segurança e dos ficheiros de destino.
  3. Prima o botão Cópia de Segurança para iniciar a operação de cópia de segurança PBAP.
  4. O progresso da operação de cópia de segurança PBAP será apresentado na janela A Receber.

Nota: a operação de cópia de segurança PBAP não inicia se não houver localizações de lista de contactos selecionadas.

Nota: não é possível efetuar cópias de segurança de localizações de listas de contactos para o mesmo ficheiro de destino.
