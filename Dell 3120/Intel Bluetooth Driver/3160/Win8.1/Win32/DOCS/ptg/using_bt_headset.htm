Ligar a auriculares e altifalantes Bluetooth®

Nota: esta secção não é aplicável se estiver a utilizar o sistema operativo Windows 8.

Pode utilizar auriculares estéreo e mono Bluetooth para ouvir música e atender chamadas VoIP no seu computador. O computador também pode transformado num altifalante Bluetooth para ser utilizado por um dispositivo Bluetooth.

Os perfis de áudio suportados incluem:

  • Sink e origem A2DP
  • Controlador e destino AVRCP
  • Gateway de áudio HFP/HSP
  • Integração com o Windows Media Player/Center
  • Integração com o Skype

Ouvir música no computador através de um auricular ou altifalante Bluetooth

Pode ligar auriculares estéreo Bluetooth ao computador para ouvir música e controlar software leitor áudio em execução no computador. Quando está ligado, o dispositivo Áudio Estéreo Bluetooth pode: parar/pausar/continuar a reprodução, mudar para a faixa seguinte/anterior e aumentar/reduzir o volume do leitor áudio em execução no computador.

Na janela Serviços do dispositivo Bluetooth adicionado (consulte Adicionar e ligar a um novo dispositivo Bluetooth®):

  1. Clique na hiperligação Ouvir Música na janela Serviços para estabelecer uma ligação.
  2. A mensagem da janela Serviços é atualizada após o estabelecimento da ligação e o auricular ou altifalante Bluetooth torna-se o dispositivo de áudio predefinido do sistema operativo para ouvir música reproduzida pelo computador.
  3. Se não conseguir ouvir música com o seu auricular Bluetooth ligado, verifique se as definições do Dispositivo de áudio predefinido estão conforme descritas na secção Dispositivo de áudio predefinido abaixo.
  4. Para desligar uma ligação áudio existente, clique em Desligar no item Ouvir Música na janela Serviços e o dispositivo de áudio predefinido do sistema operativo será alterado para a sua definição original.

Dispositivo de áudio predefinido

Quanto é estabelecida uma ligação áudio, o dispositivo áudio Bluetooth é definido automaticamente como o dispositivo de reprodução predefinido no sistema operativo do Windows. Em algumas condições excepcionais, o dispositivo áudio Bluetooth predefinido poderá não funcionar automaticamente. Por exemplo, se definir manualmente outro dispositivo áudio como dispositivo áudio predefinido antes de ser estabelecida a ligação com o dispositivo áudio Bluetooth, o dispositivo áudio Bluetooth poderá não funcionar. Neste caso, aceda às propriedades de som do Windows a partir do menu do tabuleiro do sistema ou de Gerir dispositivos de áudio da secção de Hardware e Som do Painel de Controlo, e defina o dispositivo áudio Bluetooth como dispositivo predefinido.

Utilizar um auricular Bluetooth para chamadas VoIP

Pode utilizar auriculares mono Bluetooth como dispositivos de comunicação áudio para programas de telefonia VoIP.

Na janela Serviços do dispositivo Bluetooth adicionado (consulte Adicionar e ligar a um novo dispositivo Bluetooth®):

  1. Clique na hiperligação Reencaminhar Chamadas VoIP para o Mãos-livres na janela Serviços.
  2. A mensagem da janela Serviços é atualizada após o estabelecimento da ligação e o auricular torna-se o dispositivo áudio de comunicação predefinido para programas VoIP como o Skype.
  3. Para desligar uma ligação áudio existente, clique em Desligar no item Reencaminhar Chamadas VoIP para o Mãos-livres na janela Serviços.

Receber chamadas VoIP (Voice over IP) enquanto ouve música

Os auriculares estéreo e mono Bluetooth podem estar ligados continuamente para aceitar ou rejeitar chamadas do Skype utilizando o(s) botão(ões) do dispositivo áudio Bluetooth enquanto ouve música. A música entra em pausa automaticamente durante uma conversa ativa de uma chamada do Skype e irá recomeçar após o fim da chamada.

Nota: pode permitir que estas funções estejam disponíveis continuamente ativando o Plug-in do Bluetooth no Windows Media Player e garantindo que o acesso ao Skype é permitido com o plug-in “btplayerctrl.exe”.

Reproduzir Música a partir do Dispositivo Bluetooth através dos Altifalantes do Computador

Reproduza áudio a partir de um dispositivo Bluetooth utilizando os altifalantes do computador.

Na janela Serviços do dispositivo Bluetooth adicionado (consulte Adicionar e ligar a um novo dispositivo Bluetooth®):

  1. Clique na hiperligação Reproduzir Música na janela Serviços.
  2. Após a conclusão desta operação, a transmissão áudio do dispositivo remoto Bluetooth é reencaminhada para os altifalantes do seu computador (ou outro dispositivo áudio predefinido configurado no computador).
  3. Com uma ligação Reproduzir Música ativa, é possível utilizar a janela Controlo Remoto Bluetooth para controlar o leitor multimédia remoto utilizando o Bluetooth. O ícone Controlo Remoto Bluetooth irá aparecer no tabuleiro do sistema. A janela Controlo Remoto Bluetooth será apresentada se clicar nesse ícone. A janela suporta ações padrão como Reproduzir/Parar, faixa Seguinte/Anterior e controlos para aumentar/reduzir o volume.
  4. Para parar de utilizar o computador como um altifalante Bluetooth, clique na hiperligação Desligar na janela Serviços.
