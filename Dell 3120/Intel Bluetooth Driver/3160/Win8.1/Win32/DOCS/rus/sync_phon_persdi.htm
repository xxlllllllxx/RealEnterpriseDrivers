Синхронизация личной информации через Bluetooth®

Многие телефоны и другие устройства Bluetooth предоставляют доступ к списку контактов и другим средствам управления персональной информацией (PIM) с помощью службы синхронизации Bluetooth для резервного копирования контактов с устройства Bluetooth в файл, а также восстановления контактов из файла резервной копии.

Поддерживаемые профили и роли:

  • Клиент SYNC для резервного копирования и восстановления информации
  • Клиент Phone Book Access Profile (PBAP) для резервного копирования контактов

Резервное копирование и восстановление персональной информации контактов при помощи профиля SYNC

Примечание. Резервное копирование и восстановление на устройстве с резервной копии возможны только для полной базы контактных данных.

В окне Службы добавленного устройства Bluetooth (см. Добавление и подключение нового устройства Bluetooth®) выполните одно из следующих действий:

  1. Для резервного копирования щелкните кнопку Создать резервную копию данных PIM с использованием SYNC, выберите имя файла резервной копии контактов и щелкните Сохранить в окне Сохранить как файл. Запустится процесс резервного копирования PIM с отображением хода выполнения. После того как данные будут сохранены в выбранный файл, появится сообщение о завершении.
  2. Для восстановления щелкните кнопку Восстановить данные PIM с использованием SYNC, выберите ранее сохраненный файл и щелкните Открыть в окне Открыть. Запустится процесс восстановления PIM с отображением хода выполнения. После того как данные будут восстановлены на устройство Bluetooth, появится сообщение о завершении.
    Примечание. Служба Восстановить данные PIM с использованием SYNC будет недоступна, если устройство не поддерживает профиль SYNC.

Резервное копирование персональной информации контактов при помощи PBAP

В окне служб добавленного Bluetooth-устройства (см. Добавление и подключение нового устройства Bluetooth®) выполните следующие действия.

  1. Нажмите кнопку Синхронизировать PIM-данные с помощью Phone Book Access. Появится Диалоговое окно «Резервное копирование адресной книги».
  2. Выберите местоположения адресной книги (внутренняя память телефона, память SIM-карты) для резервного копирования, а также файлы назначения.
  3. Нажмите кнопку Резервное копирование, чтобы запустить процесс резервного копирования PBAP.
  4. Ход выполнения резервного копирования PBAP будет отображаться в окне Приёма.

Примечание. Процесс резервного копирования PBAP не будет инициирован при отсутствии выбранных местоположений адресной книги.

Примечание. Резервное копирование двух местоположений адресной книги в один и тот же файл назначения невозможно.
