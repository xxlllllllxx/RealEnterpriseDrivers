Подключение гарнитур и динамиков Bluetooth®

Примечание. Этот раздел не применим к операционной системе Windows 8.

Стерео и моногарнитуры Bluetooth можно использовать для прослушивания музыки и ответа на голосовые вызовы VoIP с компьютера. Также компьютер можно использовать в качестве динамика Bluetooth для устройства Bluetooth.

Supported audio profiles include:

  • A2DP, источник и приемник;
  • AVRCP, контроллер и объект воздействия;
  • HFP/HSP, аудиошлюз;
  • интеграцию с проигрывателем Windows Media и программой Windows Media Center;
  • интеграцию с программой Skype.

Прослушивание музыки с компьютера через гарнитуру или динамик Bluetooth

Стереогарнитуры Bluetooth можно подключать к компьютеру для прослушивания музыки и управления аудиоплеером, запущенным на компьютере. Когда стереоустройство Bluetooth подключено, оно может останавливать/приостанавливать/возобновлять воспроизведение, переключаться на следующую/предыдущую запись и увеличивать/уменьшать громкость аудиоплеера, запущенного на компьютере.

В окне Службы добавленного устройства Bluetooth (см. Добавление и подключение нового устройства Bluetooth®) выполните следующее:

  1. Для установки соединения, щелкните Слушать музыку в окне Службы.
  2. После установки соединения сообщение в окне Службы будет обновлено, а гарнитура или динамик Bluetooth будет назначен операционной системой в качестве аудиоустройства по умолчанию для прослушивания музыки на компьютере.
  3. Если музыка не проигрывается через подключенную гарнитуру Bluetooth, убедитесь, что параметры Аудиоустройство по умолчанию соответствуют описанию в следующем разделе Аудиоустройство по умолчанию.
  4. Чтобы отключить существующее аудиоподключение, щелкните Отключить в области Слушать музыку в окне Службы. Операционная система в качестве аудиоустройства по умолчанию установит исходные параметры.

Аудиоустройство по умолчанию

После установки аудиоподключения аудиоустройство Bluetooth автоматически назначается в качестве устройства воспроизведения по умолчанию в операционной системе Windows. В очень редких случаях устройство Bluetooth может быть не назначено автоматически в качестве аудиоустройства по умолчанию. Например, если перед тем как установить подключение к аудиоустройству Bluetooth, в качестве аудиоустройства по умолчанию было вручную назначено другое устройство, аудиоустройство Bluetooth может не работать. В этом случае на Панели управления в разделе "Оборудование и звук" выберите "Управление звуковыми устройствами" или с помощью меню на панели задач откройте свойства звука Windows и настройте аудиоустройство Bluetooth в качестве устройства по умолчанию.

Использование гарнитуры Bluetooth для вызовов VoIP

Моногарнитуры Bluetooth можно использовать в качестве устройства аудиосвязи для программ телефонии VoIP.

В окне Службы добавленного устройства Bluetooth (см. Добавление и подключение нового устройства Bluetooth®) выполните следующее:

  1. В окне Службы щелкните Перенаправлять VoIP-звонки на устройство громкой связи.
  2. После установки соединения сообщение в окне Службы будет обновлено, а гарнитура будет назначена в качестве устройства аудиосвязи по умолчанию для программ VoIP, таких как Skype.
  3. Чтобы отключить существующее аудиоподключение, щелкните Отключить в области Перенаправлять VoIP-звонки на устройство громкой связи в окне Службы.

Прием голосовых вызовов (VoIP) во время прослушивания музыки

Стерео и моногарнитуры Bluetooth можно удобно переключать для приема или сброса входящих вызовов Skype с помощью кнопок аудиоустройства Bluetooth во время прослушивания музыки. Во время активного разговора Skype воспроизведение музыки автоматически приостанавливается и возобновляется после завершения разговора.

Примечание. Чтобы активировать эти удобные функции, в проигрывателе Windows Media включите подключаемый модуль Bluetooth и убедитесь, что в Skype разрешен доступ к модулю btplayerctrl.exe.

Воспроизведение музыки с устройства Bluetooth через динамики компьютера

Настройте воспроизведение музыки с устройства Bluetooth через динамики компьютера.

В окне Службы добавленного устройства Bluetooth (см. Добавление и подключение нового устройства Bluetooth®) выполните следующее:

  1. В окне Службы щелкните ссылку Проигрывать музыку.
  2. После этого потоковая передача аудио с удаленного устройства Bluetooth будет перенаправлена на динамики компьютера (или другое аудиоустройство по умолчанию, настроенное на компьютере).
  3. Когда подключение Проигрывать музыку активно, в окне Удаленное управление Bluetooth можно управлять удаленным мультимедийным проигрывателем с помощью Bluetooth. На панели задач появится значок Удаленное управление Bluetooth. Если щелкнуть этот значок, откроется окно Удаленное управление Bluetooth. Это окно поддерживает стандартные действия, такие как воспроизведение/остановка, следующая/предыдущая запись и увеличение/уменьшение громкости.
  4. Чтобы отменить использование компьютера в качестве динамика Bluetooth, в окне Службы щелкните ссылку Отключить.
