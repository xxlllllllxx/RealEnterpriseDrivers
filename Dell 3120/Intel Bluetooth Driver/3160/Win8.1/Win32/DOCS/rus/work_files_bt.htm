Передача файлов по Bluetooth®

Пользователь может предоставлять открытый доступ к локальным файлам и папкам на своем компьютере другим Bluetooth-устройствам, передавать любые типы файлов с компьютера на Bluetooth-устройства и обратно, а также создавать или удалять файлы и папки на Bluetooth-устройстве и использовать Мастер отправки файлов в меню Microsoft Office для отправки файлов на Bluetooth-устройства.

Поддерживаемые профили и роли:

  • FTP (File Transfer Profile) (клиент и сервер)
  • Базовый профиль обработки изображений (BIP) (инициатор/ответчик)
  • Профиль передачи объектов (OPP) (клиент/сервер)

Поддерживаемые функции:

  • Протокол Object Exchange (OBEX) 1.5, режим одного ответа (SRM)
  • Интеграция Microsoft Office для клиента Object Push Profile (OPP)

Предоставление доступа к локальной папке для других Bluetooth-устройств

Другим устройствам Bluetooth можно предоставить доступ к папкам на компьютере. Для этого выполните действия, описанные в разделе Параметры общего доступа к папкам этого руководства пользователя.

Открытая для доступа папка используется в качестве корневой папки для передачи файлов через FTP, а также в качестве папки по умолчанию для приема файлов через OPP и BIP.

Просмотр открытой для доступа папки на удаленном Bluetooth-устройстве

При подключении к Bluetooth-устройству, поддерживающему протокол FTP, может быть использовано стандартное окно проводника для обычных операций с файлами между устройством и компьютером, таких как:

  • Просмотр файлов и папок
  • Передача файлов и папок
  • Удаление файлов и папок
  • Переименование файлов и папок
    Примечание. Если удаленное устройство поддерживает роль FTP-сервера более ранней версии, чем 1.2, переименование доступно только для пустых папок.
  • Создание папок

Чтобы получить доступ к открытой для доступа папке на удаленном устройстве, находясь в окне службы:

  1. Откройте окно служб устройства (см. Добавление и подключение нового устройства Bluetooth®).
  2. Нажмите Просмотреть удаленную папку.

Чтобы получить доступ к открытой для доступа папке на удаленном устройстве, находясь в окне проводника Windows:

  1. Откройте папку Компьютер.
  2. Щелкните значок с именем удаленного устройства.

Примечание. Если удаленное устройство поддерживает роль FTP-сервера временно (т.е., роль FTP-сервера можно включить или отключить), то ссылка в папке Компьютер может отсутствовать. В этом случае используйте окно служб удаленного устройства.

Примечание. В процессе обмена данными удаленное устройство может запросить пароль. В этом случае необходимо ввести одинаковый пароль на обоих устройствах для установления соединения. Если пароли не совпадают, соединение будет прервано.

Отправить файлы на Bluetooth-устройство

Помощник по отправке файлов на Bluetooth-устройство используется для отправки файлов на удаленное Bluetooth-устройство. Один помощник позволяет отправить только один файл, однако при необходимости можно запустить несколько помощников одновременно. Ниже приведены пошаговые инструкции по выбору конечного удаленного устройства и файла для отправки. Помощник по отправке файлов на Bluetooth-устройство может использоваться в нескольких сценариях.

Примечание. Обычно BIP используется для отправки изображений с мобильного устройства на компьютер. OPP используется для отправки всех остальных типов файлов. Помощник по отправке файлов на Bluetooth-устройство поддерживает оба профиля.

Отправка файлов из Microsoft Office на устройство Bluetooth

Поддерживаются следующие версии Microsoft Office:

  • Microsoft Office 2010
  • Microsoft Office 2013

Поддерживаются следующие приложения и элементы Microsoft Office:

  • Word: документ
  • Excel: книга
  • PowerPoint: презентация
  • Access: база данных
  • Outlook: встреча, контакт, сообщение, заметка, задача

Перед отправкой на удаленное Bluetooth-устройство элемент Microsoft Office автоматически сохраняется.

Примечание. Если элемент Microsoft Office еще не был сохранен, в Word, Excel и PowerPoint отобразится запрос на сохранение.

Примечание. Если элемент Microsoft Office для отправки является только что созданной базой данных, то приложение Access должно быть перезапущено.

Чтобы отправить элемент Microsoft Office на удаленное Bluetooth-устройство, выполните следующее:

  1. В зависимости от используемой версии Microsoft Office и приложения Microsoft Office, выберите пункт Отправить на Bluetooth в подменю Общий доступ, Сохранить и отправить, или Сохранить и опубликовать. Обратитесь к справке Microsoft Office за инструкциями. Появится Помощник по отправке файлов на Bluetooth-устройство.
  2. После открытия Помощника по отправке файлов на Bluetooth-устройство выберите Bluetooth-устройство, на которое требуется отправить файл и нажмите кнопку Далее.
  3. Появится окно с ходом передачи файла, и элемент будет отправлен на выбранное Bluetooth-устройство. По окончании передачи файла вы увидите уведомление.

Отправка файлов из проводника Windows на устройство Bluetooth

Для отправки файлов из проводника Windows на устройство Bluetooth выполните следующее:

  1. В окне проводника Windows щелкните правой кнопкой мыши по значку файла и выберите Bluetooth в элементе контекстного меню Отправить. Появится Помощник по отправке файлов на Bluetooth-устройство.
  2. После открытия Помощника по отправке файлов на Bluetooth-устройство выберите Bluetooth-устройство, на которое требуется отправить файл и нажмите кнопку Далее.
  3. Откроется окно хода выполнения передачи файлов, и выбранный файл будет отправлен на выбранное устройство Bluetooth. После завершения передачи появится уведомление.

Отправка файлов из окна Службы на устройство Bluetooth

Для отправки файлов из окна Службы на устройство Bluetooth выполните следующее:

  1. В окне Службы щелкните ссылку Отправить файл. Появится диалоговое окно "Открыть".
  2. Выберите файл из списка и нажмите кнопку Открыть. Появится Помощник по отправке файлов на Bluetooth-устройство.
  3. После открытия Помощника по отправке файлов на Bluetooth-устройство нажмите кнопку Далее.
  4. Откроется окно хода выполнения передачи файлов, и выбранный файл будет отправлен на выбранное устройство Bluetooth. После завершения передачи появится уведомление.

Получение файлов с устройства Bluetooth

При передаче файлов на ваш компьютер другим Bluetooth-устройством с помощью BIP или OPP появится окно Приём файлов; для передаваемых файлов используется заданный по умолчанию конечный путь.

Примечание. Обычно BIP используется для отправки изображений с мобильного устройства на компьютер. OPP используется для отправки всех остальных типов файлов.

В окне "Прием файлов" можно изменить папку назначения, в которой по умолчанию будут сохраняться входящие файлы и изображения. Для этого выполните следующие действия.

  1. Щелкните Выбрать и выберите нужную папку.
  2. Выбрав путь, щелкните Принять, чтобы начать передачу файла.
  3. Для отмены получения входящего файла щелкните Отклонить.

Изменение файлов и папок с Bluetooth-устройства

Bluetooth-устройство с поддержкой профиля FTP допускает изменение файлов и папок в локальной папке общего доступа. Если удаленное устройство предпримет попытку перезаписать, переименовать или удалить содержимое открытой для доступа папки, будет запрошено специальное разрешение. Перед выполнением этих действий появится окно Запрос на разрешение. В окне Запрос на разрешение доступны следующие варианты:

  • Выберите Разрешить только один раз и нажмите ОК, чтобы подтвердить изменения с Bluetooth-устройства временно.
  • Выберите Всегда разрешать для этого устройства и нажмите ОК, чтобы всегда подтверждать изменения с Bluetooth-устройства, пока оно сопряжено.
  • Чтобы временно отклонить изменения с Bluetooth-устройства, нажмите Отмена.

Если передача выбранного файла не влечет за собой изменение какого-либо существующего файла, окно Запрос на разрешение не появится во время передачи.

Выполнение приема файлов

При инициации передачи файла с другого Bluetooth-устройства на компьютер и приеме этой передачи может появиться окно Идет прием:

  • Если удаленное устройство является клиентом BIP или OPP, появится окно Идет прием.
  • Если удаленное устройство является клиентом FTP, появление окна Идет прием можно настроить при помощи последовательности действий, описанной в разделе Параметры общего доступа к папкам данного руководства пользователя.

В окне Идет прием отображается следующая информация о каждом файле:

  • Состояние передачи (получено успешно, не получено, выполняется передача и т.д.)
  • Скорость передачи данных в бодах для файлов, которые передаются в текущий момент
  • Размер успешно полученных файлов
