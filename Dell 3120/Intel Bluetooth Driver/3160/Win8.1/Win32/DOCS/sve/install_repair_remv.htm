Installation, reparation, avinstallation och uppgradering

Installation

Starta installationen genom att aktivera maskinvaran och starta Setup.exe från installationsskivan. Då startar Installationsguiden.

Följ stegen som anges i installationsguiden:

  1. Klicka på Nästa på Välkommen.
  2. Läs licensavtalet och godkänn det på sidan Licensavtal. Klicka sedan på Nästa.
  3. Byt vid behov målmappen på sidan Målmapp.
    Standardinstallationen är “C:\Program Files\Intel\Bluetooth”.
  4. På sidan Redo att installera programmet startar du installationsprocessen genom att klicka på Installera. Installationsförloppet visas på skärmen.
  5. När installationsprocessen är klar, klicka på Slutför.
  6. Efter installationen kan programmet startas från Bluetooth®-ikonmenyn i systemfältet.

Reparation

Gör så här för att utföra en reparation:

  1. Öppna Kontrollpanelen.
  2. Klicka på länken Ta bort program i gruppen Program (eller dubbelklicka på ikonen Program och funktioner) i Kontrollpanelen.
  3. Markera Intel® PROSet/Wireless Software for Bluetooth® Technology i programlistan och dubbelklicka på den.
  4. Klicka på Nästa i fönstret Installationsguiden.
  5. Klicka på Reparera och följ anvisningarna.
  6. Slutför reparationen genom att starta om datorn. Alla ändringar verkställs när omstarten har slutförts.

Borttagning

Gör så här för att avinstallera:

  1. Öppna Kontrollpanelen.
  2. Klicka på länken Ta bort program i gruppen Program (eller dubbelklicka på ikonen Program och funktioner) i Kontrollpanelen.
  3. Markera Intel® PROSet/Wireless Software for Bluetooth® Technology i programlistan och dubbelklicka på den.
  4. Klicka på Nästa i fönstret Installationsguiden.
  5. Klicka på Ta bort och följ anvisningarna.
  6. Slutför borttagningen genom att starta om datorn. Bluetooth-programmet tas bort när omstarten har slutförts.

Uppgradering

Uppgradera på något av följande sätt:

  • Kör det installationspaket som har den senaste versionen av programvaran och följ anvisningarna i Guiden.
  • Avinstallera den aktuella versionen av programvaran, starta om datorn och kör det installationspaket som har den senaste versionen av programvaran.
