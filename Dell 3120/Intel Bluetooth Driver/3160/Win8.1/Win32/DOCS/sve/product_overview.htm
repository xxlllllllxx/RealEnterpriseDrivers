Produktbeskrivning

Översikt

Med Intel® PROSet/Wireless Software for Bluetooth® Technology kan man ansluta en mängd Bluetooth-enheter trådlöst till datorn, t.ex. mobiltelefoner, öronsnäckor, stereoheadset, smarta handdatorer, skrivare, digitalkameror och mycket annat. Med Bluetooth-programmet kan du dela filer, lyssna på musik, ringa samtal och använda Bluetooth-skrivare.

Intel® PROSet/Wireless Software for Bluetooth® Technology gör det enkelt och intuitivt att ansluta med hjälp av Bluetooth. Det går att spåra alla anslutna enheter och ansluta till dem automatiskt, men det går även att justera manuellt när det behövs.

Mer information om vilka operativsystem och adaptrar som stöds finns i Intel® trådlösa produkter.

Specifikationer

Stödda Bluetooth-kärngränssnitt:

  • Host Controller Interface (HCI)

Stödda Bluetooth-profiler:

  • Advanced Audio Distribution Profile (A2DP) (källa/sink)1
  • Audio/Video Remote Control Profile (AVRCP) (styrenhet/mål)1
  • Basic Imaging Profile (BIP) (initierare/responder)
  • Basic Printing Profile (BPP) (sändare)
  • File Transfer Profile (FTP) (klient/server)
  • Allmän åtkomstprofil (GAP, Generic Access Profile)
  • Allmän attributprofil (GATT, Generic Attribute Profile)
  • GAVDP (Generic Audio/Video Distribution Profile) (källa/kanalmottagare)1
  • Generic Object Exchange Profile (GOEP) (klient/server)
  • Handsfree-profil (HFP) (ljudgateway) med stöd för bredbandstal (WBS, Wide-Band Speech)1
  • HCRP (Hardcopy Cable Replacement Profile) (klient)
  • Headset Profile (HSP) (ljudgateway)1
  • HID over GATT-profil (HOGP) (värd), även kallad Low Energy HID-profil2
  • OPP (Object Push Profile) (klient/server)
  • PBAP (Phone Book Access Profile) (klient)
  • Synchronization Profile (SYNC) (klient)

Operativsystem:

  • Windows Vista (32-bit och 64-bit) SP23
  • Windows 7 (32-bit och 64-bit)
  • Windows 8 (32-bit och 64-bit)
  • Windows 8.1 (32-bit och 64-bit)

Intel® PROSet/Wireless Software introducerar även ett antal nya funktioner:

  • Integrering med Windows Media Player och Media Center1
  • Integrering med programvara för Skype Voice over IP (VoIP)1
  • Integrering med Microsoft Office
  • Möjlighet att inaktivera Bluetooth-radio3
  • Funktioner för flygplansläge i Windows 8 och Windows 8.1
  • Stöd för Bluetooth med låg energiförbrukning2

Lista över akronymer:

  • A2DP – Advanced Audio Distribution Profile
  • AMP – Alternate MAC/PHY
  • AVRCP – Audio/Video Remote Control Profile
  • BIP – Basic Imaging Profile
  • BPP – Basic Printing Profile
  • DRM – Digital Rights Management
  • EDR – Enhanced Data Rate
  • FTP – File Transfer Protocol
  • GAP – Generic Access Profile (allmän åtkomstprofil)
  • GATT – Generic Attribute Profile (allmän attributprofil)
  • HID – Human Interface Device (HID-enhet)
  • HFP – Hands-Free Profile
  • HOGP – HID over GATT-profil
  • HSP – Headset Profile
  • MAC/PHY – Media Access Control/Physical Layer
  • OBEX – Object Exchange
  • PBAP – Phone Book Access Profile
  • SCMS-T – Serial Copy Management System
  • SYNCH – Synchronization Profile
  • SRM – Single Response Mode
  • VoIP – Voice over IP
  • WBS – Wide-Band Speech (bredbandstal)

1 Windows 8 och Windows 8.1 har funktioner för ljudprofiler. Interaktion med mediespelare och VoIP-program är dock inte möjligt för Intel® PROSet/Wireless Software for Bluetooth® Technology från Windows 8 och senare.

2 Funktionen Bluetooth med låg energiförbrukning, som beskrivs i det här dokumentet, fungerar endast med Windows 7. Windows 8 och Windows 8.1 har funktioner för Bluetooth med låg energiförbrukning, men det finns inte funktioner för Bluetooth med låg energiförbrukning för Intel® PROSet/Wireless Software for Bluetooth® Technology i Windows 8 eller senare.

3 Möjlighet att inaktivera Bluetooth-radio från systemfältet finns endast i Windows 7.
