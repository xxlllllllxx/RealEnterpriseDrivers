安装、修复、删除和升级

安装

要开始安装，请启用硬件并启动安装磁盘中的 Setup.exe。这将启动安装向导。

按照安装向导指定的步骤操作：

  1. 在欢迎页面上单击下一步。
  2. 在许可协议页面上阅读并接受许可协议。然后单击下一步。
  3. 根据需要，更改目标文件夹页上的目标文件夹。
    默认安装位置是“C:\Program Files\Intel\Bluetooth”。
  4. 在准备安装程序页面上，单击安装以开始安装流程。将显示安装流程。
  5. 在安装过程完成时，单击完成。
  6. 安装完成后，可从系统任务栏区域的蓝牙® 任务栏图标菜单中启动该应用程序。

修复

要进行修复，请执行以下步骤：

  1. 打开控制面板。
  2. 单击控制面板的程序组中的卸载程序链接（或双击程序和功能图标）。
  3. 在应用程序列表中选择并双击 Intel® PROSet/Wireless Software for Bluetooth® Technology。
  4. 在设置向导窗口单击下一步。
  5. 单击修复并按照说明操作。
  6. 要完成修复，请重启系统。所有更改将在重启后执行。

删除

要卸载，请执行以下步骤：

  1. 打开控制面板。
  2. 单击控制面板的程序组中的卸载程序链接（或双击程序和功能图标）。
  3. 在应用程序列表中选择并双击 Intel® PROSet/Wireless Software for Bluetooth® Technology。
  4. 在设置向导窗口单击下一步。
  5. 单击删除并按照说明操作。
  6. 要完成删除，请重启系统。重启系统后将删除蓝牙软件。

升级

请执行以下操作之一来进行升级：

  • 运行包含最新版软件的安装软件包，并按照向导说明操作。
  • 卸载已安装的当前版本，重启计算机，然后运行包含最新版软件的安装软件包。
