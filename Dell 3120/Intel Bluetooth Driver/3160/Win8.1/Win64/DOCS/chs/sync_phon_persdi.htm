通过蓝牙® 同步个人信息

许多电话和其他蓝牙设备提供对联系人和其他个人信息管理器 (PIM) 数据的访问，使用蓝牙同步服务将联系人从蓝牙设备备份到文件以及从备份文件中还原联系人。

支持的模式和角色：

  • 用于备份和还原联系人的 SYNC 客户端
  • 用于备份联系人的电话簿访问模式 (PBAP) 客户端

使用 SYNC 配置文件备份和恢复个人联系人信息

注意： 只有完整的通讯录数据库信息才能在设备上进行备份或恢复。

在添加的蓝牙设备的服务窗口（请参阅添加并连接到新蓝牙® 设备），执行以下操作之一：

  1. 要备份，请单击使用SYNC备份PIM数据按钮，选择联系人备份文件的名称，然后在另存为文件对话框中单击保存。PIM 备份过程启动后会在显示屏上显示进度指示器，当数据保存到选定的文件后，将显示完成消息。
  2. 要还原，请单击使用SYNC还原PIM数据按钮，选择先前保存的源文件，然后在打开对话框中单击打开。PIM 过程启动后会在显示屏上显示进度指示器，当数据还原到蓝牙设备后，将显示完成消息。
    注意：如果设备不支持 SYNC 模式，则使用SYNC还原PIM数据服务链接不可用。

使用 PBAP 备份个人联系人信息

在添加的蓝牙设备的服务窗口（请参阅添加并连接到新蓝牙® 设备），执行以下操作：

  1. 单击通过电话簿访问同步PIM数据按钮。将出现电话簿备份对话。
  2. 选择要备份的电话簿位置（手机内存、SIM 内存）和目标文件。
  3. 按下备份按钮以启动 PBAP 备份操作。
  4. PBAP 备份操作进度显示在正在接收窗口中。

注意：如果未选定电话薄位置，则不会启动 PBAP 备份操作。

注意：不能将两个电话薄位置备份到相同目标文件。
