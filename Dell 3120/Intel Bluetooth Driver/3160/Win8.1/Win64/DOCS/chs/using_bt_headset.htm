连接到蓝牙® 耳机和扬声器

注意：如果您使用的是 Windows 8 操作系统，则此部分不适用。

蓝牙立体声耳机和单声道耳机可用于听音乐和接听来自计算机的 VoIP 电话。另外，计算机还可用作蓝牙设备使用的蓝牙扬声器。

支持的音频模式包括：

  • A2DP 接收器和 A2DP 发射源
  • AVRCP 控制器和 AVRCP 目标
  • HFP/HSP 音频网关
  • 与 Windows Media Player/Center 集成
  • 与 Skype 集成

使用蓝牙耳机或扬声器从计算机听音乐

可将蓝牙立体声耳机连接到计算机来听音乐和控制计算机上运行的音频播放器软件。连接后，蓝牙立体声音频设备可停止/暂停/继续播放，转到下一个/上一个曲目以及控制计算机上运行的音频播放器的音量调整。

在添加的蓝牙设备的服务窗口（请参阅添加并连接到新蓝牙® 设备）：

  1. 在服务窗口中单击听音乐链接以建立连接。
  2. 建立连接后，服务窗口消息会更新，并且蓝牙耳机或扬声器将成为操作系统的默认音频设备来聆听计算机中播放的音乐。
  3. 如果连接后蓝牙耳机中听不到音乐，请检查默认音频设备设置是否和以下默认音频设备部分所述一致。
  4. 要断开现有的音频连接，请在服务窗口的听音乐项中单击断开，操作系统的默认音频设备将切换回原始设置。

默认音频设备

建立音频连接后，蓝牙音频设备将自动设为 Windows 操作系统中的默认播放设备。在某些极少见的情况下，默认的蓝牙音频设备可能无法自动工作。例如，如果您在与蓝牙音频设备建立连接之前将其他音频设备手动设置为默认音频设备，则蓝牙音频设备可能无法工作。在此情况下，从系统任务栏菜单中转到 Windows 的“声音”属性，或转到“控制面板”的“硬件和声音”部分的“管理音频设备”，然后将蓝牙音频设备设为默认设备。

使用蓝牙耳机进行 VoIP 通话

蓝牙单声道耳机可用作 VoIP 电话程序的音频通信设备。

在添加的蓝牙设备的服务窗口（请参阅添加并连接到新蓝牙® 设备）：

  1. 单击服务窗口中的将VoIP呼叫转接到免提听筒链接。
  2. 建立连接后，服务窗口消息会更新，并且耳机会设为 VoIP 程序（如 Skype）的默认通信音频设备。
  3. 要断开现有的音频连接，请在服务窗口的将VoIP呼叫转接到免提听筒项中单击断开。

在听音乐时接听 IP 语音 (VoIP) 来电

听音乐时可使用蓝牙音频设备的按钮无缝切换蓝牙立体声和单声道耳机来接听或拒接 Skype 来电。进行 Skype 通话时音乐会自动暂停，通话结束后继续播放。

注意：您可以通过在 Windows Media Player 中启用蓝牙插件并确保允许“btplayerctrl.exe”插件访问 Skype，使这些功能能够顺利工作。

通过计算机扬声器从蓝牙设备播放音乐

通过计算机扬声器从蓝牙设备播放音频

在添加的蓝牙设备的服务窗口（请参阅添加并连接到新蓝牙® 设备）：

  1. 单击服务窗口中的播放音乐链接。
  2. 操作完成后，从远程蓝牙设备播放的音频将被转发到计算机扬声器（或计算机上配置的其他默认音频设备）。
  3. 建立活动的播放音乐连接后，可使用蓝牙远程控制窗口控制使用蓝牙功能的远程媒体播放器。蓝牙远程控制图标将显示在系统任务栏中。单击此图标可显示蓝牙远程控制窗口。此窗口支持标准操作，如播放/停止、下一个/上一个曲目和音量调整控制。
  4. 要停止将计算机用作蓝牙扬声器，请单击服务窗口中的断开链接。
