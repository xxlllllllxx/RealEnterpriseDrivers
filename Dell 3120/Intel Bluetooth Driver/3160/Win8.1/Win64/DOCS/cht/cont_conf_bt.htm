藍牙® 管理及組態

開啟及關閉藍牙無線電

如果必須停用藍牙，有三種可行的方法：軟體切換控制、硬體切換控制，或裝置管理員。強烈建議不要混用前述三種方法 (例如：用軟體切換控制關閉藍牙，又用裝置管理員開啟)。

關閉或開啟藍牙無線電的建議方法為軟體切換控制；強烈建議您不要採用在裝置管理員中停用/啟用藍牙裝置驅動程式的方式來開關裝置。不過，並非所有系統皆可使用軟體切換控制。使用軟體切換控制的流程因作業系統而異，請見下列所述。

部分系統可能以硬體切換控制取代軟體切換控制，或兩者並存。當兩個切換控制並存時，請避免同時使用 (意即如果使用軟體切換控制關閉藍牙無線電，即使重複使用硬體切換控制開關，藍牙無線電依然維持關閉狀態)。請參閱系統使用手冊，瞭解硬體切換控制的相關資訊。

若要在 Windows7 中切換藍牙無線電開關：

  1. 以滑鼠右鍵按一下藍牙系統匣圖示。
  2. 選擇開啟介面卡或關閉介面卡。

此外，也可以透過藍牙「設定」視窗關閉藍牙無線電：

  1. 以滑鼠右鍵按一下藍牙工具列圖示。
  2. 選擇開啟設定。
  3. 在選項索引標籤中，勾選關閉 Bluetooth 介面卡。

若要在 Windows 8 與其後的作業系統版本中切換藍牙無線電開關：

  1. 開啟「電腦設定」
    • 將滑鼠游標移動到右下角以顯示「快速鍵」功能表，然後選擇設定並按一下變更電腦設定。
  2. 選擇無線節點。
  3. 按一下藍牙滑桿即可反覆開啟和關閉藍牙。

資料夾共用設定

若要啟用或停用檔案共用：

  1. 以滑鼠右鍵按一下藍牙系統匣圖示，並選擇開啟設定。
  2. 選擇藍牙設定視窗中的共用資料夾索引標籤，然後執行下列其中一項任務：
    • 若要啟用檔案共用，請選擇允許遠端裝置瀏覽共用資料夾核取方塊，接著按一下瀏覽以選擇要與藍牙裝置共用的本機資料夾，然後按一下確定。
    • 若要停用檔案共用，請取消選擇允許遠端裝置瀏覽共用資料夾核取方塊。

若要啟用或停用透過 FTP 上傳檔案相關的使用者通知：

  1. 以滑鼠右鍵按一下藍牙系統匣圖示，並選擇開啟設定。
  2. 選擇「藍牙設定」視窗中的共用資料夾索引標籤，然後執行下列其中一項任務：
    • 若要啟用使用者通知，請選擇上傳檔案至共用資料夾時請通知我核取方塊，然後按一下「確定」。
    • 若要停用使用者通知，請取消選擇上傳檔案至共用資料夾時請通知我核取方塊。

音效設定

注意：如果您是使用 Windows 8 作業系統，則本節不適用。

若要管理藍牙音訊設定：

  1. 以滑鼠右鍵按一下藍牙系統匣圖示，並選擇開啟設定。
  2. 選擇藍牙設定視窗中的音效索引標籤，然後執行下列一或多項任務：
    • 若僅允許與支援 SCMS-T 數位版權管理 (DRM) 內容保護的裝置進行藍牙音訊連線，請選擇僅與支援 SCMS-T 內容保護的音效裝置連線核取方塊。
    • 若要在收聽音樂時啟用自動 Skype 來電切換，請選擇啟用 Skype 切換選項，並從下拉式功能表選擇先前連接的藍牙音訊耳機。如需詳細資訊，請參閱收聽音樂期間接聽網際網路語音協定 (VoIP) 來電。如果並未在電腦上安裝 Skype 軟體，則無法使用此選項。
