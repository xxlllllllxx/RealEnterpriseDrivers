產品說明

概觀

Intel® PROSet/Wireless Software for Bluetooth® Technology 可讓您使用各種藍牙裝置 (例如，行動電話、耳塞式耳機、立體聲耳機、智慧型手持式裝置、印表機、數位相機等等) 與電腦進行無線連線。您可以使用藍牙軟體來共用檔案、收聽音樂、撥打語音電話以及使用藍牙印表機。

Intel® PROSet/Wireless Software for Bluetooth® Technology 可追蹤所有連線裝置並自動連線至這些裝置，因此使用藍牙連線既簡單且直覺化，而且仍可視需要手動調整。

如需支援的作業系統及網卡的詳細資訊，請參閱 Intel® 無線產品。

規格

支援的藍牙核心介面：

  • 主機控制介面 (HCI)

支援的藍牙規範：

  • 立體聲傳輸規範 (A2DP) (來源/接收)1
  • 音訊/視訊遠端控制設定檔 (AVRCP) (控制器/目標)1
  • 基本圖像設定檔 (BIP) (發送方/回應方)
  • 基本列印規範 (BPP) (傳送端)
  • 檔案傳輸規範 (FTP) (用戶端/伺服器)
  • 一般存取設定檔 (GAP)
  • 一般屬性設定檔 (GATT)
  • 通用音訊/視訊傳輸設定檔 (GAVDP) (來源/接收)1
  • 通用物件交換規範 (GOEP) (用戶端/伺服器)
  • 具備寬頻語音支援 (WBS) 的免持聽筒設定檔 (HFP) (音訊閘道)1
  • 實體纜線更換設定檔 (HCRP) (用戶端)
  • 耳機規範 (HSP) (音頻閘道)1
  • HID over GATT 設定檔 (HOGP) (主機)，也稱為低耗能 HID 設定檔2
  • 物件發送設定檔 (OPP) (用戶端/伺服器)
  • 電話簿存取設定檔 (PBAP) (用戶端)
  • 同步規範 (SYNC) (用戶端)

作業系統：

  • Windows Vista (32 位元和 64 位元) SP23
  • Windows 7 (32 位元和 64 位元)
  • Windows 8 (32 位元和 64 位元)
  • Windows 8.1 (32 位元和 64 位元)

Intel® PROSet/Wireless Software 亦加入了下列新功能：

  • 整合 Windows Media Player 及媒體中心1
  • 與 Skype 網際網路語音協定 (VoIP) 軟體整合1
  • 與 Microsoft Office 整合
  • 可關閉藍牙無線電3
  • Windows 8 和 Windows 8.1 上的飛航模式支援
  • 藍牙低耗能支援2

縮寫清單：

  • A2DP - 立體聲傳輸規範
  • AMP - 替代 MAC/PHY
  • AVRCP - 音效/影像遠端控制規範
  • BIP - 基本影像規範
  • BPP - 基本列印規範
  • DRM - 數位版權管理
  • EDR - 增強資料傳輸率
  • FTP - 檔案傳輸協定
  • GAP - 一般存取設定檔
  • GATT - 一般屬性設定檔
  • HID - 人性化介面裝置
  • HFP - 免持聽筒規範
  • HOGP - HID over GATT 設定檔
  • HSP - 耳機規範
  • MAC/PHY - 媒體存取控制/實體層
  • OBEX - 物件交換
  • PBAP - 電話簿存取規範
  • SCMS-T - 序列化副本管理系統
  • SYNCH - 同步規範
  • SRM - 單一回應模式
  • VoIP - 網際網路語音協定
  • WBS - 寬頻語音

1 Windows 8 和 Windows 8.1 原生支援音訊設定檔。但是在 Windows 8 與其後的作業系統版本上，與媒體播放器和 VoIP 軟體的互動則超出 Intel® PROSet/Wireless Software for Bluetooth® Technology 的範圍。

2 本文件描述的藍牙低耗能功能僅適用於 Windows 7。Windows 8 和 Windows 8.1 原生支援藍牙低耗能功能，但是在 Windows 8 與其後的作業系統版本上，藍牙低耗能支援則超出 Intel® PROSet/Wireless Software for Bluetooth® Technology 的範圍。

3 僅限 Windows 7 版本可從系統匣功能表關閉藍牙無線電。
