連線至藍牙® 低耗能 HID 裝置

注意：本文件描述的藍牙低耗能 HID 功能僅適用於 Windows 7。Windows 8 原生支援藍牙低耗能，但是 Windows 8 上的藍牙低耗能 HID 支援超出 Intel® PROSet/Wireless Software for Bluetooth® Technology 的範圍。

新增藍牙低耗能 HID 裝置之後 (請參閱新增並連線至新的藍牙® 裝置)，將會自動建立與藍牙低耗能 HID 裝置的連線。如果連線突然中斷，則軟體會嘗試重新建立連線。

檢查低耗能 HID 裝置的電池狀態

當任何已連線之低耗能 HID 裝置的電池充電量低於 10% 時，「電池狀態」圖示會顯示在系統匣中。

若要檢查低耗能 HID 裝置的電池狀態：

  1. 按一下系統匣中的電池狀態圖示。
  2. 隨即還會顯示飛出視窗，其中並列出低耗能 HID 裝置清單及其電池狀態。電池狀態僅會在已連線的低耗能 HID 裝置電池充電量低於 10% 時顯示。
  3. 按一下飛出視窗外部即可隱藏該視窗。
