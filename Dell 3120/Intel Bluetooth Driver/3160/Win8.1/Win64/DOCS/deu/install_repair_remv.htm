Installation, Reparatur, Deinstallation und Upgrade

Installation

Um mit der Installation zu beginnen, aktivieren Sie die Hardware, und führen Sie die Datei Setup.exe aus, die sich auf dem Installationsdatenträger befindet. Daraufhin wird der Installationsassistent gestartet.

Befolgen Sie die Schritte des Installationsassistenten:

  1. Klicken Sie auf der Seite Willkommen auf Weiter.
  2. Lesen Sie sich die Lizenzvereinbarung durch, und akzeptieren Sie sie auf der Seite Lizenzvereinbarung. Klicken Sie dann auf Weiter.
  3. Falls erforderlich, ändern Sie auf der Seite Zielordner das Zielordner.
    Das Standardverzeichnis lautet „C:\Program Files\Intel\Bluetooth“.
  4. Klicken Sie auf der Seite Installation des Programms jetzt bereit auf Installieren, um den Installationsprozess zu starten. Der Fortschritt der Installation wird angezeigt.
  5. Wenn die Installation abgeschlossen ist, klicken Sie auf Fertigstellen.
  6. Nach der Installation kann die Anwendung über das Bluetooth®-Taskleistensymbol in der Taskleiste gestartet werden.

Reparatur

Gehen Sie zum Reparieren wie folgt vor:

  1. Öffnen Sie die Systemsteuerung.
  2. Klicken Sie in der Systemsteuerung in der Gruppe Programme auf den Link Programm deinstallieren (oder doppelklicken Sie auf das Symbol Programme und Funktionen).
  3. Markieren Sie in der Anwendungsliste Intel® PROSet/Wireless Software for Bluetooth® Technology, und doppelklicken Sie darauf.
  4. Klicken Sie im Fenster Setup-Assistent auf Weiter.
  5. Klicken Sie auf Reparieren, und folgen Sie den Anweisungen.
  6. Um den Reparaturvorgang abzuschließen, führen Sie einen Neustart durch. Alle Änderungen werden erst nach dem Neustart wirksam.

Deinstallation

Gehen Sie für eine Deinstallation wie folgt vor:

  1. Öffnen Sie die Systemsteuerung.
  2. Klicken Sie in der Systemsteuerung in der Gruppe Programme auf den Link Programm deinstallieren (oder doppelklicken Sie auf das Symbol Programme und Funktionen).
  3. Markieren Sie in der Anwendungsliste Intel® PROSet/Wireless Software for Bluetooth® Technology, und doppelklicken Sie darauf.
  4. Klicken Sie im Fenster Setup-Assistent auf Weiter.
  5. Klicken Sie auf Entfernen, und folgen Sie den Anweisungen.
  6. Um die Deinstallation abzuschließen, führen Sie einen Neustart durch. Die Bluetooth-Software wird erst nach dem Neustart entfernt.

Upgrade

Führen Sie für ein Upgrade eine der folgenden Aktionen aus:

  • Führen Sie das Installationspaket mit der neuesten Version der Software aus, und folgen Sie den Anweisungen des Assistenten.
  • Deinstallieren Sie die aktuelle Version der installierten Software, starten Sie den PC neu, und führen Sie das Installationspaket mit der neuesten Version der Software aus.
