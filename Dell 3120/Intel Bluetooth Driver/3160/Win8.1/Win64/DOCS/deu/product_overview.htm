Produktbeschreibung

Übersicht

Intel® PROSet/Wireless Software for Bluetooth® Technology ermöglicht einer Vielzahl von Bluetooth-Geräten wie Mobiltelefonen, Ohrhörern, Stereo-Headsets, Handheldgeräten, Druckern, Digitalkameras usw. die drahtlose Verbindung zu einem Computer. Mit der Bluetooth-Software können Sie Dateien freigeben, Musik hören, Telefonanrufe tätigen und Bluetooth-Drucker verwenden.

Das Herstellen von Bluetooth-Verbindungen mit Intel® PROSet/Wireless Software for Bluetooth® Technology ist einfach und leicht nachvollziehbar. Alle angeschlossenen Geräte lassen sich überwachen und automatisch verbinden, bei Bedarf sind jedoch auch manuelle Anpassungen möglich.

Weitere Informationen zu den unterstützten Betriebssystemen und Adaptern finden Sie unter Intel® Drahtlosprodukte.

Spezifikationen

Unterstützte Bluetooth-Schnittstellen:

  • Host Controller Interface (HCI)

Unterstützte Bluetooth-Profile:

  • Advanced Audio Distribution-Profil (A2DP) (Quelle/Senke)1
  • Profil „Remoteverwaltung für Audio/Video“ (Audio/Video Remote Control Profile, AVRCP) (Controller/Target)1
  • Profil für Bilddaten (Basic Imaging Profile, BIP) (Initiator/Responder)
  • Basic Printing-Profil (BPP) (Sender)
  • File Transfer Profile (FTP) (Client/Server)
  • Generic Access Profile (GAP)
  • Generic Attribute Profile (GATT)
  • Profil für die allgemeine Audio-/Videoübertragung (Generic Audio/Video Distribution Profile, GAVDP) (Source/Sink)1
  • Generic Object Exchange-Profil (GOEP) (Client/Server)
  • Hands-Free Profile (HFP) (Audio-Gateway) mit Unterstützung für Wide-Band Speech (WBS)1
  • Profil für drahtloses Drucken (Hardcopy Cable Replacement Profile, HCRP) (Client)
  • Headset-Profil (HSP) (Audio-Gateway)1
  • HID Over GATT Profile (HOGP) (Host), auch als Low Energy HID-Profil bezeichnet2
  • Objekt-Push-Profil (Object Push Profile, OPP) (Client/Server)
  • Profil für den Telefonbuchzugriff (Phone Book Access Profile, PBAP) (Client)
  • Synchronization-Profil (SYNC) (Client)

Betriebssysteme:

  • Windows Vista (32-Bit und 64-Bit) SP23
  • Windows 7 (32-Bit und 64-Bit)
  • Windows 8 (32-Bit und 64-Bit)
  • Windows 8.1 (32-Bit und 64-Bit)

Intel® PROSet/Wireless bringt auch einige neue Funktionen mit:

  • Integration von Windows Media Player und Media Center1
  • Integration der VoIP-Software Skype1
  • Integration von Microsoft Office
  • Möglichkeit zum Ausschalten des Bluetooth-Funks3
  • Unterstützung des Flugmodus unter Windows 8 und Windows 8.1
  • Unterstützung für Bluetooth Low Energy2

Liste der Abkürzungen:

  • A2DP – Advanced Audio Distribution-Profil
  • AMP – Alternate MAC/PHY
  • AVRCP – Audio/Video Remote Control-Profil
  • BIP – Basic Imaging-Profil
  • BPP – Basic Printing-Profil
  • DRM – Digital Rights Management
  • EDR – Enhanced Data Rate
  • FTP – File Transfer Protocol
  • GAP – Generic Access Profile
  • GATT – Generic Attribute Profile
  • HID – Human Interface Device
  • HFP – Hands-Free Profile (Freisprech-Profil)
  • HOGP – HID Over GATT Profile
  • HSP – Headset-Profil
  • MAC/PHY – Media Access Control/Physical Layer
  • OBEX – Object Exchange
  • PBAP – Phone Book Access-Profil
  • SCMS-T – Serial Copy Management System
  • SYNCH – Synchronization-Profil
  • SRM – Single Response Mode
  • VoIP – Voice over IP
  • WBS – Wide-Band Speech
