Synchronisieren persönlicher Daten über Bluetooth®

Eine Reihe von Telefonen und anderen Bluetooth-Geräten bieten über einen Bluetooth-Synchronisierungsdienst Zugang zu Kontakten und sonstigen PIM-Daten (Personal Information Manager). So können Sie die Kontakte Ihres Bluetooth-Geräts in einer Datei sichern bzw. aus der Sicherungsdatei wiederherstellen.

Unterstützte Profile und Rollen:

  • SYNC-Client zum Sichern und Wiederherstellen von Kontakten
  • Phone Book Access Profile-(PBAP-)Client zum Sichern von Kontakten

Informationen zu den persönlichen Kontakten mithilfe des SYNC-Profils sichern und wiederherstellen

Hinweis: Nur die vollständige Kontaktdatenbank kann auf dem Gerät gesichert oder wiederhergestellt werden.

Führen Sie im Fenster Dienste des hinzugefügten Bluetooth-Geräts (siehe unter Hinzufügen und Herstellen einer Verbindung mit einem neuen Bluetooth®-Gerät) eine der folgenden Aktionen aus:

  1. Um die Kontakte zu sichern, klicken Sie auf die Schaltfläche PIM Dateien mit SYNC sichern, wählen Sie den Namen der Kontaktsicherungsdatei, und klicken Sie im Dialogfeld Als Datei speichern auf Speichern. Der PIM-Sicherungsprozess wird gestartet. Während der Sicherung wird eine Fortschrittsanzeige angezeigt, und wenn die Daten in der ausgewählten Datei gespeichert wurden, erhalten Sie eine Bestätigungsmeldung.
  2. Um die Kontakte wiederherzustellen, klicken Sie auf die Schaltfläche PIM Dateien mit SYNC wiederherstellen, wählen Sie eine zuvor gespeicherte Quelldatei, und klicken Sie im Dialogfeld Öffnen auf Öffnen. Der PIM-Prozess wird gestartet. Während der Wiederherstellung wird eine Fortschrittsanzeige angezeigt, und wenn die Daten auf dem Bluetooth-Gerät wiederhergestellt wurden, erhalten Sie eine Bestätigungsmeldung.
    Hinweis: Der Link zum Dienst PIM Dateien mit SYNC wiederherstellen ist nicht verfügbar, wenn das Gerät keine Unterstützung für das SYNC-Profil bietet.

Informationen zu den persönlichen Kontakten mithilfe von PBAP sichern

Führen Sie im Fenster „Dienste“ des hinzugefügten Bluetooth-Geräts (siehe Hinzufügen und Herstellen einer Verbindung mit einem neuen Bluetooth®-Gerät) die folgenden Schritte aus:

  1. Klicken Sie auf die Schaltfläche PIM-Dateien mit Telefonbuchanfrage synchronisieren. Das Dialogfenster „Sicherung des Telefonbuchs“ wird angezeigt.
  2. Wählen Sie die Speicherorte des Telefonbuchs aus (interner Telefonspeicher, SIM-Speicher), die gesichert werden sollen, und die Zieldateien.
  3. Drücken Sie auf die Taste Datensicherung, um die PBAP-Sicherung zu starten.
  4. Der Fortschritt der PBAP-Sicherung wird im Fenster Empfang angezeigt.

Hinweis: Die PBAP-Sicherung wird nicht gestartet, wenn keine Speicherorte für das Telefonbuch ausgewählt sind.

Hinweis: Es ist nicht möglich, beide Telefonbuch-Speicherorte in dieselbe Zieldatei zu sichern.
