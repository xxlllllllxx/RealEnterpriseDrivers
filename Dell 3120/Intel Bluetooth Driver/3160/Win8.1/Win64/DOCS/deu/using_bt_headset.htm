Herstellen von Verbindungen zu Bluetooth®-Headsets und Lautsprechern

Hinweis: Dieser Abschnitt trifft nicht zu, wenn Sie das Betriebssystem Windows 8 verwenden.

Mit Bluetooth-Stereo- und Mono-Headsets können Sie Musik anhören und auf dem Computer eingehende VoIP-Anrufe beantworten. Außerdem können Sie den Computer als Bluetooth-Lautsprecher für ein Bluetooth-Gerät verwenden.

Folgende Audioprofile werden unterstützt:

  • A2DP (Senke und Quelle)
  • AVRCP (steuerndes und gesteuertes Gerät)
  • HFP/HSP (Audio-Gateway)
  • Integration von Windows Media Player/Center
  • Integration von Skype

Hören von Musik vom Computer über Bluetooth-Headsets oder -Lautsprecher

An den Computer können Sie ein Bluetooth-Stereo-Headset anschließen, um Musik zu hören und die auf dem Computer ausgeführte Audioplayersoftware zu steuern. Wenn ein Bluetooth-Stereo-Audiogerät angeschlossen ist, können Sie damit die Wiedergabe stoppen/anhalten/starten, zum nächsten/vorhergehenden Stück wechseln und die Lautstärke des auf dem Computer ausgeführten Audioplayers regeln.

Gehen Sie im Fenster Dienste des hinzugefügten Bluetooth-Geräts (siehe unter Hinzufügen und Herstellen einer Verbindung mit einem neuen Bluetooth®-Gerät) wie folgt vor:

  1. Klicken Sie im Fenster Dienste auf den Link Musik anhören, um eine Verbindung herzustellen.
  2. Sobald die Verbindung hergestellt ist, wird die Meldung im Fenster Dienste aktualisiert, und das Bluetooth-Headset bzw. der Bluetooth-Lautsprecher wird zum Standardaudiogerät des Betriebssystems für die Wiedergabe von Musik über den Computer.
  3. Wenn Sie auf dem angeschlossenen Bluetooth-Headset keine Musik anhören, prüfen Sie die Einstellungen für das Standardaudiogerät (siehe Abschnitt Standardaudiogerät weiter unten).
  4. Um eine bestehende Audioverbindung zu trennen, klicken Sie im Fenster Dienste für das Element Musik anhören auf Trennen, und das Standardaudiogerät des Betriebssystems wird auf die ursprüngliche Einstellung zurückgesetzt.

Standardaudiogerät

Wenn eine Audioverbindung hergestellt ist, wird das Bluetooth-Audiogerät unter Windows automatisch als Standardwiedergabegerät festgelegt. In seltenen Fällen funktioniert das Bluetooth-Standardaudiogerät nicht automatisch. Wenn Sie beispielsweise vor der Herstellung der Verbindung mit dem Bluetooth-Audiogerät ein anderes Audiogerät manuell als Standardaudiogerät konfiguriert haben, funktioniert das Bluetooth-Audiogerät unter Umständen nicht. Wechseln Sie in diesem Fall über das Taskleistenmenü zu den Windows-Audioeigenschaften oder in der Systemsteuerung im Abschnitt Hardware und Sound zu Audiogeräte verwalten, und konfigurieren Sie das Bluetooth-Audiogerät als Standardgerät.

Verwenden eines Bluetooth-Headsets für VoIP-Gespräche

Bluetooth-Mono-Headsets können Sie als Audiokommunikationsgeräte für VoIP-Telefonieprogramme einsetzen.

Gehen Sie im Fenster Dienste des hinzugefügten Bluetooth-Geräts (siehe unter Hinzufügen und Herstellen einer Verbindung mit einem neuen Bluetooth®-Gerät) wie folgt vor:

  1. Klicken Sie im Fenster Dienste auf den Link VoIP-Anrufe an Freisprecheinrichtung umleiten.
  2. Sobald die Verbindung hergestellt ist, wird die Meldung im Fenster Dienste aktualisiert, und das Headset wird als Standardkommunikationsgerät für VoIP-Programme wie Skype festgelegt.
  3. Um eine bestehende Audioverbindung zu trennen, klicken Sie im Fenster Dienste für das Element VoIP-Anrufe an Freisprecheinrichtung umleiten auf Trennen.

Empfangen von VoIP-Gesprächen beim Musikhören

Zwischen Stereo- und Mono-Bluetooth-Headsets kann nahtlos umgeschaltet werden, um eingehende Skype-Anrufe anzunehmen oder zu beenden. Verwenden Sie dazu, während Sie Musik anhören, die Schaltfläche(n) des Bluetooth-Audiogeräts. Die Musik wird während des aktiven Skype-Anrufs automatisch angehalten und nach Beendigung des Anrufs fortgesetzt.

Hinweis: Sie können für das nahtlose Ineinandergreifen dieser Funktionen sorgen, indem Sie das Bluetooth Plug-In im Windows Media Player aktivieren und mit dem Plug-In „btplayerctrl.exe“ den Skype-Zugriff sicherstellen.

Wiedergabe von Musik von einem Bluetooth-Gerät über die Computerlautsprecher

Sie können die Lautsprecher des Computers für die die Audiowiedergabe eines Bluetooth-Geräts nutzen.

Gehen Sie im Fenster Dienste des hinzugefügten Bluetooth-Geräts (siehe unter Hinzufügen und Herstellen einer Verbindung mit einem neuen Bluetooth®-Gerät) wie folgt vor:

  1. Klicken Sie im Fenster Dienste auf den Link Musik abspielen.
  2. Sobald der Vorgang abgeschlossen ist, wird das Audiostreaming vom Bluetooth-Remotegerät an Ihre Computerlautsprecher (oder ein anderes auf dem Computer als Standardaudiogerät konfiguriertes Gerät) weitergeleitet.
  3. Während eine Musik abspielen-Verbindung aktiv ist, können Sie den Remote-Mediaplayer im Fenster Bluetooth-Fernbedienung über Bluetooth fernsteuern. Das Symbol Bluetooth-Fernbedienung wird in der Taskleiste angezeigt. Wenn Sie auf das dieses Symbol klicken, wird das Fenster Bluetooth-Fernbedienung eingeblendet. Das Fenster unterstützt Standardaktionen wie Wiedergabe/Stopp sowie Steuerelemente zum Wechseln zum nächsten/vorhergehenden Stück und zur Regulierung der Lautstärke.
  4. Um die Verwendung des Computers als Bluetooth-Lautsprecher zu beenden, klicken Sie im Fenster Dienste auf Trennen.
