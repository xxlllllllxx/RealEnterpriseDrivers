Dateiübertragung über Bluetooth®

Geben Sie lokale Dateien und Ordner auf Ihrem Computer für andere Bluetooth-Geräte frei, übertragen Sie jeden beliebigen Dateityp zwischen Ihrem Computer und Bluetooth-Geräten, erstellen oder löschen Sie sogar Dateien und Ordner auf einem Bluetooth-Gerät, und verwenden Sie den Assistenten „Senden an“ im Menü von Microsoft Office zum Senden von Dateien an Bluetooth-Geräte.

Unterstützte Profile und Rollen:

  • File Transfer Profile (FTP) (Client/Server)
  • Profil für Bilddaten (Basic Imaging Profile, BIP) (Initiator/Responder)
  • Objekt-Push-Profil (Object Push Profile, OPP) (Client/Server)

Unterstützte Funktionen:

  • Object Exchange (OBEX) 1.5 Single Response Mode (SRM)
  • Microsoft Office-Integration für das Object Push-Profil (OPP) (Client)

Freigabe eines lokalen Ordners für andere Bluetooth-Geräte

Ordner auf Ihrem Computer können für andere Bluetooth-Geräte freigegeben werden. Folgen Sie dazu den Anweisungen im Abschnitt Einstellungen für die Ordnerfreigabe dieser Benutzeranleitung.

Der freigegebene Ordner wird als Stammverzeichnis für die Dateiübertragung über FTP verwendet und als Standardverzeichnis zum Dateiempfang über OPP und BIP.

Durchsuchen der freigegebenen Ordner eines Remote-Bluetooth-Geräts

Wenn eine Verbindung zu einem Bluetooth-Gerät besteht, das FTP unterstützt, kann ein standardmäßiges Explorer-ähnliches Fenster verwendet werden, um normale Dateivorgänge zwischen dem Gerät und dem Computer durchzuführen, ganz so, als wenn sie lokal vorgenommen werden würden:

  • Dateien und Ordner durchsuchen
  • Dateien und Ordner übertragen
  • Dateien und Ordner löschen
  • Dateien und Ordner umbenennen
    Hinweis: Wenn das Remote-Gerät FTP-Server V1.2 oder älter unterstützt, dann können nur leere Ordner umbenannt werden.
  • Ordner erstellen

So greifen Sie vom Fenster „Dienste“ auf den freigegebenen Ordner auf dem Remote-Gerät zu:

  1. Öffnen Sie das Fenster „Dienste“ (siehe Hinzufügen und Herstellen einer Verbindung mit einem neuen Bluetooth®-Gerät).
  2. Klicken Sie auf Remote-Ordner durchsuchen.

So greifen Sie vom Windows Explorer auf den freigegebenen Ordner auf dem Remote-Gerät zu:

  1. Öffnen Sie den Ordner Computer.
  2. Klicken Sie auf das Symbol mit dem Namen des Remote-Geräts.

Hinweis: Wenn das Remote-Gerät temporär als FTP-Server verwendet werden kann (d. h. die Verwendung als FTP-Server kann aktiviert oder deaktiviert werden), dann ist keine Verknüpfung zum Ordner Computer erforderlich. Verwenden Sie in diesem Fall das Fenster „Dienste“ des Remote-Geräts.

Hinweis: Während des Datenaustauschs verlangt das Remote-Gerät eventuell eine Kennworteingabe. In diesem Fall muss für beide Geräte das gleiche Kennwort eingegeben werden, damit eine Verbindung hergestellt werden kann. Wenn die Kennwörter nicht übereinstimmen, wird die Verbindung unterbrochen.

Dateien an ein Bluetooth-Gerät senden

Die Funktion Die Datei an Bluetooth-Installationassistenten senden wird verwendet, um Dateien an das Remote-Bluetooth-Gerät zu senden. Der Assistent kann nur eine einzige Datei versenden, es können jedoch mehrere Assistenten gleichzeitig ausgeführt werden. Es wird eine Schrittfolge angezeigt, um das Ziel-Remote-Gerät auszuwählen und die Datei zu versenden. Die Funktion Die Datei an Bluetooth-Installationassistenten senden kann in verschiedenen Szenarien ausgeführt werden.

Hinweis: Normalerweise wird das Protokoll BIP verwendet, um Bilder von einem Mobilgerät auf einen Computer zu übertragen. Das Protokoll OPP kann zum Übertragen aller anderen Dateitypen verwendet werden. Die Funktion Die Datei an Bluetooth-Installationassistenten senden unterstützt beide Profile.

Senden von Dateien von Microsoft Office an ein Bluetooth-Gerät

Folgenden Microsoft Office-Versionen werden unterstützt:

  • Microsoft Office 2010
  • Microsoft Office 2013

Folgende Microsoft Office-Anwendungen und -Elemente werden unterstützt:

  • Word: Dokument
  • Excel: Arbeitsmappe
  • PowerPoint: Präsentation
  • Access: Datenbank
  • Outlook: Termin, Kontakt, Nachricht, Notiz, Aufgabe

Microsoft Office-Elemente werden automatisch gespeichert, bevor Sie an ein Remote-Bluetooth-Gerät gesendet werden können.

Hinweis: Wenn ein Microsoft Office-Element nicht gespeichert wurde, werden Sie in Word, Excel und PowerPoint zum Speichern aufgefordert.

Hinweis: Wenn es sich bei einem Microsoft Office-Element um eine Datenbank handelt, die gerade erstellt wurde, muss die Access-Anwendung neu gestartet werden.

So senden Sie ein Microsoft Office-Element an ein Remote-Bluetooth-Gerät

  1. Wählen Sie abhängig von der von Ihnen verwendeten Microsoft Office-Version und Microsoft-Office-Anwendung An Bluetooth senden im Untermenü Freigeben, speichern und senden oder Speichern und veröffentlichen aus. Bitte lesen Sie in der Hilfe von Microsoft Office nach, wie Sie dazu vorgehen. Das Fenster Assistent zum Senden von Dateien an ein Bluetooth-Gerät wird angezeigt.
  2. Wenn das Fenster Die Datei an Bluetooth-Installationassistenten senden angezeigt wird, wählen Sie das Bluetooth-Gerät aus, an das die Datei gesendet werden soll, und klicken Sie auf die Schaltfläche Weiter.
  3. Das Statusfenster „Dateiübertragung“ wird angezeigt, und das Element wird an das ausgewählte Bluetooth-Gerät gesendet. Sie werden benachrichtigt, wenn die Dateiübertragung abgeschlossen ist.

Senden von Dateien von Windows Explorer an Bluetooth-Geräte

So senden Sie Dateien von Windows Explorer an ein Bluetooth-Gerät:

  1. Klicken Sie im Windows Explorer mit der rechten Maustaste auf das Dateisymbol, und wählen Sie im Kontextmenü unter Senden an die Aktion Bluetooth. Das Fenster Die Datei an Bluetooth-Installationassistenten senden wird angezeigt.
  2. Wenn das Fenster Die Datei an Bluetooth-Installationassistenten senden angezeigt wird, wählen Sie das Bluetooth-Gerät aus, an das die Datei gesendet werden soll, und klicken Sie auf die Schaltfläche Weiter.
  3. Es wird ein Fenster eingeblendet, das den Fortschritt der Dateiübertragung anzeigt, und die ausgewählte Datei wird an das ausgewählte Bluetooth-Gerät gesendet. Sobald die Übertragung abgeschlossen ist, erhalten Sie eine Benachrichtigung.

Senden von Dateien aus dem Fenster Dienste an Bluetooth-Geräte

So senden Sie Dateien aus dem Fenster Dienste an ein Bluetooth-Gerät:

  1. Klicken Sie im Fenster Dienste auf den Link Datei senden. Das Dialogfeld Öffnen wird angezeigt.
  2. Wählen Sie eine Datei aus der Liste, und klicken Sie auf die Schaltfläche Öffnen. Das Fenster Die Datei an Bluetooth-Installationassistenten senden wird angezeigt.
  3. Wenn das Fenster Die Datei an Bluetooth-Installationassistenten senden angezeigt wird, klicken Sie auf die Schaltfläche Weiter.
  4. Es wird ein Fenster eingeblendet, das den Fortschritt der Dateiübertragung anzeigt, und die ausgewählte Datei wird an das ausgewählte Bluetooth-Gerät gesendet. Sobald die Übertragung abgeschlossen ist, erhalten Sie eine Benachrichtigung.

Empfangen von Dateien von Bluetooth-Geräten

Wenn ein anderes Bluetooth-Gerät eine Dateiübertragung zu Ihrem Computer mittels BIP oder OPP startet, wird das Fenster Dateien werden empfangen angezeigt, das den standardmäßigen Zielpfad für eingehende Dateien angibt.

Hinweis: Normalerweise wird das Protokoll BIP verwendet, um Bilder von einem Mobilgerät auf einen Computer zu übertragen. Das Protokoll OPP kann zum Übertragen aller anderen Dateitypen verwendet werden.

Im Fenster „Empfangen von Dateien“ können Sie das Standardverzeichnis ändern, in dem eingehende Dateien bzw. Bilder gespeichert werden. Gehen Sie hierzu wie folgt vor:

  1. Klicken Sie auf Durchsuchen, und wählen Sie den gewünschten Ordner aus.
  2. Wenn Sie ein Verzeichnis ausgewählt haben, klicken Sie auf Annehmen, um die Dateiübertragung zu starten.
  3. Um eine eingehende Datei abzulehnen, klicken Sie auf Ablehnen.

Dateien und Ordner von einem Bluetooth-Gerät ändern

Ein Bluetooth-Gerät, das das FTP-Profil unterstützt, kann Änderungen an den Dateien und Ordnern in dem lokalen, freigegebenen Ordner vornehmen. Wenn das Remote-Gerät versucht, freigegebenen Inhalt zu überschreiben, umzubenennen oder zu löschen, wird dafür eine gesonderte Berechtigung angefordert. Bevor diese Änderungen vorgenommen werden können, wird das Fenster Berechtigungsanfrage angezeigt. Im Fenster Berechtigungsanfrage stehen Ihnen die folgenden Optionen zur Verfügung:

  • Wählen Sie Nur dieses Mal zulassen, und klicken Sie auf OK, um die Änderungen von einem Bluetooth-Gerät temporär zu genehmigen.
  • Wählen Sie Dieses Gerät immer zulassen, und klicken Sie auf OK, um die Änderungen von einem Bluetooth-Gerät zu genehmigen, solange eine Kopplung zu diesem Bluetooth-Gerät besteht.
  • Um die Änderungen von einem Bluetooth-Gerät temporär abzulehnen, klicken Sie auf Abbrechen.

Wenn durch die Übertragung der Datei keine vorhandene Datei geändert wird, dann wird sie übertragen, ohne dass das Fenster Berechtigungsanfrage angezeigt wird.

Fortschritt beim Empfangen der Dateien

Wenn ein anderes Bluetooth-Gerät eine Dateiübertragung zu Ihrem Computer startet, und die Übertragung zugelassen wird, wird eventuell das Fenster Empfang angezeigt:

  • Wenn das Remote-Gerät ein BIP- oder OPP-Client ist, dann wird das Fenster Empfang angezeigt.
  • Wenn das Remote-Gerät ein FTP-Client ist, dann kann das Erscheinungsbild des Fensters Empfang konfiguriert werden, indem Sie die Schritte ausführen, die in diesem Benutzerhandbuch im Abschnitt Einstellungen für die Ordnerfreigabe beschrieben werden.

Das Fenster Empfang zeigt für jede Datei die folgenden Informationen an:

  • Übertragungsstatus (erfolgreich empfangen, nicht empfangen, wird übertragen usw.)
  • Die Übertragungs-Baud-Rate für die Dateien, die momentan übertragen werden
  • Die Größe der Dateien, die erfolgreich empfangen wurden
