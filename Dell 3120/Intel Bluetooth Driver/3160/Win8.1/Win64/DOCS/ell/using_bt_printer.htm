Σύνδεση σε εκτυπωτές Bluetooth®

Στο παράθυρο Υπηρεσίες της συσκευής Bluetooth που προστέθηκε (ανατρέξτε στην ενότητα Προσθήκη και σύνδεση νέας συσκευής Bluetooth®):

  1. Κάντε κλικ στη σύνδεση Εκτύπωση αρχείων και επιλέξτε τα αρχεία που θέλετε να στείλετε στον εκτυπωτή Bluetooth.
  2. Μόλις ολοκληρωθεί η εκτύπωση, θα ειδοποιηθείτε.
