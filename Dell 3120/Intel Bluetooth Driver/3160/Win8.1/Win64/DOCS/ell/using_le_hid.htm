Σύνδεση σε συσκευές HID χαμηλής ενέργειας Bluetooth®

Σημείωση: Η δυνατότητα HID Bluetooth χαμηλής ενέργειας που περιγράφεται σε αυτό το έγγραφο ισχύει μόνο για τα Windows 7. Τα Windows 8 υποστηρίζουν εγγενώς τη δυνατότητα Bluetooth χαμηλής ενέργειας. Αντίστοιχα, η υποστήριξη HID Bluetooth χαμηλής ενέργειας δεν ισχύει για το λογισμικό Intel® PROSet/Wireless Software for Bluetooth® Technology στα Windows 8.

Μόλις προστεθεί η συσκευή HID Bluetooth χαμηλής ενέργειας (ανατρέξτε στην ενότητα Προσθήκη και σύνδεση νέας συσκευής Bluetooth®), η σύνδεση με τη συσκευή HID Bluetooth χαμηλής ενέργειας γίνεται αυτόματα. Σε περίπτωση ξαφνικής αποσύνδεσης, το λογισμικό προσπαθεί να δημιουργήσει πάλι τη σύνδεση.

Έλεγχος της κατάστασης της μπαταρίας σε συσκευές HID χαμηλής ενέργειας

Το εικονίδιο Κατάσταση μπαταρίας εμφανίζεται στην περιοχή ειδοποιήσεων όταν η στάθμη φόρτισης της μπαταρίας κάποιας συνδεδεμένης συσκευής HID χαμηλής ενέργειας είναι χαμηλότερη από 10%.

Για να ελέγξετε την κατάσταση της μπαταρίας σε συσκευές HID χαμηλής ενέργειας:

  1. Κάντε κλικ στο εικονίδιο Κατάσταση μπαταρίας στην περιοχή ειδοποιήσεων.
  2. Εμφανίζεται ένα αναδυόμενο παράθυρο με τη λίστα των συσκευών HID χαμηλής ενέργειας και την κατάσταση της μπαταρίας τους. Η κατάσταση της μπαταρίας εμφανίζεται μόνο για τις συνδεδεμένες συσκευές HID χαμηλής ενέργειας με στάθμη φόρτισης μπαταρίας χαμηλότερη από 10%.
  3. Αν θέλετε να αποκρύψετε το αναδυόμενο παράθυρο, κάντε κλικ σε ένα σημείο έξω από αυτό.
