Synchronizing Personal Information over Bluetooth®

A number of phones and other Bluetooth devices provide access to contacts and other Personal Information Manager (PIM) data using a Bluetooth synchronization service to backup contacts from your Bluetooth device into a file, and to restore contacts from a backup file.

Supported profiles and roles:

  • SYNC client to backup and restore contacts
  • Phone Book Access Profile (PBAP) client to backup contacts

Backup and Restore Personal Contacts Information Using SYNC Profile

Note: Only the complete contacts database can be backed up or restored on the device.

At the Services window of the added Bluetooth device (see Add and Connect to a New Bluetooth® Device) perform one of the following:

  1. To backup, click on the Backup PIM Data with SYNC button, select the name of the contact backup file, and click Save on the Save as File dialog. The PIM backup process starts with a progress indicator in the display, and a completion message will display once the data is saved to the selected file.
  2. To restore, click on the Restore PIM Data with SYNC button, select a previously saved source file, and click Open in the Open dialog. The PIM process starts with a progress indicator in the display, and a completion message will display once the data is restored to the Bluetooth device.
    Note: Service Restore PIM Data with SYNC link is not available if the device does not support the SYNC Profile.

Backup Personal Contacts Information Using PBAP

At the Services window of the added Bluetooth device (see Add and Connect to a New Bluetooth® Device) perform the following:

  1. Click on the Synchronize PIM Data with Phone Book Access button. The Phonebook Backup dialog will appear.
  2. Select phonebook locations (Internal phone memory, SIM memory) to be backed up and destination files.
  3. Press the Backup button to start PBAP backup operation.
  4. The PBAP backup operation progress will be shown in the Receiving window.

Note: PBAP backup operation will not start if there are no selected phonebook locations.

Note: It is not possible to backup both phonebook locations to the same destination file.
