Transferencia de archivos por Bluetooth®

Comparta archivos y carpetas locales del equipo con otros dispositivos Bluetooth, transfiera cualquier tipo de archivo entre el equipo y los dispositivos Bluetooth, cree o elimine archivos y carpetas de dispositivos Bluetooth o envíe archivos a dispositivos Bluetooth con el asistente Enviar a del menú de Microsoft Office.

Funciones y perfiles admitidos:

  • Perfil de transferencia de archivos (FTP) (cliente/servidor)
  • Perfil básico de imágenes (BIP) (iniciador/destinatario)
  • Perfil de inserción de objetos (OPP) (cliente/servidor)

Funciones soportadas:

  • Modo de respuesta individual (SRM) de intercambio de objetos (OBEX) 1.5
  • Integración de Microsoft Office para cliente de perfil de inserción de objetos (OPP)

Uso compartido de carpetas locales con otros dispositivos Bluetooth

Para compartir carpetas del equipo con otros dispositivos Bluetooth, siga los pasos que se explican en el apartado Configuración del uso compartido de carpetas de esta guía del usuario.

La carpeta compartida se utiliza como la carpeta raíz para la transferencia de archivos a través de FTP y como carpeta predeterminada para recibir archivos mediante OPP y BIP.

Exploración de carpeta compartida de un dispositivo Bluetooth remoto

Cuando esté conectado a un dispositivo Bluetooth compatible con FTP, se puede usar una ventana estándar similar a la de un explorador para llevar a cabo operaciones habituales con archivos entre el dispositivo y el ordenador, tal y como haría localmente:

  • Explorar archivos y carpetas
  • Transferir archivos y carpetas
  • Eliminar archivos y carpetas
  • Cambiar el nombre de archivos y carpetas
    Nota: si el dispositivo remoto soporta la función del servidor FTP menor a v1.2, sólo se puede cambiar el nombre de las carpetas vacías.
  • Crear carpetas

Para acceder a la carpeta compartida en el dispositivo remoto desde la ventana del servicio:

  1. Abra la ventana Servicios del dispositivo (consulte Agregación de dispositivos Bluetooth® nuevos y conexión a ellos).
  2. Haga clic en Examinar carpeta remota.

Para acceder a la carpeta compartida en el dispositivo remoto desde el Explorador de Windows:

  1. Abra la carpeta Equipo.
  2. Haga clic en el icono con el nombre del dispositivo remoto.

Nota: si el dispositivo remoto soporta la función del servidor FTP temporalmente (es decir, la función del servidor FTP se puede activar o desactivar), es posible que no aparezca el enlace de la carpeta Equipo. En este caso, utilice la ventana de servicios de dispositivo remoto.

Nota: durante el intercambio de datos, es posible que el dispositivo remoto solicite una contraseña; si ocurre, introduzca la misma contraseña en ambos dispositivos para permitir la conexión. Si las contraseñas no coinciden, se cancela la conexión.

Envío de archivos a un dispositivo Bluetooth

El Asistente para enviar archivos a un dispositivo Bluetooth se utiliza para enviar archivos a los dispositivos Bluetooth remotos. El asistente es capaz de enviar un archivo único pero se pueden iniciar varios asistentes al mismo tiempo. Se proporciona una secuencia de pasos para seleccionar el dispositivo remoto y el archivo de destino que se van a enviar. El Asistente para enviar archivos a un dispositivo Bluetooth puede ejecutarse en varias situaciones.

Nota: BIP se suele utilizar para enviar fotografías de dispositivos móviles a ordenadores. OPP se utiliza para enviar el resto de tipos de archivos. El Asistente para enviar archivos a un dispositivo Bluetooth es compatible con ambos perfiles.

Envío de archivos de Microsoft Office a dispositivos Bluetooth

Son compatibles las siguientes versiones de Microsoft Office:

  • Microsoft Office 2010
  • Microsoft Office 2013

Son compatibles las siguientes aplicaciones y elementos de Microsoft Office:

  • Word: documentos
  • Excel: libros de trabajo
  • PowerPoint: presentaciones
  • Access: bases de datos
  • Outlook: citas, contactos, mensajes, notas, tareas

Los elementos de Microsoft Office se guardan automáticamente antes de que se puedan enviar al dispositivo Bluetooth remoto.

Nota: si un elemento de Microsoft Office no se ha guardado anteriormente, se mostrará un mensaje para guardar en Word, Excel y PowerPoint.

Nota: si un elemento de Microsoft Office es una base de datos y se ha creado recientemente, debe reiniciar la aplicación Access.

Para enviar un elemento de Microsoft Office a un dispositivo Bluetooth remoto:

  1. En función de la versión de Microsoft Office y la aplicación de Microsoft Office que utilice, seleccione Send to Bluetooth (Enviar a Bluetooth) en el submenú Share (Compartir), Save and Send (Guardar y compartir) o Save and Publish (Guardar y publicar). Para ello, consulte en la ayuda de Microsoft Office las instrucciones correspondientes. Aparecerá el Asistente para enviar archivos a un dispositivo Bluetooth.
  2. Cuando se abra el Asistente para enviar archivos a un dispositivo Bluetooth, seleccione el dispositivo Bluetooth para enviar el archivo y haga clic en el botón Siguiente.
  3. Aparece la ventana de progreso de la transferencia de archivos y el elemento se envía al dispositivo Bluetooth seleccionado. Una vez terminada la transferencia, se muestra un aviso.

Envío de archivos del Explorador de Windows a dispositivos Bluetooth

Para enviar archivos del Explorador de Windows a un dispositivo Bluetooth:

  1. En el Explorador de Windows, haga clic con el botón derecho en el icono del archivo y seleccione la acción Bluetooth en Enviar a en el menú contextual. Aparecerá el Asistente para enviar archivos a un dispositivo Bluetooth.
  2. Cuando se abra el Asistente para enviar archivos a un dispositivo Bluetooth, seleccione el dispositivo Bluetooth para enviar el archivo y haga clic en el botón Siguiente.
  3. Aparece la ventana de progreso de la transferencia de archivos y el archivo seleccionado se envía al dispositivo Bluetooth seleccionado. Una vez terminada la transferencia, se muestra un aviso.

Envío de archivos de la ventana Servicios a dispositivos Bluetooth

Para enviar archivos de la ventana Servicios a un dispositivo Bluetooth:

  1. Haga clic en el enlace Enviar archivo de la ventana Servicios. Aparece el cuadro de diálogo Abrir.
  2. Seleccione un archivo de la lista y haga clic en el botón Abrir. Aparecerá el Asistente para enviar archivos a un dispositivo Bluetooth.
  3. Cuando se abra el Asistente para enviar archivos a un dispositivo Bluetooth, haga clic en el botón Siguiente.
  4. Aparece la ventana de progreso de la transferencia de archivos y el archivo seleccionado se envía al dispositivo Bluetooth seleccionado. Una vez terminada la transferencia, se muestra un aviso.

Recepción de archivos de dispositivos Bluetooth

Cuando otro dispositivo Bluetooth inicie una transferencia de archivos al ordenador mediante BIP u OPP, aparecerá la ventana Recibiendo archivos con una ruta de destino predeterminada para los archivos entrantes.

Nota: BIP se suele utilizar para enviar fotografías de dispositivos móviles a ordenadores. OPP se utiliza para enviar el resto de tipos de archivos.

En la ventana Recibir archivos puede cambiar la carpeta de destino predeterminada donde se guardarán los archivos o imágenes entrantes. Para ello, siga estos pasos:

  1. Haga clic en Examinar y seleccione la carpeta que desee.
  2. Después de seleccionar una ruta, haga clic en Aceptar para iniciar la transferencia de archivos.
  3. Para rechazar un archivo entrante, haga clic en Rechazar.

Modificación de archivos y carpetas desde un dispositivo Bluetooth

Un dispositivo Bluetooth que soporte el perfil de FTP puede realizar cambios en los archivos y carpetas de la carpeta local compartida. Se solicitarán permisos especiales si se intenta sobrescribir, cambiar el nombre o eliminar contenido compartido del dispositivo remoto. Aparecerá la ventana de Solicitud de permisos antes de poder aplicar estos cambios. En esta ventana, podrá hacer lo siguiente:

  • Seleccione la opción para Permitir solo esta vez y luego haga clic en Aceptar para permitir temporalmente que se realicen cambios desde un dispositivo Bluetooth.
  • Seleccione la opción para Permitir siempre este dispositivo y luego haga clic en Aceptar para permitir que se realicen cambios desde un dispositivo Bluetooth cuando está acoplado.
  • Para denegar temporalmente la realización de cambios desde un dispositivo Bluetooth, haga clic en Cancelar.

Si el archivo que desea transferir no cambia ningún archivo existente, éste se transferirá sin que aparezca la ventana de Solicitud de permisos.

Recepción del progreso de los archivos

Cuando otro dispositivo Bluetooth inicia una transferencia de archivos al ordenador y se acepta la transferencia, puede aparecer la ventana de recepción:

  • Si el dispositivo remoto es un cliente OPP o BIP, aparecerá la ventana de recepción.
  • Si el dispositivo remoto es un cliente FTP, la ventana de recepción se puede configurar siguiendo los pasos descritos en la sección Configuración del uso compartido de carpetas de esta guía del usuario.

La ventana de recepción muestra la siguiente información para cada uno de los archivos:

  • Estado de la transferencia (recibido correctamente, no recibido, transferencia en curso, etc.)
  • Velocidad de transferencia en baudios para archivos que se están transfiriendo actualmente
  • Tamaño de los archivos que se han recibido correctamente
