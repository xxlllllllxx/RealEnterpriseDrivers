Gestion et configuration Bluetooth®

Activation et désactivation du signal radio Bluetooth

Si nécessaire, vous pouvez désactiver la fonction Bluetooth grâce à trois outils : un commutateur logiciel, un interrupteur matériel ou le Gestionnaire d'appareils. Il est fortement déconseillé d'utiliser ces trois outils ensemble (par ex., désactiver le Bluetooth avec le commutateur logiciel et le rallumer avec le Gestionnaire d'appareils).

Le commutateur logiciel est l'outil recommandé pour l'activation et la désactivation de la radio Bluetooth. Il est fortement déconseillé d'utiliser le Gestionnaire d'appareils pour activer/désactiver les pilotes de l'appareil Bluetooth. Cependant, le commutateur logiciel n'est pas disponible sur tous les systèmes. Les procédures permettant d'utiliser le commutateur logiciel varient selon les systèmes d'exploitation et sont décrites ci-dessous.

Certains systèmes peuvent avoir un interrupteur matériel remplaçant, ou s'ajoutant à, un commutateur logiciel. Lorsque les deux interrupteurs sont présents, évitez de les utiliser en même temps (si la radio Bluetooth est désactivée à l'aide du commutateur logiciel, elle restera éteinte même si vous activez et désactivez l'interrupteur matériel). Consultez le manuel d'utilisation du système pour plus d'informations sur l'interrupteur matériel.

Sous Windows 7, pour activer ou désactiver la radio Bluetooth :

  1. Cliquez avec le bouton droit de la souris sur l'icône Bluetooth dans la barre des tâches.
  2. Sélectionnez Activer l'adaptateur ou Désactiver l'adaptateur.

Vous pouvez également désactiver la radio Bluetooth via la fenêtre Paramètres Bluetooth :

  1. Cliquez avec le bouton droit de la souris sur l'icône Bluetooth dans la barre des tâches.
  2. Sélectionnez Ouvrir les paramètres.
  3. Dans l'onglet Options, vérifiez que la case Désactiver l'adaptateur Bluetooth est bien cochée.

Sous Windows 8 et versions ultérieures, pour activer ou désactiver la radio Bluetooth :

  1. Ouvrez le menu Paramètres du PC.
    • Faites apparaître le menu Icônes en plaçant le curseur dans le coin inférieur droit, sélectionnez Paramètres, puis cliquez sur Modifier les paramètres du PC.
  2. Sélectionnez le nœud Sans fil.
  3. Cliquez sur le curseur Bluetooth pour activer ou désactiver la fonction Bluetooth.

Paramètres de partage de dossiers

Pour activer ou désactiver le partage de dossiers :

  1. Cliquez avec le bouton droit de la souris sur l'icône Bluetooth de la barre d'état et sélectionnez Ouvrir les paramètres.
  2. Dans la fenêtre Paramètres Bluetooth, sélectionnez l'onglet Dossier partagé, puis effectuez l'une des opérations suivantes :
    • Pour activer le partage de fichiers, cochez la case Permettre aux périphériques distants de parcourir le dossier partagé, cliquez sur Parcourir pour sélectionner le dossier local à partager avec les périphériques Bluetooth, puis cliquez sur OK.
    • Pour désactiver le partage de fichiers, décochez la case Permettre aux périphériques distants de parcourir le dossier partagé.

Pour activer ou désactiver les notifications utilisateur pour le téléchargement des fichiers via FTP :

  1. Cliquez avec le bouton droit de la souris sur l'icône Bluetooth dans la barre d'état système et sélectionnez Ouvrir les paramètres.
  2. Dans la fenêtre Paramètres Bluetooth, sélectionnez l'onglet Dossier partagé, puis effectuez l'une des opérations suivantes :
    • Pour activer les notifications utilisateur, cochez la case Me prévenir lors du chargement de fichiers dans le dossier partagé, puis cliquez sur OK.
    • Pour désactiver les notifications utilisateur, décochez la case Me prévenir lors du chargement de fichiers dans le dossier partagé.

Paramètres Audio

Remarque : cette section ne s'applique pas si vous utilisez le système d'exploitation Windows 8.

Pour gérer les paramètres audio Bluetooth :

  1. Cliquez avec le bouton droit de la souris sur l'icône Bluetooth de la barre d'état et sélectionnez Ouvrir les paramètres.
  2. Dans la fenêtre Paramètres Bluetooth, sélectionnez l'onglet Audio, puis effectuez l'une ou plusieurs des opérations suivantes :
    • Pour autoriser la connexion audio Bluetooth aux seuls périphériques prenant en charge la protection du contenu DRM (Digital Rights Management - Gestion des droits numériques) SCMS-T, cochez la case Connectez à un périphérique audio uniquement s'il prend en charge la protection du contenu SCMS-T.
    • Pour activer la commutation d'appel Skype automatique lorsque vous écoutez de la musique, sélectionnez l'option Activer la commutation Skype et le nom du casque audio Bluetooth précédemment connecté dans le menu déroulant. Pour plus d'informations, reportez-vous à la section Recevoir des appels voix sur IP (VoIP) tout en écoutant de la musique. Cette option n'est pas disponible si le logiciel Skype n'est pas installé sur l'ordinateur.
