Installation, réparation, suppression et mise à niveau

Installation

Pour commencer l'installation, activez le matériel et lancez le fichier Setup.exe à partir du disque d'installation. Ce faisant, vous démarrez l'Assistant d'installation.

Suivez les différentes étapes de l'Assistant d'installation :

  1. Cliquez sur Suivant sur la page Bienvenue.
  2. Lisez et acceptez l'accord de licence de la page Accord de licence. Cliquez sur Suivant.
  3. Si nécessaire, modifiez le dossier de destination sur la page Dossier de destination.
    Le dossier d'installation par défaut est “C:\Program Files\Intel\Bluetooth”.
  4. Sur la page Prêt pour l'installation du programme, cliquez sur Installer pour lancer le processus d'installation. La progression de l'installation apparaît.
  5. Lorsque le processus d'installation est terminé, cliquez sur Terminer.
  6. Une fois l'application installée, vous pouvez la lancer à partir de l'icône Bluetooth® de la barre d'état système.

Réparation

Pour effectuer des réparations, procédez comme suit :

  1. Ouvrez le Panneau de configuration.
  2. Cliquez sur le lien Désinstaller un programme dans le groupe Programmes (ou double-cliquez sur l'icône Programmes et fonctionnalités) du Panneau de configuration.
  3. Sélectionnez Intel® PROSet/Logiciel sans fil pour technologie Bluetooth® dans la liste des applications et double-cliquez dessus.
  4. Cliquez sur Suivant dans la fenêtre Assistant d'installation.
  5. Cliquez sur Réparer et suivez les instructions.
  6. Pour terminer la réparation, redémarrez le système. Tous les changements sont effectifs après le redémarrage.

Suppression

Pour désinstaller le module Bluetooth, procédez comme suit :

  1. Ouvrez le Panneau de configuration.
  2. Cliquez sur le lien Désinstaller un programme dans le groupe Programmes (ou double-cliquez sur l'icône Programmes et fonctionnalités) du Panneau de configuration.
  3. Sélectionnez Intel® PROSet/Logiciel sans fil pour technologie Bluetooth® dans la liste des applications et double-cliquez dessus.
  4. Cliquez sur Suivant dans la fenêtre Assistant d'installation.
  5. Cliquez sur Désinstaller et suivez les instructions.
  6. Pour terminer la désinstallation, redémarrez le système. Le logiciel Bluetooth est désinstallé une fois le redémarrage effectué.

Mettre à niveau

Procédez de l'une des manières suivantes pour mettre le logiciel à niveau :

  • Exécutez le progiciel du programme d'installation doté de la version la plus récente du logiciel et suivez les instructions de l'Assistant.
  • Désinstallez la version actuelle du logiciel installé, redémarrez le PC et exécutez le progiciel du programme d'installation doté de la version la plus récente du logiciel.
