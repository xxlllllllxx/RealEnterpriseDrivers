Synchroniser les informations personnelles sur Bluetooth®

Un certain nombre de téléphones et autres périphériques Bluetooth permettent d'accéder aux contacts et autres données PIM (Personal Information Manager - Gestionnaire d'informations personnelles) à l'aide d'un service de synchronisation Bluetooth pour sauvegarder les contacts de votre périphérique Bluetooth dans un fichier et les restaurer à partir d'un fichier de sauvegarde.

Profils et rôles pris en charge :

  • Client SYNC pour la sauvegarde et la restauration des contacts
  • Client PBAP (Profil d'accès à l'annuaire) pour la sauvegarde des contacts

Sauvegarde et restauration des informations de contacts personnels à l'aide d'un profil SYNC

Remarque : Seule une base de données de contacts complète peut être sauvegardée ou restaurée sur le périphérique.

Dans la fenêtre Services du périphérique Bluetooth ajouté (voir Ajout et connexion d'un nouveau périphérique Bluetooth®), effectuez l'une des opérations suivantes :

  1. Pour sauvegarder, cliquez sur le bouton Sauvegarder les données PIM avec SYNC, sélectionnez le nom du fichier de sauvegarde des contacts et cliquez sur Enregistrer dans la boîte de dialogue Enregistrer sous forme de fichier. Le processus de sauvegarde PIM démarre avec un indicateur de progression à l'écran et un message de fin apparaît une fois les données enregistrées dans le fichier sélectionné.
  2. Pour restaurer, cliquez sur le bouton Restaurer les données PIM avec SYNC, sélectionnez un fichier source précédemment enregistré et cliquez sur Ouvrir dans la boîte de dialogue Ouvrir. Le processus de restauration PIM démarre avec un indicateur de progression à l'écran et un message de fin apparaît une fois les données restaurées sur le périphérique Bluetooth.
    Remarque : le lien du service Restaurer les données PIM avec SYNC n'est pas disponible si le périphérique ne prend pas en charge le profil SYNC.

Sauvegarde des informations de contacts personnels à l'aide du PBAP

Dans la fenêtre Services du périphérique Bluetooth ajouté (reportez-vous à la section Ajout et connexion d'un nouveau périphérique Bluetooth®), effectuez les opérations suivantes :

  1. Cliquez sur le bouton Synchroniser les données PIM avec l'annuaire. La Boîte de dialogue Sauvegarde du carnet d'adresses s'affiche.
  2. Sélectionnez l'emplacement des carnets d'adresses (mémoire interne du téléphone, mémoire de la carte SIM) à sauvegarder et les fichiers de destination.
  3. Appuyez sur le bouton Sauvegarde pour lancer l'opération de sauvegarde PBAP.
  4. L'avancement de l'opération de sauvegarde PBAP est indiqué dans la fenêtre Réception.

Remarque : l'opération de sauvegarde PBAP se lance uniquement si l'emplacement de carnets d'adresses est sélectionné.

Remarque : il n'est pas possible de sauvegarder les carnets d'adresses des deux emplacements dans le même fichier de destination.
