Trasferimento file tramite Bluetooth®

Consente di condividere file e cartelle locali sul computer con altri dispositivi Bluetooth, trasferire qualsiasi tipo di file tra il computer e i dispositivi Bluetooth, persino creare o eliminare file e cartelle sul dispositivo Bluetooth e utilizzare la procedura guidata di invio dal menu Microsoft Office per inviare file ai dispositivi Bluetooth.

Profili e ruoli supportati:

  • FTP (File Transfer Profile) (client/server)
  • Basic Imaging Profile (BIP) (Iniziatore/Soccorritore)
  • Profilo OPP (Object Push Profile) (Client/Server)

Funzioni supportate:

  • SRM (Single Response Mode) OBEX (Object Exchange) 1.5
  • Integrazione di Microsoft Office per il client OPP (Object Push Profile)

Condivisione della cartella locale con altri dispositivi Bluetooth

È possibile condividere le cartelle sul computer con altri dispositivi Bluetooth attenendosi alla procedura descritta nella sezione Impostazione di condivisione della cartella riportata in questo Manuale per l'utente.

La cartella condivisa è utilizzata come cartella principale per il trasferimento dei file mediante FTP e come cartella predefinita per la ricezione dei file mediante OPP e BIP.

Esplorazione di una cartella condivisa su un dispositivo Bluetooth remoto

In caso di connessione a un dispositivo Bluetooth che supporta la tecnologia FTP, è possibile utilizzare una finestra standard analoga a Explorer per eseguire normali operazioni con i file tra il dispositivo e il computer, come avverrebbe in locale:

  • Esplorazione di file e cartelle
  • Trasferimento di file e cartelle
  • Eliminazione di file e cartelle
  • Ridenominazione di file e cartelle
    Nota: se il dispositivo remoto supporta un ruolo server FTP precedente alla versione v1.2, è possibile rinominare solo le cartelle vuote.
  • Creazione di cartelle

Per accedere alla cartella condivisa sul dispositivo remoto dalla finestra Servizi:

  1. Aprire la finestra Servizi del dispositivo (vedere Aggiunta e collegamento di un nuovo dispositivo Bluetooth®).
  2. Fare clic su Sfoglia cartella remota.

Per accedere alla cartella condivisa sul dispositivo remoto da Windows Explorer:

  1. Aprire la cartella Computer.
  2. Fare clic sull'icona con il nome del dispositivo remoto.

Nota: se il dispositivo remoto supporta temporaneamente il ruolo server FTP (cioè il ruolo server FTP può essere attivato o disattivato), è possibile che il collegamento nella cartella Computer non sia presente. In questo caso, utilizzare la finestra Servizi del dispositivo remoto.

Nota: durante lo scambio dati, il dispositivo remoto potrebbe richiedere una password. In tal caso entrambi i dispositivi devono immettere la stessa password per consentire il collegamento. Se le password non corrispondono, il collegamento verrà terminato.

Invio di file a un dispositivo Bluetooth

La Procedura guidata per inviare file al dispositivo Bluetooth viene utilizzata per inviare i file al dispositivo Bluetooth remoto. La procedura guidata consente di inviare un singolo file, ma è possibile avviare più procedure guidate contemporaneamente. È disponibile una sequenza di operazioni per selezionare il dispositivo remoto di destinazione e il file da inviare. La Procedura guidata per inviare file al dispositivo Bluetooth può essere eseguita in diversi scenari.

Nota: il profilo BIP viene in genere utilizzato per l'invio di immagini da un dispositivo mobile a un computer. Il profilo OPP viene utilizzato per inviare tutti gli altri tipi di file. La Procedura guidata per inviare file al dispositivo Bluetooth supporta entrambi i profili.

Invio di file da Microsoft Office a un dispositivo Bluetooth

Sono supportate le seguenti versioni di Microsoft Office:

  • Microsoft Office 2010
  • Microsoft Office 2013

Sono supportati i seguenti elementi e applicazioni di Microsoft Office:

  • Word: documento
  • Excel: cartella di lavoro
  • PowerPoint: presentazione
  • Access: database
  • Outlook: appuntamento, contatto, messaggio, nota, attività

L'elemento Microsoft Office viene salvato automaticamente prima di essere inviato a un dispositivo Bluetooth remoto.

Nota: se l'elemento Microsoft Office non è mai stato salvato, la richiesta di salvataggio viene visualizzata in Word, Excel e PowerPoint.

Nota: se l'elemento Microsoft Office è il database ed è stato appena creato, l'applicazione Access viene riavviata.

Per inviare un elemento Microsoft Office a un dispositivo Bluetooth remoto:

  1. In base alla versione di Microsoft Office e all'applicazione Microsoft Office in uso, selezionare Spedire a Bluetooth nel sottomenu Condividi, Salva e invia o Salva e pubblica. Consultare la Guida di Microsoft Office per le istruzioni su come eseguire questa operazione. Verrà avviata la Procedura guidata per inviare file al dispositivo Bluetooth.
  2. Una volta avviata la Procedura guidata per inviare file al dispositivo Bluetooth, selezionare il dispositivo Bluetooth al quale inviare il file e fare clic sul pulsante Avanti.
  3. Verrà visualizzata la finestra dello stato del trasferimento e l'elemento verrà inviato al dispositivo Bluetooth selezionato. Al termine del trasferimento verrà ricevuta una notifica.

Invio di file da Windows Explorer a un dispositivo Bluetooth

Per inviare file da Windows Explorer a un dispositivo Bluetooth:

  1. In Windows Explorer, fare clic con il pulsante destro del mouse sull'icona del file e selezionare l'azione Bluetooth nella voce Spedire a nel menu contestuale. Verrà avviata la Procedura guidata per inviare file al dispositivo Bluetooth.
  2. Una volta avviata la Procedura guidata per inviare file al dispositivo Bluetooth, selezionare il dispositivo Bluetooth al quale inviare il file e fare clic sul pulsante Avanti.
  3. Verrà visualizzata la finestra dello stato del trasferimento e il file scelto verrà inviato al dispositivo Bluetooth selezionato. Al termine del trasferimento verrà ricevuta una notifica.

Invio di file dalla finestra Servizi a un dispositivo Bluetooth

Per inviare file dalla finestra Servizi a un dispositivo Bluetooth:

  1. Fare clic sul collegamento Invia file nella finestra Servizi. Verrà visualizzata la finestra di dialogo Apri.
  2. Scegliere un file dall'elenco e fare clic sul pulsante Apri. Verrà avviata la Procedura guidata per inviare file al dispositivo Bluetooth.
  3. Una volta avviata la Procedura guidata per inviare file al dispositivo Bluetooth, fare clic sul pulsante Avanti.
  4. Verrà visualizzata la finestra dello stato del trasferimento e il file scelto verrà inviato al dispositivo Bluetooth selezionato. Al termine del trasferimento verrà ricevuta una notifica.

Ricezione di file da un dispositivo Bluetooth

Se un altro dispositivo Bluetooth avvia un trasferimento file al computer mediante profilo BIP od OPP, verrà visualizzata la finestra Ricezione di file con un percorso di destinazione predefinito per i file in entrata.

Nota: il profilo BIP viene in genere utilizzato per l'invio di immagini da un dispositivo mobile a un computer. Il profilo OPP viene utilizzato per inviare tutti gli altri tipi di file.

Nella finestra Ricezione file è possibile modificare la cartella di destinazione predefinita nella quale vengono salvati i file e le foto in arrivo. A tale scopo, attenersi alla seguente procedura:

  1. Fare clic su Sfoglia e selezionare la finestra della cartella desiderata.
  2. Dopo aver selezionato il percorso, fare clic su Accetta per avviare il trasferimento dei file.
  3. Per rifiutare un file in arrivo, fare clic su Rifiuta.

Modifica di file e cartelle da un dispositivo Bluetooth

Un dispositivo Bluetooth che supporta il profilo FTP può apportare modifiche ai file e alle cartelle contenuti nella cartella condivisa locale. Saranno richieste autorizzazioni speciali se il dispositivo remoto tenterà di sovrascrivere, rinominare o eliminare i contenuti condivisi. Prima di apportare queste modifiche, verrà visualizzata la finestra Richiesta di autorizzazione. All'interno della finestra Richiesta di autorizzazione sono disponibili le seguenti opzioni:

  • Selezionare Consenti solo questa volta e fare clic su OK per consentire temporaneamente modifiche da un dispositivo Bluetooth.
  • Selezionare Autorizza sempre questo dispositivo e fare clic su OK per consentire modifiche da un dispositivo Bluetooth durante l'associazione.
  • Per rifiutare temporaneamente modifiche da un dispositivo Bluetooth, fare clic su Annulla.

Se il file da trasferire non causa la modifica di nessun file esistente, è possibile trasferirlo senza che venga visualizzata la finestra Richiesta di autorizzazione.

Avanzamento ricezione file

Quando un altro dispositivo Bluetooth avvia un trasferimento file al computer e il trasferimento viene accettato, è possibile che la finestra Ricezione venga visualizzata:

  • Se il dispositivo remoto è un client BIP o OPP, viene visualizzata la finestra Ricezione.
  • Se il dispositivo remoto è un client FTP, è possibile configurare l'aspetto della finestra Ricezione attenendosi alla procedura descritta nella sezione Impostazione di condivisione della cartella riportata nel presente Manuale per l'utente.

La finestra Ricezione mostra le seguenti informazioni per ciascun file:

  • Stato del trasferimento (ricevuto correttamente, non ricevuto, trasferimento in corso, ecc.)
  • Velocità di trasmissione per i file attualmente in fase di trasferimento
  • Dimensioni dei file ricevuti correttamente
