설치, 복구, 제거 및 업그레이드

설치

설치를 시작하려면 하드웨어를 작동시키고 설치 디스크에서 Setup.exe를 실행합니다. 이렇게 하면 설치 마법사가 시작됩니다.

설치 마법사의 안내에 따라 다음 단계를 수행하십시오.

  1. 환영합니다 페이지에서 다음을 클릭합니다.
  2. 이용 약관 페이지에서 이용 약관을 읽고 내용에 동의합니다. 그리고 나서 다음을 클릭합니다.
  3. 필요한 경우 대상 폴더 페이지에서 설치 폴더를 변경합니다.
    기본 설치 경로는 “C:\Program Files\Intel\Bluetooth”입니다.
  4. 프로그램 설치 준비 페이지에서 설치를 클릭하여 설치 절차를 시작합니다. 설치 진행률이 표시됩니다.
  5. 설치 과정이 완료되면 완료를 클릭합니다.
  6. 설치 후에 시스템 트레이 영역의 Bluetooth® 트레이 아이콘 메뉴에서 응용 프로그램을 시작할 수 있습니다.

복구

복구를 수행하려면 다음 단계를 따르십시오.

  1. 제어판을 엽니다.
  2. 제어판의 프로그램 그룹에서 프로그램 제거 링크를 클릭합니다(또는 프로그램 및 기능 아이콘 두 번 클릭).
  3. 응용 프로그램 목록에서 Intel® PROSet/Wireless Software for Bluetooth® Technology를 선택하고 두 번 클릭합니다.
  4. 설치 마법사 창에서 다음을 클릭합니다.
  5. 복구를 클릭하고 지침을 따릅니다.
  6. 복구를 마치려면 시스템을 재부팅합니다. 변경 사항은 재부팅이 완료된 후에 적용됩니다.

제거

제거하려면 다음 단계를 따르십시오.

  1. 제어판을 엽니다.
  2. 제어판의 프로그램 그룹에서 프로그램 제거 링크를 클릭합니다(또는 프로그램 및 기능 아이콘 두 번 클릭).
  3. 응용 프로그램 목록에서 Bluetooth® 기술용 Intel® PROSet/Wireless Software를 선택합니다.
  4. 설치 마법사 창에서 다음을 클릭합니다.
  5. 제거를 클릭하고 지침을 따릅니다.
  6. 제거를 마치려면 시스템을 재부팅합니다. Bluetooth 소프트웨어는 재부팅이 완료된 후에 제거됩니다.

업그레이드

업그레이드를 수행하려면 다음 중 하나를 수행하십시오.

  • 최신 버전의 소프트웨어가 포함된 설치 프로그램 패키지를 실행하고 마법사의 지침을 따릅니다.
  • 현재 설치된 소프트웨어 버전을 제거하고 PC를 재부팅한 후 최신 버전의 소프트웨어가 포함된 설치 프로그램 패키지를 실행합니다.
