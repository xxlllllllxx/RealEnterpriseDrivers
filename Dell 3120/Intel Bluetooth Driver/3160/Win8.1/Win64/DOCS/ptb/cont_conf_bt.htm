Gerenciamento e configuração de Bluetooth®

Ligar e desligar o rádio Bluetooth

Quando for necessário desativar o Bluetooth, pode haver três métodos disponíveis: um interruptor de software, um interruptor de hardware ou o Gerenciador de Dispositivos. É altamente recomendado utilizar um método por vez, e nunca misturá-los (por exemplo, desativar o Bluetooth com o interruptor de software e ativá-lo com o Gerenciador de Dispositivo).

O interruptor de software é o método recomendado para ativar e desativar o rádio Bluetooth; é altamente recomendado não fazê-lo através de desativação/ativação dos drivers do dispositivo Bluetooth no Gerenciador de Dispositivos. No entanto, o interruptor de software pode não estar disponível em todos os sistemas. Os procedimentos para o uso do interruptor de software diferem entre sistemas operacionais e estão descritos abaixo.

Alguns sistemas podem ter um interruptor de hardware no lugar de um interruptor de software, ou adicional a ele. Quando ambos os interruptores estiverem presentes, evite utilizá-los ao mesmo tempo (por exemplo, se o rádio Bluetooth for desativado com o interruptor de software, ele permanecerá desativado, independentemente do ciclo do interruptor de software). Consulte o Manual do Usuário do sistema para obter informações sobre o interruptor de hardware.

No Windows 7, para alternar o rádio Bluetooth entre ativado e desativado:

  1. Clique com o botão direito no ícone Bluetooth.
  2. Selecione Ativar Adaptador ou Desativar Adaptador.

Como alternativa, o rádio Bluetooth pode ser desativado através da janela Configurações de Bluetooth:

  1. Clique com o botão direito no ícone Bluetooth.
  2. Selecione Abrir Configurações.
  3. Na guia Opções, selecione Desativar adaptador Bluetooth.

No Windows 8 e posterior, para alternar o rádio Bluetooth entre ativado e desativado:

  1. Abra Configurações do PC
    • Deslize pelo menu Charm, movendo o cursor para o canto inferior direito, selecione Configurações e clique em Mudar configurações do PC.
  2. Selecione o nó Sem Fio.
  3. Clique no controle deslizante do Bluetooth para Ativar e Desativar o ciclo do Bluetooth.

Configurações de compartilhamento de pasta

Para habilitar ou desabilitar o compartilhamento de arquivos:

  1. Clique com o botão direito do mouse no ícone Bluetooth da bandeja do sistema e selecione Abrir Configurações.
  2. Na janela Configurações de Bluetooth, selecione a guia Pasta Compartilhada e faça o seguinte:
    • Para habilitar o compartilhamento de arquivos, marque a caixa de seleção Permitir que o dispositivos remotos procurem a pasta compartilhada, clique em Procurar para selecionar a pasta local para compartilhamento com os dispositivos Bluetooth e, por fim, clique em OK.
    • Para desabilitar o compartilhamento de arquivos, desmarque a caixa de seleção Permitir que o dispositivos remotos procurem a pasta compartilhada.

Para ativar ou desativar notificações do usuário sobre o carregamento de arquivos via FTP:

  1. Clique com o botão direito do mouse no ícone Bluetooth da bandeja do sistema e selecione Abrir Configurações.
  2. Na janela Configurações de Bluetooth, selecione a guia Pasta Compartilhada e faça o seguinte:
    • Para habilitar notificação de usuário, selecione a caixa de seleção Notificar-me ao fazer upload de arquivos para pasta compartilhada e; em seguida, clique em OK.
    • Para desativar notificação de usuário, desmarque a caixa de seleção Notificar-me ao fazer upload de arquivos para pasta compartilhada.

Configurações de áudio

Observação: esta seção não será aplicável se você estiver usando o sistema operacional Windows 8.

Para gerenciar as configurações de áudio do Bluetooth:

  1. Clique com o botão direito do mouse no ícone Bluetooth da bandeja do sistema e selecione Abrir Configurações.
  2. Na janela Configurações de Bluetooth, selecione a guia Áudio e execute uma ou mais das ações a seguir:
    • Para permitir uma conexão de áudio Bluetooth apenas com dispositivos que suportam proteção de conteúdo por DRM (Digital Rights Management, gerenciamento de direitos digitais) SCMS-T, marque a caixa de seleção Conectar a um dispositivo de áudio, somente se ele suportar a proteção de conteúdo SCMS-T.
    • Para habilitar a comutação automática de chamada para o Skype enquanto estiver ouvindo música, selecione a opção Ativar Alternância do Skype. Em seguida, no menu suspenso, selecione o nome do fone de ouvido Bluetooth previamente conectado. Para obter mais detalhes, consulte Recebendo chamadas de voz via IP (Voice over IP, VoIP) ao ouvir música. Essa opção não está disponível em computadores que não possuem o software do Skype instalado.
