Conectar fones de ouvido e alto-falantes Bluetooth®

Observação: esta seção não é aplicável se você estiver usando o sistema operacional Windows 8.

Fones de ouvido Bluetooth estéreo e mono podem ser usados para ouvir música e atender a chamadas VoIP no seu computador. O computador também pode ser transformado em um alto-falante Bluetooth para ser usado por um dispositivo Bluetooth.

Os perfis de áudio compatíveis incluem:

  • Sink e origem A2DP
  • Controlador e destino AVRCP
  • Gateway de áudio HFP/HSP
  • Integração com Windows Media Player/Center
  • Integração com Skype

Ouvir músicas do computador em um fone de ouvido ou alto-falante Bluetooth

Você pode conectar fones de ouvido Bluetooth estéreo ao computador para ouvir música e para controlar o software leitor de áudio em execução no computador. Quando conectado, o dispositivo de áudio Bluetooth estéreo pode parar/pausar/retomar a reprodução, ir para a faixa seguinte/anterior e aumentar/abaixar o volume do leitor de áudio em execução no computador.

Na janela Serviços do dispositivo Bluetooth adicionado (veja Adicionar e conectar novo dispositivo Bluetooth®):

  1. Clique no link Ouvir Música na janela Serviços para estabelecer uma conexão.
  2. A mensagem da janela Serviços será atualizada assim que a conexão for estabelecida, e o fone de ouvido ou alto-falante Bluetooth passará a ser o dispositivo de áudio padrão do sistema operacional para escutar música tocada no computador.
  3. Se após conectar o fone de ouvido Bluetooth você não estiver conseguindo ouvir música, verifique se as configurações do Dispositivo padrão de áudio estão de acordo com o que está descrito na seção Dispositivo padrão de áudio abaixo.
  4. Para interromper uma conexão de áudio existente, clique em Desconectar no item Ouvir Música da janela Serviços e o dispositivo padrão de áudio do sistema operacional retornará a sua configuração original.

Dispositivo de áudio padrão

Quando se estabelece uma conexão de áudio, o dispositivo Bluetooth de áudio é automaticamente definido como dispositivo padrão de reprodução no sistema operacional Windows. Em algumas raras condições, o dispositivo padrão de áudio Bluetooth pode não funcionar automaticamente. Por exemplo, se você configurar manualmente outro dispositivo de áudio como padrão antes de estabelecer a conexão com o dispositivo de áudio Bluetooth, então este último pode não funcionar. Nesse caso, vá até as propriedades de Som do Windows (a partir do menu da bandeja do sistema ou de Gerenciar os dispositivos de áudio na seção Hardware e Sons do Painel de Controle) e configure o dispositivo Bluetooth de áudio como dispositivo padrão.

Use o fone de ouvido Bluetooth para chamadas de VoIP

Fones de ouvido Bluetooth mono podem ser usados como dispositivo de comunicação para programas de telefonia VoIP.

Na janela Serviços do dispositivo Bluetooth adicionado (veja Adicionar e conectar novo dispositivo Bluetooth®):

  1. Clique no link Encaminhar Chamadas VoIP para Viva-voz na janela Serviços.
  2. A mensagem da janela Serviços será atualizada assim que a conexão for estabelecida, e o fone de ouvido será definido como dispositivo de áudio padrão para comunicação em programas VoIP como o Skype.
  3. Para interromper uma conexão de áudio existente, clique em Desconectar no item Encaminhar Chamadas VoIP para Viva-voz na janela Serviços.

Recebendo chamadas de voz via IP (Voice over IP, VoIP) ao ouvir música

Você pode alternar entre fones de ouvido Bluetooth estéreo e mono diretamente para aceitar ou recusar chamadas Skype recebidas usando os botões do dispositivo Bluetooth enquanto ouve música. A música será automaticamente pausada durante a conversação pelo Skype e será retomada após a chamada ser encerrada.

Observação: você pode permitir que esses recursos funcionem diretamente habilitando o plug-in Bluetooth no Windows Media Player e assegurando que o acesso do Skype seja permitido com o plug-in “btplayerctrl.exe”.

Tocar música de um dispositivo Bluetooth nos alto-falantes do computador

Reproduza áudio a partir de um dispositivo Bluetooth usando os alto-falantes do computador.

Na janela Serviços do dispositivo Bluetooth adicionado (veja Adicionar e conectar novo dispositivo Bluetooth®):

  1. Clique no link Reproduzir Música na janela Serviços.
  2. Depois que essa operação for concluída, o streaming de áudio do dispositivo Bluetooth remoto será encaminhado para os alto-falantes do seu computador (ou algum outro dispositivo padrão de áudio do seu computador).
  3. Com uma conexão Reproduzir Música ativada, é possível usar a janela Controle remoto Bluetooth para controlar o leitor de mídia remoto usando Bluetooth. O ícone Controle remoto Bluetooth será exibido na bandeja do sistema. Se você clicar nesse ícone, a janela Controle remoto Bluetooth será exibida. Essa janela oferece suporte a ações padrão como executar/parar, faixa seguinte/anterior e controles para aumentar/diminuir o volume.
  4. Para deixar de usar o computador como alto-falante Bluetooth, clique no link Desconectar na janela Serviços.
