Transferência de arquivo pelo Bluetooth®

Compartilhe arquivos e pastas locais de seu computador com outros dispositivos Bluetooth, transfira qualquer tipo de arquivo entre seu computador e dispositivos Bluetooth, crie ou até mesmo exclua arquivos e pastas de um dispositivo Bluetooth e use o Assistente Enviar para do menu do Microsoft Office para enviar arquivos para dispositivos Bluetooth.

Perfis e funções suportadas:

  • Perfil de transferência de arquivo (File Transfer Profile, FTP) (Cliente/servidor)
  • Perfil básico de imagem (BIP) (Iniciador/Resposta)
  • Perfil de transmissão do objeto (OPP) (Cliente/Servidor)

Recursos compatíveis:

  • Modo de resposta única (Single Response Mode, SRM) do Object Exchange (OBEX) 1.5
  • Integração do Microsoft Office para o cliente do perfil de envio de objetos (Object Push Profile, OPP)

Compartilhar pasta local com outros dispositivos Bluetooth

Você pode compartilhar pastas do seu computador com outros dispositivos Bluetooth seguindo as etapas descritas na seção Configurações de compartilhamento de pasta deste Guia do usuário.

Pasta compartilhada é usada como a pasta raiz para a transferência de arquivos por meio de FTP, e como a pasta padrão para receber arquivos por meio de OPP e BIP.

Procurar pasta compartilhada de um dispositivo Bluetooth remoto

Quando conectado a um dispositivo Bluetooth com suporte a FTP, uma janela padrão como a do Explorer pode ser utilizada para realizar operações de arquivo normais entre o dispositivo e o computador, da mesma maneira que seriam feitas localmente:

  • Procurar arquivos e pastas
  • Transferir arquivos e pastas
  • Excluir arquivos e pastas
  • Renomear arquivos e pastas
    Observação: se o dispositivo remoto for compatível com a função do servidor FTP anterior a v1.2, apenas as pastas vazias podem ser renomeadas.
  • Criar pastas

Para acessar a pasta compartilhada no dispositivo remoto a partir da Janela de Serviços:

  1. Abra a janela Serviços do dispositivo (consulte Adicionar e conectar novo dispositivo Bluetooth®).
  2. Clique em Procurar Pasta Remota.

Para acessar a pasta compartilhada no dispositivo remoto a partir do Windows Explorer:

  1. Abra a pasta Computador.
  2. Clique no ícone com o nome do dispositivo remoto.

Observação: se o dispositivo remoto for compatível temporariamente com a função do servidor FTP (ou seja, a função pode ser ativada ou desativada), o link na pasta Computador pode ficar ausente. Nesse caso, use a janela de Serviços do dispositivo remoto

Observação: durante a troca de dados, o dispositivo remoto pode solicitar uma senha, e, nesse caso, ambos os dispositivos devem inserir a mesma senha para habilitar a conexão. Se as senhas não corresponderem, a conexão será encerrada.

Enviar arquivos a um dispositivo Bluetooth

O recurso Enviar Arquivo para o Assistente do Dispositivo Bluetooth é usado para enviar arquivos para o dispositivo Bluetooth remoto. O assistente pode enviar apenas um único arquivo, porém vários assistentes podem ser iniciados ao mesmo tempo. Uma sequência de etapas é disponibilizada para selecionar o dispositivo remoto de destino e o arquivo a ser enviado. O recurso Enviar Arquivo para o Assistente do Dispositivo Bluetooth pode ser executado em diversas situações.

Observação: normalmente, BIP é usado para enviar imagens de um dispositivo móvel para um computador. OPP é utilizado para enviar todos os outros tipos de arquivos. O recurso Enviar Arquivo para o Assistente do Dispositivo Bluetooth é compatível com ambos os perfis.

Enviar arquivos do Microsoft Office para um dispositivo Bluetooth

As seguintes versões do Microsoft Office são compatíveis:

  • Microsoft Office 2010
  • Microsoft Office 2013

Os seguintes aplicativos e itens do Microsoft Office são compatíveis:

  • Word: documentos
  • Excel: planilhas
  • PowerPoint: apresentações
  • Access: banco de dados
  • Outlook: compromissos, contatos, mensagens, anotações, tarefas

O item do Microsoft Office é salvo automaticamente antes de ser enviado ao dispositivo Bluetooth remoto.

Observação: Se o item do Microsoft Office não tiver sido salvo ainda, uma solicitação para salvar será mostrada no Word, Excel e PowerPoint.

Observação: Se o item do Microsoft Office for o banco de dados e tiver sido criado recentemente, o aplicativo do Access deverá ser reiniciado.

Para enviar um item do Microsoft Office para um dispositivo Bluetooth remoto:

  1. Dependendo da versão do Microsoft Office e do aplicativo do Microsoft Office que você usar, selecione no submenu Compartilhar, Salvar e Enviar, ou Salvar e Publicar, a ação Enviar para Bluetooth. Para obter instruções sobre como fazer isso, consulte a Ajuda do Microsoft Office. O recurso Assistente para Enviar Arquivos para Dispositivos Bluetooth será exibido.
  2. Após o recurso Enviar Arquivo para o Assistente do Dispositivo Bluetooth abrir, selecione o dispositivo para enviar o arquivo e clique no botão Avançar.
  3. A janela de andamento da transferência de arquivos será exibida, e o item será enviado para o dispositivo Bluetooth selecionado. Você será notificado quando a transferência for concluída.

Enviar arquivos do Windows Explorer para um dispositivo Bluetooth

Para enviar arquivos do Windows Explorer para um dispositivo Bluetooth:

  1. No Windows Explorer, clique com o botão direito do mouse no ícone do arquivo e selecione a ação Bluetooth em Enviar para no menu de contexto. O recurso Enviar Arquivo para o Assistente do Dispositivo Bluetooth será exibido.
  2. Após o recurso Enviar Arquivo para o Assistente do Dispositivo Bluetooth abrir, selecione o dispositivo para enviar o arquivo e clique no botão Avançar.
  3. A janela de andamento da transferência de arquivo será exibida, e o documento escolhido será enviado para o dispositivo Bluetooth selecionado. Você será notificado quando a transferência estiver concluída.

Enviar arquivos da janela Serviços para um dispositivo Bluetooth

Para enviar arquivos da janela Serviços para um dispositivo Bluetooth:

  1. Clique no link Enviar Arquivo na janela Serviços. A janela de diálogo Abrir será exibida.
  2. Escolha um arquivo na lista e clique no botão Abrir. O recurso Enviar Arquivo para o Assistente do Dispositivo Bluetooth será exibido.
  3. Após o recurso Enviar Arquivo para o Assistente do Dispositivo Bluetooth abrir, clique no botão Avançar.
  4. A janela de andamento da transferência de arquivo será exibida, e o documento escolhido será enviado para o dispositivo Bluetooth selecionado. Você será notificado quando a transferência estiver concluída.

Receber arquivos de um dispositivo Bluetooth

Quando outro dispositivo Bluetooth inicia uma transferência de arquivos para o seu computador utilizando perfil BIP ou OPP, a janela Recebendo Arquivos é exibida fornecendo um caminho de destino padrão para os arquivos de entrada.

Observação: normalmente, BIP é usado para enviar imagens de um dispositivo móvel para um computador. OPP é utilizado para enviar todos os outros tipos de arquivos.

Na janela Recebendo arquivos, você pode alterar a pasta de destino padrão onde os arquivos/imagens recebidas são salvos. Para fazer isso, siga estas etapas:

  1. Clique em Procurar e selecione a janela da pasta desejada.
  2. Depois de selecionar um caminho, clique em Aceitar para iniciar a transferência de arquivos.
  3. Para rejeitar um arquivo em vias de recebimento, clique em Rejeitar.

Modificar arquivos e pastas de um dispositivo Bluetooth

Um dispositivo Bluetooth compatível com o perfil FTP pode fazer alterações em arquivos e pastas na pasta local compartilhada. Permissões especiais serão solicitadas se o dispositivo remoto tentar substituir, renomear ou excluir conteúdos compartilhados. A janela Solicitação de permissão será exibida antes que essas alterações possam ser realizadas. Na janela Solicitação de permissão, as seguintes opções estarão disponíveis:

  • Selecione Permitir somente esta vez e clique em OK para permitir alterações de um dispositivo Bluetooth temporariamente.
  • Selecione Sempre permitir este dispositivo e clique em OK para permitir alterações de um dispositivo Bluetooth enquanto um dispositivo Bluetooth é emparelhado.
  • Para recusar as alterações de um dispositivo Bluetooth temporariamente, clique em Cancelar.

Se o arquivo a ser transferido não altera nenhum arquivo existente, então ele deve ser transferido sem a janela Solicitação de permissão ser exibida.

Receber progresso de arquivos

Quando outro dispositivo Bluetooth inicia uma transferência de arquivos no computador e a transferência é aceita, a janela Receber pode ser exibida:

  • Se o dispositivo remoto for BIP ou Cliente OPP, então a janela Receber é exibida.
  • Se o dispositivo remoto for Cliente FTP, então a exibição da janela Receber pode ser configurada através das seguintes etapas descritas na seção Configurações de compartilhamento de pasta deste Guia do Usuário.

A janela Receber mostra as seguintes informações para todos os arquivos:

  • Status de transferência (recebidos com êxito, não recebidos, transferência em andamento etc)
  • Taxa de velocidade de transmissão de arquivos com transferência em andamento
  • Tamanho dos arquivos recebidos com êxito
