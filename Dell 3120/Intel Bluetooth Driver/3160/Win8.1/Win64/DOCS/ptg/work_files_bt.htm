Transferência de ficheiros através do Bluetooth®

Partilhe ficheiros e pastas locais que se encontrem no seu computador com outros dispositivos Bluetooth, transfira qualquer tipo de ficheiro entre o seu computador e dispositivos Bluetooth, crie ou elimine ficheiros e pastas num dispositivo Bluetooth e use o assistente Enviar para do menu Microsoft Office para enviar ficheiros para dispositivos Bluetooth.

Perfis e papéis suportados:

  • Perfil de transferência de arquivos (FTP) (Cliente/Servidor)
  • Perfil básico de imagem (BIP) (Iniciador/Dispositivo de resposta)
  • Perfil de transferência de objetos (OPP) (Cliente/Servidor)

Funcionalidades suportadas:

  • Modo de Resposta Única (SRM) de Troca de Objectos (OBEX) 1.5
  • Integração com o Microsoft Office para o cliente do Perfil de Emissão de Objectos (OPP)

Partilhar pasta local com outros dispositivos Bluetooth

As pastas do seu computador podem ser partilhadas com outros dispositivos Bluetooth seguindo os passos descritos na secção Definições de partilha de pastas deste Guia do Utilizador.

A pasta Partilhada é utilizada como pasta raiz para a transferência de ficheiros via FTP e como pasta predefinida para receber ficheiros via OPP e BIP.

Procurar a pasta partilhada de um dispositivo Bluetooth remoto

Quando estiver ligado a um dispositivo Bluetooth que suporte FTP, pode utilizar uma janela normal como a do Explorador do Windows para efetuar operações normais relativas a ficheiros entre o dispositivo e o computador, tal como se faria a nível local:

  • Procurar ficheiros e pastas
  • Transferir ficheiros e pastas
  • Eliminar ficheiros e pastas
  • Mudar o nome de ficheiros e pastas
    Nota: se o dispositivo remoto suportar a função de servidor FTP inferior a v1.2, é possível mudar o nome apenas a pastas vazias.
  • Criar pastas

Para aceder à pasta partilhada no dispositivo remoto a partir da janela Serviços:

  1. Abra a janela Serviços do dispositivo (consulte Adicionar e ligar a um novo dispositivo Bluetooth®).
  2. Clique em Procurar Pasta Remota.

Para aceder à pasta partilhada no dispositivo remoto a partir do Explorador do Windows:

  1. Abra a pasta Computador.
  2. Clique no ícone com o nome do dispositivo remoto.

Nota: se o dispositivo remoto suportar a função de servidor FTP temporariamente (ou seja, a função de servidor FTP pode ser ligada ou desligada), a ligação na pasta Computador poderá estar ausente. Neste caso, use a janela Serviços do dispositivo remoto.

Nota: durante a troca de dados, o dispositivo remoto poderá pedir uma palavra-passe, pelo que deverá introduzir-se a mesma palavra-passe em ambos os dispositivos para ativar a ligação. Se as palavras-passe não corresponderem, a ligação será terminada.

Enviar ficheiros para um dispositivo Bluetooth

O Assistente para Enviar Ficheiros Para Dispositivos Bluetooth é utlizado para enviar ficheiros para o dispositivo Bluetooth remoto. O assistente apenas consegue enviar um único ficheiro, mas é possível iniciar vários assistentes em simultâneo. É disponibilizada uma sequência de passos para selecionar o dispositivo remoto alvo e o ficheiro a enviar. O Assistente para Enviar Ficheiros Para Dispositivos Bluetooth pode ser executado em diversos cenários.

Nota: normalmente, o BIP é utilizado para enviar imagens de um dispositivo móvel para um computador. O OPP é utilizado para enviar os restantes tipos de ficheiro. O Assistente para Enviar Ficheiros Para Dispositivos Bluetooth suporta ambos os perfis.

Enviar ficheiros do Microsoft Office para um dispositivo Bluetooth

São suportadas as seguintes versões do Microsoft Office:

  • Microsoft Office 2010
  • Microsoft Office 2013

São suportadas as seguintes aplicações e itens do Microsoft Office:

  • Word: documento
  • Excel: livro
  • PowerPoint: apresentação
  • Access: base de dados
  • Outlook: compromisso, contacto, mensagem, nota, tarefa

O item do Microsoft Office é guardado automaticamente antes de poder ser enviado para um dispositivo Bluetooth remoto.

Nota: se o item do Microsoft Office não tiver sido guardado, ser-lhe-á solicitado que o guarde em Word, Excel e PowerPoint.

Nota: se o item do Microsoft Office for uma base de dados e esta acabar de ser criada, a aplicação Access deve ser reiniciada.

Para enviar um item do Microsoft Office para um dispositivo Bluetooth remoto:

  1. Dependendo da versão do Microsoft Office e da aplicação Microsoft Office que utilizar, selecione a ação Enviar para Bluetooth no submenu Partilhar, Guardar e enviar ou Guardar e publicar. Consulte a Ajuda do Microsoft Office para obter instruções sobre como o fazer. O Assistente para Enviar Ficheiros Para Dispositivos Bluetooth será apresentado.
  2. Quando o Assistente para Enviar Ficheiros Para Dispositivos Bluetooth abrir, selecione o dispositivo Bluetooth para o qual pretende enviar o ficheiro e clique no botão Seguinte.
  3. Será apresentada a janela de progresso da transferência do ficheiro e o item será enviado para o dispositivo Bluetooth selecionado. Receberá uma notificação assim que a transferência tiver sido concluída.

Enviar ficheiros do Explorador do Windows para um dispositivo Bluetooth

Para enviar ficheiros do Explorador do Windows para um dispositivo Bluetooth:

  1. No Explorador do Windows, clique com o botão direito do rato no ícone do ficheiro e selecione a ação Bluetooth no item Enviar para no menu de contexto. O Assistente para Enviar Ficheiros Para Dispositivos Bluetooth será apresentado.
  2. Quando o Assistente para Enviar Ficheiros Para Dispositivos Bluetooth abrir, selecione o dispositivo Bluetooth para o qual pretende enviar o ficheiro e clique no botão Seguinte.
  3. A janela de progresso de transferência do ficheiro irá aparecer e o ficheiro escolhido será enviado para o dispositivo Bluetooth selecionado. Será avisado quando a transferência estiver concluída.

Enviar ficheiros da janela Serviços para um dispositivo Bluetooth

Para enviar ficheiros da Janela Serviços para um dispositivo Bluetooth:

  1. Clique na hiperligação Enviar Ficheiro na janela Serviços. A janela de diálogo Abrir irá aparecer.
  2. Escolha um ficheiro a partir da lista e clique no botão Abrir. O Assistente para Enviar Ficheiros Para Dispositivos Bluetooth será apresentado.
  3. Quando o Assistente para Enviar Ficheiros Para Dispositivos Bluetooth abrir, clique no botão Seguinte.
  4. A janela de progresso de transferência do ficheiro irá aparecer e o ficheiro escolhido será enviado para o dispositivo Bluetooth selecionado. Será avisado quando a transferência estiver concluída.

Receber ficheiros a partir de um dispositivo Bluetooth

Quando outro dispositivo Bluetooth inicia uma transferência de ficheiro para o seu computador através de BIP ou OPP, é apresentada a janela A Receber Ficheiros com o local de destino predefinido para os ficheiros recebidos.

Nota: normalmente, o BIP é utilizado para enviar imagens de um dispositivo móvel para um computador. O OPP é utilizado para enviar os restantes tipos de ficheiro.

Na janela A Receber Ficheiros, poderá alterar a pasta de destino predefinida onde os ficheiros/imagens a receber são guardados. Para o fazer, siga estes passos:

  1. Clique em Procurar e selecione a janela da pasta pretendida.
  2. Após ter seleccionado um caminho, clique em Aceitar para iniciar a transferência de ficheiros.
  3. Para rejeitar um ficheiro recebido, clique em Rejeitar.

Modificar ficheiros e pastas a partir de um dispositivo Bluetooth

Um dispositivo Bluetooth que suporte o perfil FTP pode efetuar alterações aos ficheiros e às pastas na pasta partilhada local. Serão necessárias permissões especiais se o dispositivo remoto tentar substituir, mudar o nome ou eliminar conteúdo partilhado. A janela Pedido de permissão será apresentada antes de poder efetuar estas alterações. Na janela Pedido de permissão pode:

  • Selecionar Permitir apenas desta vez e clicar em OK para permitir temporariamente alterações a partir de um dispositivo Bluetooth.
  • Selecionar Permitir sempre este dispositivo e clicar em OK para permitir alterações a partir de um dispositivo Bluetooth enquanto um dispositivo Bluetooth está emparelhado.
  • Recusar temporariamente alterações a partir de um dispositivo Bluetooth clicando em Cancelar.

Se o ficheiro a ser transferido não alterar um ficheiro existente, será transferido sem que a janela Pedido de permissão seja apresentada.

Progresso da receção de ficheiros

Quando outro dispositivo Bluetooth inicia uma transferência de ficheiro para o seu computador e a transferência é aceite, a janela A Receber poderá ser apresentada:

  • Se o dispositivo remoto for um cliente BIP ou OPP, a janela A Receber é apresentada.
  • Se o dispositivo remoto for um cliente FTP, a apresentação da janela A Receber pode ser configurada seguindo os passos descritos na secção Definições de partilha de pastas deste Guia do utilizador.

A janela A Receber apresenta as seguintes informações para cada ficheiro:

  • Estado da transferência (recebido com sucesso, não recebido, transferência em curso, etc.)
  • Velocidade da transferência para os ficheiros que se encontram a ser transferidos atualmente
  • Tamanho dos ficheiros recebidos com sucesso
