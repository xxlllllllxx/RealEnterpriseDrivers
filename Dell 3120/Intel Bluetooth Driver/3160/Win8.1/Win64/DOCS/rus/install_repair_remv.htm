Установка, восстановление, удаление и обновление

Установка

Для начала установки подключите оборудование и запустите файл Setup.exe с установочного диска. Запустится Мастер установки.

Выполните шаги, указанные мастером установки:

  1. Щелкните Далее для перехода на страницу Приветствие.
  2. На странице Лицензионное соглашение прочитайте и примите условия лицензионного соглашения. Щелкните Далее.
  3. Если нужно, измените папку установки на странице Папка установки.
    Путь установки по умолчанию – "C:\Program Files\Intel\Bluetooth".
  4. На странице Все готово для установки программы щелкните Установить для запуска процесса установки. Отобразится ход выполнения установки.
  5. Когда процесс установки завершится, нажмите кнопку Готово.
  6. После установки приложение можно запускать с помощью значка Bluetooth® на панели задач.

Восстановление

Для восстановления выполните следующие действия:

  1. Откройте Панель управления.
  2. Щелкните ссылку Удаление программы в группе Программы (или дважды щелкните значок Программы и компоненты) на Панели управления.
  3. В списке приложений выберите и дважды щелкните Intel® PROSet/Wireless Software for Bluetooth® Technology.
  4. В окне Мастера настройки щелкните Далее.
  5. Щелкните Восстановить и следуйте инструкциям.
  6. Для завершения восстановления перезагрузите систему. Все изменения вступят в силу после перезагрузки.

Удаление

Для удаления выполните следующие действия.

  1. Откройте Панель управления.
  2. Щелкните ссылку Удаление программы в группе Программы (или дважды щелкните значок Программы и компоненты) на Панели управления.
  3. В списке приложений выберите и дважды щелкните Intel® PROSet/Wireless Software for Bluetooth® Technology.
  4. В окне Мастера настройки щелкните Далее.
  5. Щелкните Удалить и следуйте инструкциям.
  6. Для завершения удаления перезагрузите систему. Программное обеспечение Bluetooth будет полностью удалено после перезагрузки.

Обновление

Для обновления выполните одно из следующих действий:

  • Запустите пакет установщика с самой последней версией программного обеспечения и следуйте инструкциям Мастера.
  • Удалите установленную версию программного обеспечения, перезагрузите ПК и запустите пакет установщика с самой последней версией программного обеспечения.
