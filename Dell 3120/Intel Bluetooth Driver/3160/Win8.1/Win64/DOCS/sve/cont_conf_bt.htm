Hantering och konfiguration av Bluetooth®

Aktivera eller avaktivera Bluetooth-radion

När det är nödvändigt att inaktivera Bluetooth kan du använda någon av tre metoder: programvaruswitch, maskinvaruswitch eller Bluetooth-enhetshanteraren. Vi avråder bestämt från att omväxlande använda de här tre metoderna (till exempel genom att aktivera Bluetooth med programvaruswitchen och inaktivera med enhetshanteraren).

Programvaruswitchen är den rekommenderade metoden för att aktivera och inaktivera Bluetooth-radion. Vi avråder bestämt från att göra det genom att aktivera/inaktivera drivrutinerna för Bluetooth-enheten i enhetshanteraren. Det kan emellertid hända att programvaruswitchen inte är tillgänglig i alla system. Proceduren för att använda programvaruswitchen varierar beroende på operativsystem. De olika procedurerna anges nedan.

Vissa system kan ha en maskinvaruswitch istället för, eller förutom, en programvaruswitch. När båda typerna av switch finns bör du undvika att använda dem samtidigt (till exempel om Bluetooth-radion inaktiveras med programvaruswitchen går det inte att aktivera den igen med maskinvaruswitchen). Mer information om maskinvaruswitchen finns i användarhandboken till systemet.

Så här aktiverar eller inaktiverar du Bluetooth-radion i Windows 7:

  1. Högerklicka på Bluetooth-ikonen i systemfältet.
  2. Välj Slå på adapter eller Slå av adapter.

Alternativt kan Bluetooth-radion inaktiveras i fönstret Bluetooth-inställningar:

  1. Högerklicka på Bluetooth-ikonen i systemfältet.
  2. Välj Öppna inställningar.
  3. På fliken Alternativ markerar du Inaktivera Bluetooth-adaptern.

Så här aktiverar eller inaktiverar du Bluetooth-radion i Windows 8 och senare:

  1. Öppna Datorinställningar
    • Visa snabbknappsmenyn genom att flytta muspekaren till det nedre högra hörnet, välj Inställningar och klicka på Ändra datorinställningar.
  2. Välj noden Trådlöst.
  3. Klicka på Bluetooth-reglaget för att aktivera eller inaktivera Bluetooth.

Inställningar för mappdelning

Så här aktiverar eller avaktiverar du fildelning:

  1. Högerklicka på Bluetooth-ikonen i systemfältet och välj Öppna inställningar.
  2. Klicka på fliken Delad mapp i fönstret för Bluetooth-inställningar och gör sedan något av följande:
    • Aktivera fildelning genom att markera kryssrutan Tillåt att andra enheter bläddrar i delad mapp, klicka på Bläddra och välj den lokala mapp som ska delas med Bluetooth-enheterna och klicka därefter på OK.
    • Avaktivera fildelning genom att avmarkera kryssrutan Tillåt att andra enheter bläddrar i delad mapp.

Så här aktiverar eller inaktiverar du användaraviseringar om överföring av filer via FTP:

  1. Högerklicka på Bluetooth-ikonen i systemfältet och välj Öppna inställningar.
  2. Klicka på fliken Delad mapp i fönstret Bluetooth-inställningar och gör sedan något av följande:
    • Om du vill aktivera användaraviseringar markerar du Meddela mig vid överföring av filer till den delade mappen och klickar sedan på OK.
    • Om du vill inaktivera användaraviseringar avmarkerar du Meddela mig vid överföring av filer till den delade mappen.

Ljudinställningar

Obs: Det här avsnittet gäller inte om du använder operativsystemet Windows 8.

Hantera ljudinställningar för Bluetooth:

  1. Högerklicka på Bluetooth-ikonen i systemfältet och välj Öppna inställningar.
  2. Klicka på fliken Ljud i fönstret för Bluetooth-inställningar och gör sedan något eller några av följande:
    • Om du vill att Bluetooth-ljudanslutning endast ska vara möjlig med enheter som har stöd för SCMS-T/DRM innehållsskydd markerar du kryssrutan Anslut endast till ljudenhet om den stöder SCMS-T innehållsskydd.
    • Om du automatiskt vill kunna växla till Skype-samtal medan du lyssnar på musik väljer du alternativet Aktivera Skype-omkoppling och väljer namnet på ett tidigare anslutet Bluetooth-headset i den nedrullningsbara menyn. Mer information finns i Ta emot VoIP-samtal (Voice over IP) medan du lyssnar på musik. Detta alternativ är inte tillgängligt om inte programmet Skype är installerat på datorn.
