การติดตั้ง

ในการเริ่มต้นการติดตั้ง ให้เปิดฮาร์ดแวร์และเริ่ม Setup.exe จากแผ่นดิสก์การติดตั้งของคุณ ซึ่งจะเริ่มต้น Wizard สำหรับการติดตั้ง

ปฏิบัติตามขั้นตอนที่ Installation Wizard ระบุ ดังนี้

  1. คลิก ถัดไป ในหน้า ยินดีต้อนรับ
  2. อ่านข้อตกลงการอนุญาตใช้สิทธิ์และยอมรับในหน้า ข้อตกลงการอนุญาตใช้สิทธิ์ จากนั้นคลิก ถัดไป
  3. หากจำเป็น เปลี่ยนโฟลเดอร์ปลายทางบนหน้า โฟลเดอร์ปลายทาง
    การติดตั้งตามค่าเริ่มต้น อยู่ที่ C:\Program Files\Intel\Bluetooth
  4. ในหน้า พร้อมที่จะติดตั้งโปรแกรม ให้คลิก ติดตั้ง เพื่อเริ่มต้นกระบวนการติดตั้ง ความคืบหน้าของการติดตั้งจะปรากฏ
  5. เมื่อขั้นตอนการติดตั้งเสร็จสมบูรณ์ คลิก เสร็จสิ้น
  6. หลังจากการติดตั้ง สามารถเริ่มแอปพลิเคชันได้จากเมนูไอคอนถาด Bluetooth® ในพื้นที่ถาดระบบ

การแก้ไข

ในการซ่อมแซม ให้ทำตามขั้นตอนเหล่านี้:

  1. เปิด แผงควบคุม
  2. คลิกลิงก์ ถอนการติดตั้งโปรแกรม ในกลุ่ม โปรแกรม (หรือดับเบิลคลิกที่ไอคอน โปรแกรมและคุณลักษณะ) ของ แผงควบคุม
  3. เลือก Intel® PROSet/Wireless Software for Bluetooth® Technology ในรายการแอปพลิเคชัน และดับเบิลคลิก
  4. คลิก ถัดไป ในหน้าต่าง Wizard สำหรับการติดตั้ง
  5. คลิก ซ่อมแซม และทำตามคำแนะนำ
  6. ในการเสร็จสิ้นการซ่อมแซม ให้ปิดแล้วเปิดเครื่องใหม่ การเปลี่ยนแปลงทั้งหมดจะเกิดขึ้นหลังจากปิดแล้วเปิดเครื่องใหม่

การลบ

ในการยกเลิกการติดตั้ง ให้ทำตามขั้นตอนเหล่านี้:

  1. เปิด แผงควบคุม
  2. คลิกลิงก์ ถอนการติดตั้งโปรแกรม ในกลุ่ม โปรแกรม (หรือดับเบิลคลิกที่ไอคอน โปรแกรมและคุณลักษณะ) ของ แผงควบคุม
  3. เลือก Intel® PROSet/Wireless Software for Bluetooth® Technology ในรายการแอปพลิเคชัน และดับเบิลคลิก
  4. คลิก ถัดไป ในหน้าต่าง Wizard สำหรับการติดตั้ง
  5. คลิก ลบออก และทำตามคำแนะนำ
  6. ในการเสร็จสิ้นการลบ ให้ปิดแล้วเปิดเครื่องใหม่ ซอฟต์แวร์ Bluetooth จะถูกลบออกหลังจากปิดแล้วเปิดเครื่องใหม่

ปรับรุ่น

ทำอย่างใดอย่างหนึ่งต่อไปนี้เพื่ออัปเกรด:

  • ดำเนินการชุดตัวช่วยทำการติดตั้งที่มีซอฟต์แวร์รุ่นใหม่ล่าสุด และทำตามคำแนะนำของ Wizard
  • ยกเลิกการติดตั้งซอฟต์แวร์รุ่นที่ติดตั้งในปัจจุบัน ปิดแล้วเปิดเครื่อง PC ใหม่ และดำเนินการชุดตัวช่วยทำการติดตั้งที่มีซอฟต์แวร์รุ่นใหม่ล่าสุด
