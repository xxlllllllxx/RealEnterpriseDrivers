คำอธิบายผลิตภัณฑ์

ภาพรวม

Intel® PROSet/Wireless Software for Bluetooth® Technology อนุญาตการเชื่อมต่อไร้สายกับคอมพิวเตอร์โดยใช้อุปกรณ์ Bluetooth ต่างๆ เช่น โทรศัพท์มือถือ หูฟัง ชุดหูฟังสเตอริโอ สมาร์ทแฮนด์เฮลด์ เครื่องพิมพ์ กล้องดิจิตอล และอื่นๆ คุณสามารถใช้ซอฟต์แวร์ Bluetooth เพื่อใช้แฟ้มร่วมกัน ฟังเพลง โทรออก และใช้เครื่องพิมพ์ Bluetooth

Intel® PROSet/Wireless Software for Bluetooth® Technology ช่วยให้การเชื่อมต่อโดยใช้ Bluetooth ง่ายขึ้นโดยการติดตามอุปกรณ์ที่เชื่อมต่อทั้งหมด และเชื่อมต่อโดยอัตโนมัติ แต่ยังสามารถปรับแต่งด้วยตนเองได้เมื่อต้องการ

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับระบบปฏิบัติการและอะแดปเตอร์ที่รองรับ ให้ดู ผลิตภัณฑ์ Intel® Wireless

ข้อมูลจำเพาะ

รองรับส่วนต่อประสานหลักของ Bluetooth:

  • ส่วนต่อประสานตัวควบคุมโฮสต์ (HCI)

รองรับโปรไฟล์ Bluetooth:

  • โปรไฟล์การกระจายเสียงขั้นสูง (A2DP) (ที่มา/ที่เก็บ)1
  • โปรไฟล์การควบคุมเสียง/วิดีโอระยะไกล (AVRCP) (ตัวควบคุม/เป้าหมาย)1
  • โปรไฟล์การสร้างภาพพื้นฐาน (BIP) (ตัวเริ่ม/ตัวตอบสนอง)
  • โปรไฟล์การพิมพ์พื้นฐาน (BPP) (ผู้ส่ง)
  • โปรไฟล์การถ่ายโอนไฟล์ (FTP) (ไคลเอนต์/เซิร์ฟเวอร์)
  • โปรไฟล์การเข้าถึงทั่วไป (GAP)
  • โปรไฟล์คุณสมบัติทั่วไป (GATT)
  • โปรไฟล์การกระจายเสียง/ภาพทั่วไป (GAVDP) (แหล่ง/ตัวรับ)1
  • โปรไฟล์การแลกเปลี่ยนข้อมูลทั่วไป (GOEP) (ไคลเอนต์/เซิร์ฟเวอร์)
  • โปรไฟล์แฮนด์ฟรี (HFP) (เกตเวย์เสียง) ที่รองรับเสียงพูดแถบกว้าง (WBS)1
  • โปรไฟล์การเปลี่ยนสายเคเบิล Hardcopy (HCRP) (ไคลเอนต์)
  • โปรไฟล์ชุดหูฟัง (HSP) (เกตเวย์เสียง)1
  • โปรไฟล์ HID ผ่าน GATT (HOGP) (โฮสต์) หรือโปรไฟล์ HID พลังงานต่ำ2
  • โปรไฟล์การส่งออบเจกต์ (OPP) (ไคลเอนต์/เซิร์ฟเวอร์)
  • โปรไฟล์การเข้าถึงสมุดโทรศัพท์ (PBAP) (ไคลเอนต์)
  • โปรไฟล์การปรับข้อมูลให้ตรงกัน (SYNC) (ไคลเอนต์)

ระบบปฏิบัติการ:

  • Windows Vista (32 บิต และ 64 บิต) SP23
  • Windows 7 (32 บิต และ 64 บิต)
  • Windows 8 (32 บิต และ 64 บิต)
  • Windows 8.1 (32 บิต และ 64 บิต)

Intel® PROSet/Wireless Software ยังแนะนำคุณสมบัติใหม่ๆ เหล่านี้

  • การรวม Windows Media Player และ Media Center1
  • การรวมกับซอฟต์แวร์ Skype Voice over IP (VoIP)1
  • การรวมกับ Microsoft Office
  • ความสามารถในการปิดสัญญาณวิทยุ Bluetooth3
  • การสนับสนุนโหมดเครื่องบินใน Windows 8 และ Windows 8.1
  • การรองรับ Bluetooth Low Energy2

รายการคำย่อ:

  • A2DP  โปรไฟล์การกระจายเสียงขั้นสูง
  • AMP  ตัวเลือก MAC/PHY
  • AVRCP  โปรไฟล์การควบคุมเสียง/วิดีโอจากระยะไกล
  • BIP  โปรไฟล์การสร้างภาพพื้นฐาน
  • BPP  โปรไฟล์การพิมพ์พื้นฐาน
  • DRM  การจัดการสิทธิ์ดิจิตอล
  • EDR  อัตราข้อมูลขั้นสูง
  • FTP  โปรไฟล์การถ่ายโอนไฟล์
  • GAP  โปรไฟล์การเข้าถึงทั่วไป
  • GATT  โปรไฟล์คุณสมบัติทั่วไป
  • HID  อุปกรณ์ส่วนต่อประสานกับมนุษย์
  • HFP  โปรไฟล์อุปกรณ์แฮนด์ฟรี
  • HOGP  โปรไฟล์ HID ผ่าน GATT
  • HSP  โปรไฟล์ชุดหูฟัง
  • MAC/PHY  การควบคุมการเข้าถึงสื่อ/ระดับเชิงกายภาพ
  • OBEX  การแลกเปลี่ยนข้อมูล
  • PBAP  โปรไฟล์การเข้าถึงสมุดโทรศัพท์
  • SCMS-T  ระบบการจัดการการคัดลอกแบบอนุกรม
  • SYNCH  โปรไฟล์การปรับข้อมูลให้ตรงกัน
  • SRM  โหมดการตอบครั้งเดียว
  • VoIP  เสียงผ่าน IP
  • WBS  เสียงพูดแถบกว้าง

1 Windows 8 และ Windows 8.1 รองรับโปรไฟล์เสียงโดยธรรมชาติ การโต้ตอบกับ Media Player และซอฟต์แวร์ VoIP อยู่นอกขอบเขตของ Intel® PROSet/Wireless Software for Bluetooth® Technology ตั้งแต่ Windows 8 เป็นต้นไป

2 คุณสมบัติ Low Energy ของ Bluetooth ที่อธิบายในเอกสารนี้สำหรับ Windows 7 เท่านั้น Windows 8 และ Windows 8.1 รองรับ Low Energy ของ Bluetooth โดยธรรมชาติ และ Low Energy ของ Bluetooth อยู่นอกขอบเขตของ Intel® PROSet/Wireless Software for Bluetooth® Technology ตั้งแต่ Windows 8 เป็นต้นไป

3 ความสามารถในการปิดสัญญาณวิทยุ Bluetooth จากเมนูถาดระบบสำหรับ Windows 7 เท่านั้น
