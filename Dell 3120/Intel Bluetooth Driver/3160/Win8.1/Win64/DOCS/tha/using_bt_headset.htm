เชื่อมต่อกับชุดหูฟังและลำโพง Bluetooth®

หมายเหตุ: ส่วนนี้จะไม่เกี่ยวข้องหากคุณใช้ระบบปฏิบัติการ Windows 8

สามารถใช้ชุดหูฟังสเตอริโอและโมโน Bluetooth เพื่อฟังเพลงและรับสาย VoIP จากคอมพิวเตอร์ของคุณได้ นอกจากนั้น ยังสามารถเปลี่ยนคอมพิวเตอร์เป็นลำโพง Bluetooth เพื่อให้อุปกรณ์ Bluetooth ใช้ได้ด้วย

โปรไฟล์เสียงที่รองรับ ได้แก่

  • ที่มาและที่เก็บ A2DP
  • ตัวควบคุมและเป้าหมาย AVRCP
  • เกตเวย์เสียง HFP/HSP
  • การรวมกับ Windows Media Player/Center
  • การรวมกับ Skype

ฟังเพลงจากคอมพิวเตอร์ผ่านชุดหูฟังหรือลำโพง Bluetooth

สามารถเชื่อมต่อชุดหูฟังสเตอริโอ Bluetooth กับคอมพิวเตอร์เพื่อฟังเพลง และควบคุมซอฟต์แวร์เครื่องเล่นเสียงที่ทำงานในคอมพิวเตอร์ได้ เมื่อเชื่อมต่อ อุปกรณ์ Bluetooth Stereo Audio สามารถหยุด/หยุดชั่วคราว/เล่นต่อ ไปที่เพลงถัดไป/ก่อนหน้า และเพิ่ม/ลดระดับเสียงของเครื่องเล่นเสียงที่ทำงานในคอมพิวเตอร์ได้

ในหน้าต่าง บริการต่างๆ ของอุปกรณ์ Bluetooth ที่เพิ่ม (ดู เพิ่มและเชื่อมต่อกับอุปกรณ์ Bluetooth® ใหม่):

  1. คลิกลิงก์ ฟังเพลง ในหน้าต่าง บริการต่างๆ เพื่อสร้างการเชื่อมต่อ
  2. ข้อความในหน้าต่าง บริการต่างๆ จะอัปเดตเมื่อสร้างการเชื่อมต่อ และชุดหูฟังหรือลำโพง Bluetooth จะเป็นอุปกรณ์เสียงเริ่มต้นของระบบปฏิบัติการสำหรับฟังเพลเล่นจากคอมพิวเตอร์
  3. หากไม่ได้ยินเสียงเพลงจากชุดหูฟัง Bluetooth ของคุณเมื่อเชื่อมต่อ โปรดตรวจสอบว่าการตั้งค่า อุปกรณ์เสียงเริ่มต้น เหมือนกับที่อธิบายในส่วน อุปกรณ์เสียงเริ่มต้น ด้านล่าง
  4. ในการตัดการเชื่อมต่อเสียงที่มีอยู่ ให้คลิก ตัดการเชื่อมต่อ ในรายการ ฟังเพลง ในหน้าต่าง บริการต่างๆ และอุปกรณ์เสียงเริ่มต้นของระบบปฏิบัติการจะเปลี่ยนกลับเป็นการตั้งค่าเดิม

อุปกรณ์เสียงตามค่าเริ่มต้น

เมื่อสร้างการเชื่อมต่อเสียงแล้ว อุปกรณ์เสียง Bluetooth จะถูกตั้งค่าเป็นอุปกรณ์การเล่นเริ่มต้นในระบบปฏิบัติการ Windows โดยอัตโนมัติ ในบางกรณีที่พบได้ไม่บ่อย อุปกรณ์เสียง Bluetooth เริ่มต้นอาจไม่ทำงานโดยอัตโนมัติ ตัวอย่างเช่น หากคุณตั้งค่าอุปกรณ์เสียงอื่นเป็นอุปกรณ์เสียงเริ่มต้นด้วยตนเองก่อนสร้างการเชื่อมต่อกับอุปกรณ์เสียง Bluetooth อุปกรณ์เสียง Bluetooth อาจไม่ทำงาน ในกรณีนี้ ให้ไปที่คุณสมบัติเสียงของ Windows จากถาดระบบ หรือจัดการอุปกรณ์เสียงจากส่วนฮาร์ดแวร์และเสียงของแผงควบคุม และตั้งค่าอุปกรณ์เสียง Bluetooth เป็นอุปกรณ์เริ่มต้น

ใช้ชุดหูฟัง Bluetooth สำหรับสาย VoIP

สามารถใช้ชุดหูฟังโมโน Bluetooth เป็นอุปกรณ์สื่อสารเสียงสำหรับโปรแกรมโทรศัพท์ VoIP

ในหน้าต่าง บริการต่างๆ ของอุปกรณ์ Bluetooth ที่เพิ่ม (ดู เพิ่มและเชื่อมต่อกับอุปกรณ์ Bluetooth® ใหม่):

  1. คลิกลิงก์ โอนสาย VoIP ไปยังอุปกรณ์แฮนด์ฟรี ในหน้าต่าง บริการต่างๆ
  2. ข้อความในหน้าต่าง บริการต่างๆ จะอัปเดตเมื่อสร้างการเชื่อมต่อ และชุดหูฟังจะถูกตั้งค่าเป็นอุปกรณ์สื่อสารเสียงเริ่มต้นสำหรับโปรแกรม VoIP เช่น Skype
  3. ในการตัดการเชื่อมต่อเสียงที่มีอยู่ ให้คลิก ตัดการเชื่อมต่อ ในรายการ โอนสาย VoIP ไปยังอุปกรณ์แฮนด์ฟรี ในหน้าต่าง บริการต่างๆ

การรับเสียงผ่านการโทร IP(VoIP) ขณะฟังเพลง

สามารถเปลี่ยนชุดหูฟังสเตอริโอและโมโน Bluetooth เพื่อรับหรือวางสายเรียกเข้า Skype โดยใช้ปุ่มของอุปกรณ์e Bluetooth ขณะฟังเพลงได้ เพลงจะหยุดชั่วคราวโดยอัตโนมัติระหว่างการสนทนาสาย Skype และเล่นต่อหลังจากวางสายแล้ว

หมายเหตุ: คุณสามารถอนุญาตให้คุณสมบัติเหล่านี้ทำงานได้โดยเปิดปลั๊กอิน Bluetooth ใน Windows Media Player และอนุญาตให้ปลั๊กอิน btplayerctrl.exe เข้าถึง Skype ได้

เล่นเพลงจากอุปกรณ์ Bluetooth ผ่านลำโพงของคอมพิวเตอร์

เล่นเสียงจากอุปกรณ์ Bluetooth โดยใช้ลำโพงของคอมพิวเตอร์

ในหน้าต่าง บริการต่างๆ ของอุปกรณ์ Bluetooth ที่เพิ่ม (ดู เพิ่มและเชื่อมต่อกับอุปกรณ์ Bluetooth® ใหม่):

  1. คลิกลิงก์ เล่นเพลง ในหน้าต่าง บริการต่างๆ
  2. หลังจากดำเนินการนี้แล้ว สตรีมเสียงจากอุปกรณ์ Bluetooth ระยะไกลจะส่งต่อไปยังลำโพงของคอมพิวเตอร์ของคุณ (หรืออุปกรณ์เสียงเริ่มต้นอื่นที่กำหนดค่าในคอมพิวเตอร์)
  3. เมื่อมีการเชื่อมต่อ เล่นเพลง ที่ใช้งานได้ จะสามารถใช้หน้าต่าง รีโมทคอนโทรล Bluetooth เพื่อควบคุม Media Player ระยะไกลโดยใช้ Bluetooth ได้ ไอคอน รีโมทคอนโทรล Bluetooth จะปรากฏในถาดระบบ หน้าต่าง รีโมทคอนโทรล Bluetooth จะปรากฏเมื่อคลิกไอคอน หน้าต่างรองรับการดำเนินการมาตรฐาน เช่น การควบคุมเล่น/หยุด เพลงถัดไป/ก่อนหน้า และเพิ่ม/ลดระดับเสียง
  4. ในการหยุดใช้คอมพิวเตอร์เป็นลำโพง Bluetooth ให้คลิกลิงก์ ตัดการเชื่อมต่อ ในหน้าต่าง บริการต่างๆ
